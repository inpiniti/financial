# EDU 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-21 | 투자의견: 매수 | 확신도: 7/10

## 한 줄 결론
좋은 기업(ROC 약 29%)을 적당히 싼 가격(EY 약 12.5%)에 살 수 있는 종목입니다. 마법 공식 두 지표 모두 상위권 — 다만 회계상 음(-)의 운전자본 구조와 대형주라는 점, 중국 규제 리스크는 별도로 인지해야 합니다.

## 내 철학에서 본 이 기업
저는 EDU가 무슨 사업을 하는지, 경영진이 훌륭한지 같은 이야기는 궁금하지 않습니다. 제가 궁금한 건 딱 두 가지입니다. "이 사업체는 투입한 돈 대비 얼마나 버는가?"(ROC) 그리고 "지금 그 사업체를 얼마에 살 수 있는가?"(EY). 아래에서 직접 계산해 보겠습니다. 계산이 곧 제 철학의 전부입니다.

## 체크리스트 평가

1. **자본수익률(ROC)** — ○
   - EBIT $489M(FY2025 확정 연간 영업이익) ÷ 투입 유형자본 $1,693M(보수적 방식) = **28.9%**
   - 25% 이상이면 우량 기업으로 보는 제 기준을 충족합니다.
2. **이익수익률(EY)** — ○
   - EBIT $489M ÷ EV $3,902M = **12.5%**
   - 무위험 금리 대비 3배 이상 스프레드 — 싸다는 신호입니다.
3. **순위 결합** — ○
   - 두 지표 모두 상위권에 해당하는 "좋으면서 싼" 기업. 제가 가장 선호하는 조합입니다.
4. **규모 필터** — △
   - 시가총액 $8.21B(비희석 $7.90B)로 원래 마법 공식이 설계된 소형주(시총 $50M~$1B대) 유니버스를 훨씬 벗어난 대형주입니다. 공식의 통계적 근거가 원래 상정한 표본과 다릅니다.
5. **재무 건전성(부가)** — ○
   - 유동비율 1.66, Debt/Equity 18.3%, 이자보상배율 사실상 1,000배 이상 — 무차입에 가까운 우량한 재무구조. 껌 가게 비유로 치면 대출 없이 현금으로 가게를 운영하는 셈입니다.
6. **음의 운전자본 구조** — △
   - 선수금(학원 선납 수강료) 규모가 매우 커서 순운전자본이 구조적으로 마이너스(-$2,381M, 최근분기 기준). 자본이 거의 필요 없는 비즈니스라는 좋은 신호이지만, 공식을 기계적으로 적용할 때 ROC 계산이 왜곡될 위험이 있어 보수적으로 조정했습니다.

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

### 1단계 — EBIT
- FY2025(2025.5 마감) 확정 연간 영업이익 = **$489M**
- (참고: FY2026 진행 중 분기 합산 Q1~Q3 = 311+66+180 = $557M로 개선 흐름이지만, FY2025 Q1~Q3 개별분기 데이터가 데이터 팩에 없어 정확한 TTM 산출은 확인 불가 → 확정 연간치를 기본 케이스로 채택)

### 2단계 — 투입 유형자본 (최근분기 2026-02-28 스냅샷)

| 항목 | 값 |
|---|---|
| 유동자산 | $5,427M |
| (−) 현금+단기투자 | $4,802M |
| = 영업용 유동자산 | $625M |
| 유동부채 | $3,279M |
| (−) 이자부 유동부채(단기리스) | $273M |
| = 무이자성 유동부채 | $3,006M |
| **순운전자본(NWC)** = 625 − 3,006 | **-$2,381M** |
| 순고정자산(Net PP&E) | $1,693M |
| 원안 투입자본 = NWC + Net PP&E | -$688M (음수, 왜곡) |
| **보수적 투입자본** = max(NWC,0) + Net PP&E | **$1,693M** (채택) |

### 3단계 — ROC
```
ROC = EBIT ÷ 투입 유형자본 = 489 ÷ 1,693 = 28.9%
```

### 4단계 — EV

| 항목 | 값 |
|---|---|
| 시가총액(non-diluted) | $7,900M |
| (+) 이자부 총부채(FY2025) | $804M |
| (−) 현금성자산(최근분기) | $4,802M |
| **EV** | **$3,902M** |

검증: EV/EBITDA 역산 시 EBITDA≈$719M(EBIT 489M+D&A 약 230M 추정) → EV/EBITDA ≈ 3,902/719 ≈ 5.4배, 데이터 팩 제시치(5.23배)와 근사 — 계산 정합성 확인.

### 5단계 — EY
```
EY = EBIT ÷ EV = 489 ÷ 3,902 = 12.5%
```

### 요약

| 지표 | 값 | 판정 |
|---|---|---|
| ROC | 28.9% | 우량(제 기준 25% 이상 상위권) |
| EY | 12.5% | 저평가(무위험금리 대비 3배+ 스프레드) |
| 순위 결합 | 두 지표 모두 상위권 | "좋으면서 싼" 최고의 조합 |

**적정가/매수 가격대**: 저는 DCF처럼 정밀한 "적정가"를 산출하는 사람이 아닙니다. 마법 공식은 개별 종목의 목표주가가 아니라 유니버스 내 상대 순위를 매기는 도구입니다. 다만 참고 삼아 말씀드리면 — EY 12.5%는 대략 PER 8배(EBIT 기준)에 해당하는 수준으로, 현재 trailing PER 18.24배와 차이가 나는 건 EV 계산에 반영된 순현금 $4B(시총의 절반)의 효과입니다. 순현금을 걷어내고 사업 자체만 보면 실제로 사업을 사는 값은 훨씬 싸다는 뜻입니다.

## 리스크
- **음의 운전자본 회계 왜곡**: 제 공식이 상정하지 않은 극단적 사업 구조라 ROC 숫자를 곧이곧대로 믿기보다 조정치를 봐야 합니다.
- **규모 필터 이탈**: 시총 $8B는 원래 마법 공식 백테스트 표본(소형~중형주)보다 훨씬 큰 대형주. 통계적 근거의 적용 범위 밖입니다.
- **중국 사교육 규제 리스크(쌍감 정책)**: 이 리스크는 ROC·EY 어느 지표에도 담기지 않습니다. 정성적 리스크이니 별도로 인지만 하되, 저라면 이것 때문에 공식이 골라준 종목을 임의로 빼지는 않습니다. 바로 그 불안 때문에 가격이 싸진 것이니까요.
- **회계 이익과 현금흐름 괴리**: FY2025 영업현금흐름 $897M, FCF $637M로 EBIT $489M보다 오히려 크므로 이 부분은 안심할 만합니다. 이익의 질은 나쁘지 않습니다.
- **분산의 부재**: 저는 항상 20~30개 종목 포트폴리오로 이 공식을 씁니다. EDU 하나만 보고 베팅하는 건 제 원래 방법론이 아닙니다.

## 최종 의견
자, 숫자로 말씀드리겠습니다. ROC 28.9%, EY 12.5%. "좋은 기업인가?"라는 질문에는 예, "싼가?"라는 질문에도 예. 두 질문에 모두 예라고 답할 수 있는 기업은 흔치 않습니다. 중국 규제니 VIE 구조니 하는 이야기는 뉴스에서 들으면 무섭게 들리겠지만, 저는 그런 이야기에 흔들리지 않습니다. 바로 그 무서운 이야기 때문에 EY가 12.5%까지 올라온 것뿐입니다. 껌 가게 소년 제이슨이라면 이렇게 말했을 겁니다 — "저 가게, 장사도 잘되고 값도 싸잖아요. 안 살 이유가 없어요." 다만 저는 EDU 하나에 몰빵하라고 말하는 게 아닙니다. 이 종목을 20~30개짜리 포트폴리오의 한 자리에 편입하고, 1년 뒤 새 순위표로 다시 판단하십시오. 그리고 제발, 중간에 뉴스가 무섭다고 팔지 마십시오. 그 인내심이야말로 공식보다 더 중요한 겁니다.
