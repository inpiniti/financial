# SKHY 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-21 | 투자의견: 관망 | 확신도: 5/10

## 한 줄 결론
껌 가게로 치면 "동네에서 제일 잘 버는 가게"인 건 확실한데(ROC 68.9%), 지금 가격표가 그 사실을 이미 다 알고 붙인 값(EY 5.93%)이라 서둘러 살 이유가 없다.

## 내 철학에서 본 이 기업
나는 이 회사가 HBM에서 어떤 위치인지, 엔비디아와 협상이 어떻게 되는지 같은 이야기를 놓고 토론하지 않는다. 그런 스토리는 결국 두 개의 숫자로 압축되어야 의미가 있다. "얼마나 잘 버는 회사인가?"(ROC)와 "지금 얼마에 살 수 있는가?"(EY). SK하이닉스는 첫 번째 질문에서 만점에 가까운 답을 낸다. 두 번째 질문에서는 낙제점까지는 아니지만 우등생은 못 된다. 이게 내가 볼 전부다.

## 체크리스트 평가

| 항목 | 평가 | 근거 수치 |
|---|---|---|
| 자본수익률(ROC) 높은가 | ○ | ROC = 68.9% (EBIT 77.37조 ÷ 투입자본 112.31조). 경쟁우위(HBM 점유율 62%, 영업이익률 71.5%)가 숫자로 그대로 찍혀 나옴 |

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

### 1. 투입 유형자본 (분모)

**순운전자본(NWC)** — 26Q1 대차대조표 기준
```
유동자산 106.51조 - 현금성자산 21.17조(무이자성 자산만 남기기 위해 차감) - 유동부채 40.70조
= 44.64조원
```

**순고정자산(Net PP&E)** — *추정치, 근거: 데이터 팩에 PP&E 직접 항목 미제공. 자기자본 164.29조 + 총부채 21.83조 = 총자산 186.12조. 여기서 유동자산 106.51조를 빼면 비유동자산 79.61조. 메모리 제조사는 팹·클린룸·장비 등 자산 중량형 사업이라 비유동자산 중 PP&E 비중이 매우 높다고 보고 85%를 적용.*
```
비유동자산 79.61조 × 0.85 = 67.67조원
```

**투입 유형자본**
```
NWC 44.64조 + 순고정자산 67.67조 = 112.31조원
```

### 2. EBIT (TTM)
```
25Q2 9.21조 + 25Q3 11.38조 + 25Q4 19.17조 + 26Q1 37.61조 = 77.37조원
```

### 3. 자본수익률(ROC)
```
ROC = EBIT ÷ 투입자본 = 77.37조 ÷ 112.31조 = 68.9%
```

### 4. 기업가치(EV)
```
EV = 시가총액 1,303.3조 + 총부채 21.83조 - 현금성자산 21.17조 = 1,303.96조원
```

### 5. 이익수익률(EY)
```
EY = EBIT ÷ EV = 77.37조 ÷ 1,303.96조 = 5.93%
```

| 지표 | 값 | 판정 |
|---|---|---|
| ROC | 68.9% | 최상위권 — "좋은 기업" 확실 |
| EY | 5.93% | 중하위권 — "싼 기업"은 아님 |

마법공식은 개별 종목을 단독으로 사고팔라는 도구가 아니라 전체 유니버스에서 순위를 매기는 도구다. 다만 이 두 숫자만으로도 판단은 명확하다: ROC 68.9%는 반도체 업종은 물론 시장 전체에서도 상위권일 가능성이 높지만, EY 5.93%는 딱히 "헐값"이라 부를 수준이 아니다. 시장이 이미 그 훌륭한 자본수익률을 가격에 반영해 놨다는 뜻이다.

## 리스크
- **EBIT의 지속가능성**: TTM EBIT 77.37조는 메모리 슈퍼사이클 정점 실적을 그대로 반영한다. 사이클이 꺾이면 EBIT이 크게 줄어 ROC와 EY 모두 동시에 악화된다 — 지금의 "싸지 않음"이 "훨씬 더 비쌈"으로 바뀔 수 있다.
- **PP&E 추정의 불확실성**: 순고정자산을 실측이 아닌 추정(비유동자산의 85%)으로 계산했다. 실제 무형자산·투자자산 비중이 크면 ROC는 이보다 더 높게, 작으면 더 낮게 나올 수 있다.
- **희석 리스크**: 해외 교환사채 미전환 753만주(약 6.6조원), 유상증자 1,779만주 — 잠재 희석률 3~4%. 시가총액 기준 EV 계산에는 아직 완전 반영이 안 됐을 수 있다.
- **회계이익 vs 현금흐름**: FCF(TTM 대략 25Q2~26Q1 합산 41.7조 수준)는 EBIT 77.37조 대비 낮다 — 대규모 capex(용인·청주 등 수십조원)가 진행 중이라 실제 주주 몫 현금창출력은 EBIT만큼 화려하지 않다.

## 최종 의견
자, 숫자로 이야기해봅시다. "HBM 1위다", "엔비디아 물량 60~70% 배정받았다" — 다 좋은 이야기입니다. 그런데 저는 스토리를 사지 않습니다. ROC가 얼마고 EY가 얼마입니까? ROC는 68.9%로 나무랄 데 없습니다. 껌 가게 제이슨이 100원 투자해서 69원을 버는 셈이니, 이보다 좋은 사업은 흔치 않습니다.

문제는 가격입니다. EY 5.93%는 "이 사업을 통째로 사면 매년 6%씩 돌려받는다"는 뜻인데, 나쁘진 않지만 제가 찾는 "비이성적으로 낮은 가격"은 절대 아닙니다. 미스터 마켓이 지난 1년반 동안 16배를 올려놓고 이제야 겨우 고점 대비 38% 조정을 준 겁니다. 이건 헐값 세일이 아니라 여전히 프리미엄이 붙은 가격표입니다.

저는 좋은 기업만 비싸게 사지 않습니다. 좋으면서 동시에 싼 기업을 삽니다. SK하이닉스는 전자에는 해당하지만 후자에는 아직 아닙니다. 지금 사는 건 "직관이 공식과 충돌할 때 직관을 따르고 싶은 충동"에 굴복하는 것과 비슷합니다 — HBM 스토리가 매력적이라고 EY를 무시하면 안 됩니다. 조정이 좀 더 진행되어 EY가 두 자릿수에 가까워지거나, 다른 저평가 우량주가 순위표 상단에 더 많이 보일 때 이 종목을 포트폴리오에 편입하겠습니다. 지금은 지켜만 봅니다.
