# 대한조선(439260) 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-21 | 투자의견: 매수 | 확신도: 7/10

## 한 줄 결론
자본수익률(ROC) 약 30%, 이익수익률(EY) 약 30% — 마법 공식이 찾는 "좋으면서 싼 기업"의 교과서적인 조합이다. 다만 조선업 특유의 실적 순환성 때문에 이 숫자가 몇 년 뒤에도 그대로일지는 아무도 장담 못 한다. 그래서 이 종목 하나에 몰빵하지 말고, 20~30개 포트폴리오의 한 자리로 담아라.

## 내 철학에서 본 이 기업
껌 가게 소년 제이슨의 비유로 말하면 이렇습니다. 대한조선은 100원을 투자해서 매년 약 30원을 벌어들이는 사업입니다(ROC 30%). 그리고 지금 시장은 이 사업 전체를 사는 데 100원 벌이당 약 340원만 받으면 되는 가격을 부르고 있습니다(EY 30%는 EBIT/EV의 역수 P/EBIT ≈ 3.4배 수준). 저는 스토리를 보지 않습니다. "탱커 운임이 어떻다", "2대주주가 어떻다" 하는 이야기는 나중에 리스크 항목에서 다루겠지만, 마법 공식의 첫 질문은 딱 두 개입니다 — 좋은 기업인가? 싼가? 대한조선은 둘 다 "그렇다"입니다.

## 밸류에이션 판단 (마법 공식 계산 과정)

### 1) 자본수익률 (ROC = EBIT ÷ 투입 유형자본)

**EBIT(ttm, 최근 4개 분기 합산)**
| 분기 | 영업이익(억원) |
|---|---|
| 25.2Q | 625 |
| 25.3Q | 665 |
| 25.4Q | 953 |
| 26.1Q | 826 |
| **합계(ttm EBIT)** | **3,069** |

**투입 유형자본 = 순운전자본(현금 제외) + 순고정자산**

순운전자본(현금 제외, 26.1Q 기준):
```
유동자산(12,190) - 현금(6,023) = 6,167억 (현금 제외 유동자산)
6,167 - 유동부채(4,622) = 1,545억  ← 순운전자본(NWC, 현금 제외)
```

순고정자산(비유동자산, 총자산 - 유동자산으로 역산):
```
데이터팩에 총자산이 직접 주어지지 않아, ROA(ttm) 12.8% = 순이익(ttm) ÷ 총자산 식으로 역산
순이익(ttm) = 437+658+787+774 = 2,656억
총자산 ≈ 2,656 ÷ 0.128 ≈ 20,750억
비유동자산(≈순고정자산 근사치) = 20,750 - 12,190(유동자산) ≈ 8,560억
```
(주의: PP&E 세부 공시가 없어 "총자산 - 유동자산"을 순고정자산의 근사치로 사용. 투자자산·이연법인세 등이 섞여 다소 과대추정 가능성 있음 — 보수적으로 봐야 함)

```
투입 유형자본 = 1,545(NWC) + 8,560(순고정자산 근사) ≈ 10,105억

ROC = EBIT(3,069) ÷ 투입 유형자본(10,105) ≈ 30.4%
```

교차검증(FY2025 연간 기준): EBIT 2,941 ÷ 10,105 ≈ 29.1% — ttm과 큰 괴리 없음.

### 2) 이익수익률 (EY = EBIT ÷ EV)

```
시가총액 = 16,400억 (1.64조원)
+ 이자부 부채(26.1Q) = 3억
- 현금(26.1Q) = 6,023억
= EV ≈ 10,380억
```

```
EY = EBIT(ttm 3,069) ÷ EV(10,380) ≈ 29.6%
```

교차검증(FY2025 EBIT 기준): 2,941 ÷ 10,380 ≈ 28.3%.

|---|---|---|---|

두 지표 다 "상위권"에 동시에 걸리는 종목은 흔치 않습니다. 실제로 EV/EBITDA 3.25배는 동종 대형 3사(13~19배)의 4분의 1 수준이면서, 영업이익률(26.8%)은 오히려 업계 최고입니다. "싸구려라서 싼 게 아니라, 좋은데 시장이 겁먹어서 싸진" 경우에 가깝습니다.

**적정가 산출 관련**: 마법 공식은 개별 종목의 목표주가를 산출하는 도구가 아니라 유니버스 전체를 ROC·EY로 순위 매겨 상위권을 기계적으로 사는 스크리닝 도구입니다. 따라서 "적정 주가"는 산출하지 않습니다. 다만 참고로, EY가 무위험금리(3~4%대)의 7~10배에 달하는 지금 수준은 절대적 저평가 구간이라는 점만 짚어둡니다.

## 리스크
- **일시적 요인에 의한 지표 왜곡 가능성**: 조선업은 수주 시점 선가가 나중에 매출로 인식되는 구조입니다. 지금의 고ROC·고EY는 신조선가지수가 2008년 역대 최고점 근접·상회하는 "사이클 정점" 구간의 결과물일 수 있습니다(가치 드라이버 #1). 사이클이 꺾이면 2~3년 뒤 마진이 훼손되고, 그때는 지금 계산한 ROC·EY가 무의미해집니다.
- **표본 부족**: 상장 1년 미만, 분기 데이터 5개뿐. 그린블라트가 강조하는 "3~5년 반복" 검증이 불가능한 신규 상장주라는 점은 감안해야 합니다.
- **오버행 리스크**: 2대주주(안다H자산, 지분 24.89%)의 근질권 계약 이슈로 주가가 고점(116,000원) 대비 -63% 하락. 이는 공식이 포착하지 못하는 지분구조 리스크입니다. 다만 계약 만기가 2027-10-01로 연장되어 즉각적 블록딜 우려는 완화된 상태입니다.
- **개별 종목 집중 리스크**: 마법 공식의 전제는 20~30개 분산입니다. 이 종목 하나로 승부를 보면 조선업 사이클 반전 시 큰 손실을 볼 수 있습니다.

## 최종 의견
숫자만 놓고 보면 이보다 더 좋을 수 없습니다. ROC 30%, EY 30% — 저는 이걸 "좋으면서 동시에 싼" 교과서 사례라고 부릅니다. 애널리스트들이 "탱커 운임이 어떻다", "오버행이 어떻다" 하는 이야기를 늘어놓을 때, 저는 늘 되묻습니다 — "그래서 ROC는 얼마고 EY는 얼마입니까?" 답은 이미 위에 다 계산해 놨습니다.

단, 한 가지는 분명히 해둡시다. 이 종목이 마법 공식에 걸린 이유가 진짜 구조적 경쟁우위(고선가 수주잔고, 셔틀탱커 등 고부가 제품믹스) 때문인지, 아니면 조선업 사이클의 일시적 정점 때문인지는 지금 이 순간 아무도 100% 확신할 수 없습니다. 그래서 저는 이 종목을 포트폴리오 전체를 걸 자리가 아니라, 20~30개 종목 중 하나로 담으라고 말씀드립니다. 공식이 2~3년 시장에 뒤처져도 포기하지 않듯이, 개별 종목의 스토리에 흔들리지 말고 숫자가 이끄는 대로 사고, 1년 뒤 순위가 바뀌면 다시 평가하십시오.
