# 리노공업(058470) 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)

## 한 줄 결론

제이슨의 껌 가게로 치면, 리노공업은 동네에서 마진을 가장 잘 남기는 가게(ROC ~30%)가 맞습니다. 하지만 지금 그 가게를 사려면 벌어들이는 돈의 28배(EY 3.5%대)를 내야 합니다. 좋은 기업이지만, 지금은 싸지 않습니다.

## 내 철학에서 본 이 기업

나는 스토리를 안 믿습니다. "2나노 WMCM", "로봇 테마", "테스트 핀 ASP 상승" — 다 좋은 이야기입니다. 하지만 저는 두 숫자만 봅니다. 얼마나 잘 버는가(ROC), 얼마나 싸게 살 수 있는가(EY). 리노공업은 첫 번째 질문엔 만점에 가깝게 답합니다. 두 번째 질문엔 낙제점에 가깝게 답합니다. 이게 제가 이 기업을 보는 방식의 전부입니다.

## 밸류에이션 판단 (마법공식 계산 과정)

### 1) 자본수익률 (ROC = EBIT ÷ 투입 유형자본)

데이터 팩에는 세부 유형자산(PP&E) 항목이 없어, 두 가지 방식으로 교차 계산했습니다. 정밀치가 아니라 근사치임을 먼저 밝힙니다.

**EBIT**: TTM 영업이익 1,893.6억원 (무차입 경영이라 세전영업이익과 영업이익이 사실상 동일)

**방법 A: 순운전자본(NWC) + 순고정자산(NFA)**

| 항목 | 값(억원) | 산출 |
|---|---|---|
| 유동자산 | 5,624.7 | 2026-03-31 |
| (−) 초과현금 | 852.0 | 인수자 관점에서 사업 운영에 불필요한 현금 제외 |
| (−) 유동부채 | 1,149.8 | |
| **NWC** | **3,622.9** | 5,624.7 − 852.0 − 1,149.8 |
| 순고정자산(NFA, 추정) | 약 2,783.8 | 총자산 근사치(자기자본 7,108.7 + 유동부채 1,149.8 + 소액 비유동부채 가정) − 유동자산 5,624.7 (PP&E 원자료 미확보, 잔여법 근사) |
| **투입자본(NWC+NFA)** | **약 6,406.7** | |

ROC(방법 A) = 1,893.6 / 6,406.7 ≈ **29.6%**

**방법 B (교차검증): 자기자본 + 이자부채 − 현금**

| 항목 | 값(억원) |
|---|---|
| 자기자본 | 7,108.7 |
| (+) 이자부채 | 1.9 |
| (−) 현금 | 852.0 |
| **투입자본(방법 B)** | **6,258.6** |

ROC(방법 B) = 1,893.6 / 6,258.6 ≈ **30.3%**

→ 두 방법 모두 **ROC ≈ 29~30%**로 수렴합니다. 껌 가게로 치면 100원 투자해서 매년 30원을 버는 셈입니다. 아주 훌륭합니다. 48% 영업이익률(TTM)이 어디서 오는지 이 숫자가 말해줍니다.

### 2) 이익수익률 (EY = EBIT ÷ EV)

| 항목 | 값 |
|---|---|
| 시가총액(2026-07-21) | 5.4조원 = 54,000억원 |
| (+) 이자부채 | 1.9억원 |
| (−) 현금 | 852.0억원 |
| **EV** | **53,149.9억원** |
| EBIT(TTM) | 1,893.6억원 |

EY = 1,893.6 / 53,149.9 ≈ **3.56%**

(FY2025 확정 EBIT 1,770.0억원 기준으로도 3.33% — 어느 쪽으로 계산해도 3%대 초반~중반)

→ 국고채 금리대(대략 2.8~3%대로 추정)를 겨우 웃도는 수준입니다. 인수자가 이 사업 전체를 지금 가격에 사면 회수하는 데 걸리는 시간이 EY 역수로 약 28년입니다. 저는 이런 가격을 "싸다"고 부르지 않습니다.

### 3) 순위 결합 판단

- ROC 축: 상위권 (좋은 기업)
- EY 축: 하위권 (비싼 가격)
- 합산 원리상, 한 지표가 압도적이어도 다른 지표가 바닥이면 상위 20~30개 안에 들기 어렵습니다. 이 기업은 정확히 "회피" 카테고리 중 하나 — **좋지만 너무 비싼 기업**에 해당합니다.

## 리스크

- **제 방식의 리스크**: 이건 "이 회사가 나쁘다"는 판단이 아닙니다. 사업 자체는 제가 좋아하는 유형입니다(높은 ROC, 무차입, 소모품 반복매출). 문제는 순전히 가격입니다.
- **투입자본 계산의 한계**: PP&E 원자료를 확보하지 못해 NFA를 잔여법으로 근사했습니다. 정밀치와 다를 수 있으나, 두 방법(A/B)이 29~30%로 수렴하므로 방향성은 신뢰할 만합니다.
- **회계적 왜곡 가능성**: 최근 최대주주 블록딜(8,600억원 규모)은 EBIT·투입자본 계산에 직접 영향을 주지 않지만, 지배구조 불확실성이 시장의 EY 재평가(디레이팅)를 이미 상당 부분 반영한 상태일 수 있습니다.
- 개별 종목으로 매수/매도를 논하는 것 자체가 제 원래 방법론(20~30개 분산, 1년 보유, 3~5년 반복)과는 맞지 않다는 점도 밝혀둡니다. 이 보고서는 어디까지나 두 지표로 본 스냅샷입니다.

## 최종 의견

여러분, 제이슨의 껌 가게를 다시 떠올려보세요. 100원 투자해서 30원 버는 가게, 정말 훌륭합니다. 그런데 그 가게를 사겠다고 EY 3.5%짜리 가격을 부른다면, 저는 손사래를 칩니다. 좋은 사업과 좋은 투자는 다른 이야기입니다. 마법 공식은 둘 다 확인하라고 말합니다. ROC는 통과, EY는 낙제입니다. 지금 이 가격에 사는 것은 제 원칙에 맞지 않습니다. 가격이 4만원대로 내려오거나, 이익이 지금보다 한참 더 커져서 EY가 자연스럽게 올라올 때까지 저는 지켜만 봅니다. "공식이 항상 통한다면 더 이상 통하지 않을 것이다" — 지금 이 종목이 인기 테마(AI·로봇·HBM)로 뜨거운 것도 같은 이유에서 경계 신호입니다.
