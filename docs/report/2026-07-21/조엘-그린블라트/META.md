# META 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-21 | 투자의견: 보유 | 확신도: 6/10

## 한 줄 결론
META는 마법 공식의 두 지표 중 자본수익률(ROC)에서는 최상급이지만, 이익수익률(EY)은 "거저 줍는" 수준에 못 미치는 중위권이다 — 좋은 기업이지만 아주 싼 기업은 아니다.

## 내 철학에서 본 이 기업
저는 스토리를 보지 않습니다. 두 숫자만 봅니다. "이 사업체는 얼마나 돈을 잘 버는가?"(ROC)와 "그 사업체를 지금 얼마에 살 수 있는가?"(EY). META의 광고 사업은 유형자산을 거의 쓰지 않고도(순운전자본과 PP&E net 합쳐 $243B) 세전영업이익 $83B를 뽑아내는 압도적으로 효율적인 사업입니다. 이것은 경쟁우위의 명백한 신호입니다. 문제는 가격입니다. 시가총액 $1.639조까지 오른 지금, 이 훌륭한 사업을 인수자 입장에서 사면 세전 기준 연 5%대의 수익률밖에 나오지 않습니다. 좋은 사업 + 적당한 가격 조합이지, 좋은 사업 + 헐값 조합은 아닙니다.

## 체크리스트 평가

1. **자본수익률(ROC) 계산**
   - ROC = EBIT ÷ (순운전자본 + 순고정자산) = $83,276M ÷ $243,286M = **34.2%** → ○ (최상급, 미국 대형주 평균의 2배 이상)
2. **이익수익률(EY) 계산**
   - EY = EBIT ÷ EV = $83,276M ÷ 약 $1.616~1.641조 = **5.1~5.2%** → △ (무위험금리 대비는 우위지만 마법공식 상위권 기준(통상 8%+)에는 못 미침)
3. **순위 결합 관점**
   - ROC는 유니버스 최상위권 추정, EY는 중위권 추정 → 합산 순위는 "우수하지만 최고 기회는 아닌" 구간 → △
4. **규모 필터**
   - 시가총액 $1.6조+ → 대형주, 필터 통과 → ○
5. **포트폴리오 관점**
   - 개별 종목 스토리(AI capex 논쟁, Reality Labs 손실, 반독점 소송)는 제 판단에 개입시키지 않습니다. 그것이 저의 원칙입니다 → 참고만

## 밸류에이션 판단 (마법공식 계산 과정)

### 원자료 (FY2025, 2025-12-31 기준, 단위 $M)
| 항목 | 값 |
|---|---|
| EBIT (Income from operations) | 83,276 |
| 시가총액 (2026-07-20 종가 기준) | 1,639,000 |
| 유동자산 | 108,722 |
| 유동부채 | 41,836 |
| 순운전자본(NWC) = 유동자산 − 유동부채 | 66,886 |
| 순고정자산(PP&E net) | 176,400 |
| 장기부채 | 58,744 |
| 운용리스부채(유동+비유동) | 25,153 |
| 현금+시장성증권 | 81,592 |

### 자본수익률(ROC)
```
투입 유형자본 = 순운전자본 + 순고정자산
             = 66,886 + 176,400
             = 243,286 ($M)

ROC = EBIT ÷ 투입 유형자본
    = 83,276 ÷ 243,286
    = 34.2%
```

### 이익수익률(EY)
```
EV(표준) = 시가총액 + 장기부채 − 현금성자산
         = 1,639,000 + 58,744 − 81,592
         = 1,616,152 ($M)

EV(리스 포함, 보수적) = EV(표준) + 운용리스부채
                      = 1,616,152 + 25,153
                      = 1,641,305 ($M)

EY(표준) = 83,276 ÷ 1,616,152 = 5.15%
EY(보수적) = 83,276 ÷ 1,641,305 = 5.07%
```

### 마법공식 종합
| 지표 | 값 | 평가 |
|---|---|---|
| ROC | 34.2% | 최상급 (경쟁우위 뚜렷) |
| EY | 5.1~5.2% | 중위권 (염가는 아님) |

EY가 8%(마법공식 상위권 기준)에 도달하려면 EV가 약 $1.04조 수준까지 내려가야 하며, 이는 부채·현금 불변 가정 시 시가총액 약 $1.06조, 주가 약 **$410대**에 해당합니다. 현재가 $645.85 대비 약 -36% 하락 구간입니다. 이것이 제 공식이 "확실한 매수" 신호를 줄 가격대입니다.

## 리스크
- ROC 계산의 분자(EBIT)가 2026년 capex 급증($125-145B)의 감가상각 반영으로 향후 압박받을 수 있습니다. PP&E net이 이미 FY2024 $121B → FY2025 $176B로 45% 증가했고, 이는 분모를 키워 ROC를 자연히 낮출 것입니다.
- 회계 이익과 실제 현금흐름의 괴리: SBC $20.4B가 EBIT에는 비용 반영되지만 현금흐름표에서는 가산되는 항목이라 다소 왜곡 가능성이 있습니다.
- 반독점 항소, 청소년 SNS 소송($1.4조 청구) 등 질적 리스크는 두 지표에 전혀 반영되지 않습니다. 저는 이것을 무시하지 않지만, 공식이 담지 못하는 부분이라는 것은 인정합니다.
- 마법공식은 개별 종목 추천이 아니라 20~30개 분산 포트폴리오로 설계된 방법론입니다. META 한 종목의 결론에 과도한 의미를 부여하면 안 됩니다.

## 최종 의견
자, 숫자로 말해봅시다. ROC 34.2% — 이건 100원 투자해서 34원을 버는 사업입니다. 껌 가게로 치면 동네에서 제일 장사 잘되는 가게죠. 그런데 EY가 5.1%밖에 안 됩니다. 이 훌륭한 가게를 통째로 사는데, 연 수익률이 5%대라는 뜻입니다. 나쁘지 않지만, 제가 찾는 "거저 줍는" 가격은 아닙니다.

역설적으로 들리겠지만, 제가 원하는 건 "좋은 기업"과 "싼 가격"이 동시에 존재하는 순간입니다. META는 전자는 확실히 갖췄지만 후자는 아직 부족합니다. 시장이 겁을 먹고 이 주식을 $400대로 던질 때 — capex 우려든, 소송 리스크든, 그 이유는 중요하지 않습니다 — 그때가 제 공식이 신호를 보내는 순간입니다. 지금은 아닙니다.

저는 이 주식을 사지 않겠다는 게 아니라, 지금 이 가격에서는 "최고의 기회"라고 부르기엔 이르다는 겁니다. 보유자라면 계속 들고 가도 좋습니다. 새로 살 사람이라면 EY가 더 매력적인 수준까지 오길 기다리거나, 20~30개 분산 유니버스 안에서 순위가 더 높은 다른 종목을 먼저 보십시오. 공식은 감정에 흔들리지 않습니다. 저도 그렇습니다.
