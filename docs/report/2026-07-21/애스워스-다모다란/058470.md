# 058470 투자 보고서 — 애스워스 다모다란 (주식 가치평가 완벽 정리)
> 날짜: 2026-07-21 | 투자의견: 매도(비중축소) | 확신도: 6/10

## 한 줄 결론
리노공업은 영업이익률 48%, 무차입, ROE 22%대의 우량 기업이지만, 현재가 69,000원은 제가 세운 낙관 시나리오(주당 약 51,600원)조차 넘어서는 가격입니다. 이 가격을 정당화하려면 향후 5년간 연 30% FCF 성장 또는 8%대의 비현실적으로 낮은 할인율이 필요합니다 — 스토리가 숫자를 따라가지 못하는 국면입니다.

## 내 철학에서 본 이 기업

스토리부터 정리하겠습니다. 리노공업은 반도체 테스트 핀·소켓이라는 좁지만 깊은 소모품 시장에서, 패키징 미세화(핀 수 P·신호밀도 Q 동시 증가)로 인한 ASP 상승과 북미 대형 AP 고객사의 2nm/WMCM 패키징 전환이라는 두 개의 강력한 구조적 순풍을 타고 있습니다. 매출이 FY2023 2,556억 → FY2025 3,725억으로 반등했고, TTM 영업이익률은 48.1%에 달합니다. 사실상 무차입(이자부채 1.9억)으로 재무 리스크는 거의 없습니다.

하지만 이것은 "얼마나 좋은 기업인가"에 대한 스토리이지, "이 가격이 정당한가"에 대한 답은 아닙니다. 저는 이 둘을 분리해서 봅니다. 좋은 기업과 좋은 투자는 다른 질문입니다.

## 밸류에이션 판단 (계산 과정)

### A. 2단계 DCF — 낙관/기본/비관 시나리오

**공통 가정**
- 기준연도 FCF(FCF₀): 1,200억원 (FY2025 실적 1,192.3억원 + TTM 추세 반영, 반올림)
- 순현금: 약 1,530억원(현금 852억 - 부채 1.1억 기준, 데이터 팩 10절) → 영업가치에 가산
- 고성장 기간: 5년, 이후 영구성장(터미널)
- 발행주식수: 75,895,600주

| 시나리오 | 연도별 FCF 성장률(Y1~Y5) | 터미널 성장률 | 할인율(자기자본비용) |
|---|---|---|---|
| 비관 | 8% → 6% → 5% → 4% → 3% | 2.5% | 11% |
| 기본 | 18% → 15% → 12% → 9% → 6% | 3.0% | 10% |
| 낙관 | 25% → 22% → 18% → 14% → 10% | 3.5% | 9.5% |

*할인율 차등의 근거: 비관 시나리오는 단일 대형 고객사 이탈·지배구조 불확실성이 현실화된 상황을 반영해 위험프리미엄을 높였고, 낙관 시나리오는 다변화(비메모리·로봇)가 성공해 위험이 낮아진 상황을 가정했습니다.*

**기본 시나리오 계산 과정 (단위: 억원)**

| 연도 | 성장률 | FCF | 할인계수(10%) | PV |
|---|---|---|---|---|
| Y1 | 18% | 1,416.0 | 0.9091 | 1,287.3 |
| Y2 | 15% | 1,628.4 | 0.8264 | 1,345.6 |
| Y3 | 12% | 1,823.8 | 0.7513 | 1,370.4 |
| Y4 | 9% | 1,987.9 | 0.6830 | 1,357.7 |
| Y5 | 6% | 2,107.2 | 0.6209 | 1,308.5 |
| **PV(FCF) 합계** | | | | **6,669.5** |
| 터미널가치 = FCF₅×(1.03)/(0.10-0.03) = 2,170.4/0.07 | | | | **31,005.7** |
| PV(터미널가치) = 31,005.7×0.6209 | | | | **19,255.6** |
| **영업가치 합계** | | | | **25,925.1** |
| +순현금 | | | | **1,530.0** |
| **자기자본 가치** | | | | **27,455.1억원** |
| ÷ 75,895,600주 | | | | **≈ 36,177원/주** |

**비관 시나리오 요약**: FCF₅ = 1,545.3억 → 터미널가치 18,634.1억 → PV 합계(FCF+TV) 16,302.0억 + 순현금 1,530억 = **17,832.0억원 → 주당 약 23,494원**

**낙관 시나리오 요약**: FCF₅ = 2,707.9억 → 터미널가치 46,712.3억 → PV 합계 37,645.1억 + 순현금 1,530억 = **39,175.1억원 → 주당 약 51,617원**

### DCF 시나리오 요약

| 시나리오 | 주당 가치 | 현재가(69,000원) 대비 |
|---|---|---|
| 비관 | 23,494원 | -66% (고평가) |
| 기본 | 36,177원 | -48% (고평가) |
| 낙관 | 51,617원 | -25% (고평가) |

제 낙관 시나리오조차 현재가를 25% 밑돕니다. "대략적으로 맞는 것이 정확하게 틀리는 것보다 낫다"는 원칙을 적용해도, 세 시나리오 모두 같은 방향(고평가)을 가리킨다는 점이 중요합니다.

### B. 역산 DCF — 현재가(69,000원)를 정당화하는 조건

목표 자기자본 가치 = 69,000원 × 75,895,600주 = **52,368억원**, 영업가치 필요분 = 52,368 - 1,530(순현금) = **50,838억원**

**① 할인율 10% 고정 시, 필요한 5년 FCF 성장률은?**

| 시도 성장률(5년 균등) | Y5 FCF | 터미널가치 | PV(FCF)+PV(TV) | 자기자본가치 | 주당가치 |
|---|---|---|---|---|---|
| 20% | 2,986.0억 | 43,937.1억 | 35,137.5억 | 36,667.5억 | 48,314원 |
| **30%** | 4,455.5억 | 65,560.0억 | 50,897.6억 | 52,427.6억 | **69,079원** ✓ |

→ **할인율 10%를 고정하면, 향후 5년간 연평균 약 30%의 FCF 성장(그 후 3% 영구성장으로 급격히 수렴)이 있어야 현재가가 정당화됩니다.** 데이터 팩상 최근 4개 분기 순이익 성장률(YoY +27.9%~+37.7%)이 근접한 수준이긴 하나, 이것이 "5년간" 지속된다고 가정하는 것은 제가 항상 경계하는 실수 — 현재의 고성장이 영원하다는 가정 — 그 자체입니다.

**② 성장률을 컨센서스(EPS 성장률 21.1%, 5년 균등 적용, 터미널 3%)로 고정 시, 필요한 할인율은?**

| 시도 할인율 | 자기자본가치 | 주당가치 |
|---|---|---|
| 8% | 53,715.5억 | 70,777원 |
| ~8.2%(보간) | ≈ 52,400억 | ≈ 69,000원 ✓ |

→ **컨센서스 성장률(21%)을 그대로 받아들이면, 현재가를 정당화하기 위해 필요한 할인율은 약 8.2%입니다.** 이는 무위험이자율(한국 10년물 약 3.5%)에 리스크프리미엄 4.7%p만 얹은 수준으로, 유동주식비율 13.2%·최대주주 대량매각·단일 대형 고객사 의존이라는 리스크를 가진 코스닥 중형주에 적용하기에는 지나치게 낮습니다. 저라면 이 기업에 9.5~11%의 할인율을 적용하겠습니다.

**결론**: 두 역산 경로 모두 "시장이 매우 공격적인 가정(초고성장 지속 또는 초저위험)을 가격에 반영하고 있다"는 동일한 결론에 도달합니다.

### C. 상대가치 교차검증
- PER(TTM) 32.1배, PEG(추정) 약 1.1배 — 린치의 기준(PEG<1 저평가)으로 보면 아슬아슬하게 합격선이지만, 이 지표는 위험을 무시합니다. 유동주식비율이 낮고 지배구조 불확실성이 있는 기업에 성장주와 동일한 PEG 잣대를 대는 것은 위험합니다.
- EV/Sales 13.1배는 동종 컴프(TSE·ISC·오킨스전자) 대비 현저히 높은 편은 아니나, 하드웨어 소모품 기업치고는 절대적으로 높은 수준입니다.
- 목표주가 평균 121,154원은 제 낙관 DCF(51,617원)의 약 2.3배 — 애널리스트 컨센서스가 반영한 성장 가정이 제가 산정한 낙관 시나리오보다도 훨씬 공격적임을 시사합니다.

## 리스크
- **추정 오차(가장 큰 리스크)**: 제 FCF₀ 1,200억원, 성장률, 할인율 모두 추정치입니다. 이 추정이 틀릴 가능성을 항상 전제해야 합니다.
- **단일 대형 고객사 의존**: 북미 AP 고객사의 2nm/WMCM 전환 지연·취소 시 낙관 시나리오의 근거가 무너집니다.
- **지배구조**: 최대주주 지분 34.66%→25.48%로 축소(2026-06 블록딜), "경영권 매각설" 재부상 — 향후 지배구조 변화는 할인율 상승 요인입니다.
- **유동성**: 유동주식비율 13.2%로 가격 변동성이 확대되기 쉬운 구조 — 가격과 가치의 괴리가 더 커질 수도, 더 빨리 좁혀질 수도 있습니다.
- **노사 리스크**: 파업 찬반투표 찬성률 92.6%, 단기 실적 변동 요인.

## 최종 의견

제 가치평가는 거의 확실히 틀렸을 겁니다. FCF₀ 추정, 성장 경로, 할인율 — 어느 하나도 정밀하지 않습니다. 하지만 비관·기본·낙관 세 시나리오가 전부 현재가 69,000원 아래에 위치하고, 역산 DCF 역시 "5년간 연 30% 성장" 또는 "8%대 할인율"이라는, 이 기업의 위험 프로필에 비해 지나치게 낙관적인 가정을 요구합니다. 이것은 하나의 잘못된 숫자가 아니라 일관된 방향입니다.

리노공업은 좋은 기업입니다. 그러나 좋은 기업과 좋은 가격에 사는 것은 다른 문제입니다. 지금 이 가격은, 시장이 그린 스토리가 제가 계산할 수 있는 숫자보다 앞서 나가 있다는 신호입니다. 원하는 결론을 위해 할인율이나 성장률을 억지로 끌어올리고 싶은 유혹이 있겠지만, 그것은 가치평가가 아니라 숫자 맞추기입니다. 저는 이 가격에서는 비중을 줄이거나 신규 매수를 보류하겠습니다.
