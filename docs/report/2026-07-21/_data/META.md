## 13. 업종 특화 지표 — 해당 없음
META는 은행/리츠/보험/바이오텍이 아니며, SaaS 지표(ARR/NRR 등)도 광고 기반 비즈니스 특성상 공개되지 않아 해당 필드 생략. 핵심 운영지표(DAU/MAU, 광고단가)는 확인 불가(이번 조회 범위에서 미수집).
