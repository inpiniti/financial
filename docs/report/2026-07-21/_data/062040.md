# 산일전기 (062040.KS) 데이터 팩 — 기준일 2026-07-21

## 0. 기업 개요
- **사업모델**: 산업용 특수변압기(전력용·배전용·리액터 등) 제조. 1987년 설립, 안산 소재. 데이터센터·신재생에너지(BESS)·전력망 교체 수요향 초고압/특수변압기 비중 확대 중. 수출비중 약 97%, 그중 미국향이 75% 이상(2025년 기준).
- **주요 제품**: Power/Distribution/Pad-mounted/Oil-filled/Cast-resin/Pole-mounted/Dry-type 변압기, 고·저압 소프트스타터.
- **경쟁사(전력기기 밸류체인)**: LS ELECTRIC(010120), HD현대일렉트릭(267260), 효성중공업(298040), 일진전기(103590), 제룡전기(033100) — 8번 컴프 참조.
- **주가**: 종가 156,500원(-2.25%, 사용자 제공, 2026-07-21) / Yahoo 전일 종가 160,100원(2026-07-20). 시총 약 4.78조원(발행주식 30,555,097주 × 156,500원 기준).
