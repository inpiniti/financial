# SKHY (SK hynix Inc. ADR) 데이터 팩

**조회일: 2026-07-21** | 통화: ADR은 USD, 원주는 KRW 표기 명시
**데이터 소스 안내**: SKHY ADR은 2026-07-13 나스닥 신규상장(임시티커 SKHYV → SKHY 전환)으로 유동성·이력이 짧아, **재무·밸류에이션·컨센서스는 원주(000660.KS, KRX) 기준으로 대체 수집**하고 그 사실을 각 표에 별도 표기하지 않는 한 원주 기준임. ADR 고유 정보(전환비율, 상장가, 환헤지 등)만 별도 표기.

## 기업 개요
- **사업 모델**: 메모리 반도체(DRAM/NAND) 설계·제조·판매. HBM(고대역폭메모리)이 AI 서버향 핵심 성장동력.
- **주요 세그먼트**: DRAM(서버·모바일·PC·그래픽), NAND/SSD, MCP, 파운드리(비메모리 일부)
- **경쟁사**: 삼성전자(005930.KS), Micron(MU), (HBM 한정) 삼성전자·Micron 3파전
- **GICS 섹터**: Technology > Semiconductors
