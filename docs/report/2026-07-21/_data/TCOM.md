## 1. 기업 개요
- 사업 모델: 중국 최대 온라인여행플랫폼(OTA). 숙박예약, 교통(항공권·기차), 패키지투어, 기업여행(Corporate Travel) 4대 세그먼트. 해외 플랫폼 Trip.com/Skyscanner로 인바운드·아웃바운드 국제여행 확장 중.
- 경쟁사: Meituan(호텔 예약 1위, 반값 경쟁), Tongcheng Travel(0780.HK, Tencent 계열), Alibaba Fliggy, Booking Holdings(BKNG)·Expedia(EXPE, 글로벌 OTA 비교군).
- 시장 지위: 중국 아웃바운드 여행 플랫폼 점유율 약 54.7%(Fliggy 27.1%, Tongcheng 11.5% 대비 압도적 1위) — 단, 국내 호텔 예약은 Meituan에 밀림.
- 본사: 싱가포르(경영) / 상하이 기반. ADR 1주 = 보통주 1주(참고, 비율 별도 확인 필요).
