# TSM 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-21 | 투자의견: 관망 (조건부 매수 대기) | 확신도: 6/10

## 한 줄 결론
펀더멘털은 마켓 위저드들이 가장 좋아하는 그림(사상 최대 실적, 40%대 가이던스 상향, 압도적 가격결정력)이지만, 기술적 신호는 정반대(50일선·200일선 이탈, MACD 음전환)를 가리키고 있다 — 마커스의 원칙대로 "둘이 같은 방향을 가리킬 때"까지는 정찰만 하고 실탄은 아낀다. 지금 진입한다면 그것은 분석이 아니라 "너무 좋은 회사라서 놓치기 싫다"는 감정이다.

## 내 철학에서 본 이 기업
내가 인터뷰한 열여섯 명의 위저드 중 누구도 "이 회사가 좋으니까 산다"고 말하지 않았다. 그들은 하나같이 "지금이 살 때인가, 그리고 틀렸을 때 어디서 나갈 것인가"를 먼저 물었다. TSM의 스토리는 매혹적이다 — 2Q2026 순이익 TWD 706.6B(+77% YoY), EPS $4.31로 컨센서스 대비 +10.9% 서프라이즈, HPC 비중 66%, 총마진 67.7%(장기 목표 53%를 15%p 이상 상회). 그런데 이 뉴스가 나온 다음 날(2026-07-17) 주가는 -3% 빠졌고, 이후 7월 내내 밀려 현재 $402.30은 50일선($448.09)과 200일선($430.79)을 모두 하회했다. 7월 13일 50일선 이탈, 7월 2일 MACD 음전환 — 이건 "재료 소진 후 추세 전환"의 교과서적 패턴이다. 오닐이라면 이렇게 물었을 것이다. "시장(반도체 섹터)의 방향이 지금 상승인가, 개별 종목이 시장을 거스르고 있는가?" 6/24 하루 -6.62% 급락(섹터 전반 조정+관세 리스크)까지 겹쳤다는 건 개별 이슈가 아니라 섹터 전체가 흔들리고 있다는 신호다.

## 밸류에이션 판단 — 트레이더 관점 계산 (산출 안 함 항목 제외)

내 철학은 "적정가"를 계산하지 않는다. 대신 진입가·손절가·목표가·손익비·포지션 사이징을 계산한다. 이것이 내가 인터뷰한 위저드들의 공통 언어다.

### 1) 현재 가격 구조

| 지표 | 값 |
|---|---:|
| 현재가 | $402.30 |
| 52주 고점 | $479.00 |
| 52주 저점 | $223.70 |
| 50일 이동평균 | $448.09 |
| 200일 이동평균 | $430.79 |
| 최근 지지구간 | $402.47 ~ $419.82 |
| 저항(50일선 회복 지점) | $448 부근 |

현재가($402.30)는 위 지지구간($402.47~419.82) **하단을 이미 살짝 이탈**한 상태 — 지지가 지지로 작동하는지 아직 확인 안 됨.

### 2) 시나리오 A — 지금은 관망, 추세 재확인 후 진입 (기본 시나리오)

와인스타인의 원칙: "조건이 정렬되지 않으면 아무리 매력적이어도 기다린다." 진입 트리거는 두 가지 중 하나:

**트리거 1 (돌파 진입)**: 종가 기준 50일선($448) 재돌파 + 거래량 동반 시 → 추세 복귀 확인 후 진입
- 진입가: $450 (50일선 회복 확인 후 소폭 프리미엄)
- 손절가: $430 (200일선 하회 재발 = 추세 재붕괴 신호), 손실폭 = ($450-$430)/$450 = **-4.4%**
- 1차 목표가: 애널리스트 평균 목표주가 $522.82 (52주 고점 $479 상회하는 신고가 영역)
  - 수익폭 = ($522.82-$450)/$450 = **+16.2%**
- 손익비 = 16.2% / 4.4% = **약 3.7:1**

**트리거 2 (눌림목 진입, 더 보수적)**: 지지구간($402~420) 재확인 반등(더블바텀 패턴) 시
- 진입가: $415 (지지 반등 확인 종가)
- 손절가: $395 (52주 저점 방향 이탈 아님, 지지 붕괴 확정선), 손실폭 = ($415-$395)/$415 = **-4.8%**
- 1차 목표가: 50일선 $448 (단기), 손익폭 = ($448-$415)/$415 = **+8.0%** → 손익비 **1.7:1** (단기 목표)
- 2차 목표가: $522.82 (애널리스트 평균), 손익폭 = ($522.82-$415)/$415 = **+26.0%** → 손익비 **5.4:1** (연장 목표, 데니스식 "맞으면 길게 끌고 간다")

### 3) 포지션 사이징 (하이트의 1% 룰 적용)

가정: 계좌 자본 $100,000, 단일 거래 최대 손실 한도 = 자본의 1.5%(TSM처럼 펀더멘털이 강한 종목은 위저드들도 표준 밴드 1~2% 상단을 쓰는 경우가 많음) = **$1,500**

**트리거 1 기준**:
- 주당 손절폭 = $450 - $430 = $20
- 최대 매수 주식수 = $1,500 / $20 = **75주**
- 포지션 규모 = 75주 × $450 = **$33,750** (계좌의 33.75% — ADR 변동성 대비 다소 크므로, 베타 1.246 감안해 **50주(포지션 $22,500, 계좌의 22.5%)로 축소 권고**, 리스크는 $1,000로 낮아짐(자본의 1.0%))

**트리거 2 기준**:
- 주당 손절폭 = $415 - $395 = $20
- 최대 매수 주식수 = $1,500 / $20 = **75주** (동일 로직 적용, 50~75주 범위)

### 4) 지금 당장 매수한다면? (비교용, 권장하지 않음)

- 진입가: $402.30 (현재가)
- 손절가: $395 (최근 지지 이탈 확정선, 52주 저점과는 거리 있음), 손실폭 = **-1.8%**
- 목표가: $522.82, 수익폭 = **+30.0%**
- 손익비 = 30.0/1.8 = **16.7:1** (숫자만 보면 매우 매력적)

이 숫자가 가장 좋아 보이는 이유가 바로 함정이다 — 추세가 아직 하락 전환 중인 상태에서 "숫자가 좋아 보인다"는 이유로 진입하는 것은 추세를 거스르는 배팅이다. 오닐이라면 "떨어지는 칼날을 잡지 마라"고 했을 것이다. 손절가 $395도 최근 며칠 새 두 번 근접했던 가격대라 손절 폭이 좁아 보이지만 실제로는 노이즈에 걸릴 확률이 높다.

## 리스크

- **추세 리스크**: 50일선·200일선 동반 이탈, MACD 음전환 — 기술적으로는 하락 국면 초입일 가능성.
- **섹터 리스크**: 6/24 반도체 섹터 전체 -6.62% 급락 — 개별 기업 문제가 아니라 섹터 밸류에이션 조정 국면일 수 있음.
- **지정학 리스크(테일 리스크)**: 대만해협 긴장, 미국 관세(최대 100% 시사), ITC 특허소송 예비판결 — 이건 손절로 방어가 안 되는 갭 리스크. 튜더 존스라면 이런 이벤트 리스크에 대해 포지션 자체를 줄이라고 했을 것이다.
- **재료 소진 패턴**: 사상 최대 실적에도 주가 하락 — "좋은 뉴스에 파는" 전형적 패턴이 반복되면 단기 반등도 약할 수 있음.
- **밸류에이션 갭**: 애널리스트 목표주가 스프레드가 $354~$700로 극단적으로 넓음 — 이는 시장 참여자들 사이에 방향성 합의가 없다는 뜻이고, 이럴 때는 가격이 스스로 방향을 알려줄 때까지 기다리는 것이 낫다.

## 최종 의견

내가 인터뷰한 마이클 마커스라면 이 차트를 보고 이렇게 말했을 겁니다. "펀더멘털이 아무리 좋아도, 기술적으로 추세가 안 서 있으면 나는 기다린다." 지금 TSM은 정확히 그 상황입니다 — 실적은 역사상 최고인데 주가는 50일선도, 200일선도 깨고 내려와 있습니다. 폴 튜더 존스식으로 물어보죠. "이 포지션에서 얼마를 잃을 수 있는가?" 지금 당장 사면 숫자상 손익비는 근사하게 나오지만(16.7:1), 그건 추세를 거스르는 배팅에 그럴듯한 숫자를 붙인 것에 불과합니다. 희망은 전략이 아닙니다. 나라면 50일선 재돌파($448 부근)를 확인하고 들어가거나, 지지구간($402~420)에서 명확한 반등 캔들을 보고 들어갑니다. 그때까지는 기다리는 것도 유효한 포지션입니다. 그리고 들어갈 때는 반드시 —주식수 계산 먼저, 손절가 먼저, 목표가는 그 다음입니다.
