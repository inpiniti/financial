# EDU 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-17 | 투자의견: 관망 | 확신도: 5/10

## 한 줄 결론
제가 인터뷰한 트레이더들이라면 이렇게 물었을 겁니다 — "추세가 아직 확인되지 않았는데 왜 지금 들어가려 하는가?" EDU는 펀더멘털은 좋아졌지만 가격은 아직 그 사실을 증명하지 않았습니다. 200일선을 돌파하는 순간까지 기다리는 것도 하나의 포지션입니다.

## 내 철학에서 본 이 기업
저는 이 기업이 좋은 회사인지 나쁜 회사인지를 판단하는 사람이 아닙니다. 제 관심은 늘 하나입니다 — "지금 이 포지션에 들어가면 얼마를 잃을 수 있고, 그 손실을 감당할 만한 손익비가 설계되는가?"

데이터를 보면 흥미로운 괴리가 있습니다. FY26Q3 실적은 매출 +19.8% YoY, 영업이익 +44.8% YoY, EPS 서프라이즈 +12.6%, 가이던스 상향까지 나왔습니다(펀더멘털은 명백히 개선). 그런데 주가는 52주 고점($64.97) 대비 -23.7%인 $49.59에 머물러 있고, 200일 이동평균($53.93)도 하회 중입니다(기술적으로는 여전히 하락 추세). 마이클 마커스라면 이 상황에서 절대 매수 확신을 갖지 않았을 겁니다 — 그는 펀더멘털과 기술적 신호가 "같은 방향"을 가리킬 때만 크게 베팅했습니다. 지금 EDU는 펀더멘털은 청신호, 기술적 신호는 아직 적신호입니다.

## 밸류에이션 판단 (계산 과정 명시)

저는 내재가치를 계산하지 않습니다. 제가 계산하는 것은 진입·손절·목표·손익비·사이징입니다.

### 추세 진단
```
현재가 $49.59
50일 이동평균 $48.19 (현재가 하회, 근접)
200일 이동평균 $53.93 (현재가 상회, 저항)
52주 고가 $64.97 / 저가 $41.62
→ 중기 하락 추세 내 단기 반등 국면. "확인된 상승 추세" 아님(오닐 기준 미충족)
```

### 시나리오 A — 추세 확인 후 진입 (채택안)
| 항목 | 값 | 계산 |
|---|---:|---|
| 진입가 | $54.20 | 200일선($53.93) 상향 돌파 확정 후 |
| 손절가 | $49.86 | 진입가 대비 -8% (오닐 규칙) |
| 손절폭 | 8.0% | (54.20-49.86)/54.20 |
| 1차 목표 | $64.97 | 52주 고점 |
| 목표 상승폭 | +19.9% | (64.97-54.20)/54.20 |
| **손익비** | **2.49 : 1** | 19.9% / 8.0% |

### 시나리오 B — 현재가 즉시 진입 (기각안)
| 항목 | 값 | 계산 |
|---|---:|---|
| 진입가 | $49.59 | 현재가 |
| 손절가 | $41.20 | 52주 저가 $41.62 하회 확정 시, 추가 여유 -0.9% |
| 손절폭 | 16.9% | (49.59-41.20)/49.59 |
| 목표 | $72.00 | 애널리스트 목표주가 중앙값 |
| 목표 상승폭 | +45.2% | (72.00-49.59)/49.59 |
| **손익비** | 2.67 : 1 (수치상 양호하나 기각) | 45.2%/16.9% — 손절폭 16.9%가 단일거래로는 지나치게 넓어 리스크 관리 원칙 위배 |

### 포지션 사이징 (하이트 원칙 — 단일거래 손실 상한 1~2%, 가정 계좌 $100,000)
```
리스크 상한(중간값 1.5%) = $100,000 × 1.5% = $1,500
손절폭 = 8.0% (시나리오 A)
매수 가능 포지션 규모 = $1,500 / 8.0% = $18,750
매수 가능 주식수 = $18,750 / $54.20 ≈ 346주
계좌 대비 포지션 비중 = 18.75%

보수적 버전(리스크 상한 1.0%):
= $1,000 / 8.0% = $12,500 → 계좌의 12.5%, 약 231주
```

## 리스크
- **추세 미확인 리스크**: 200일선 돌파 없이 진입하면 손절폭이 16.9%까지 벌어져(시나리오 B) 손익비 설계가 무너진다.
- **정책·규제 리스크**: 2021년 "쌍감정책" 트라우마가 여전히 valuation에 반영 중(펀더멘털 개선에도 주가가 52주 고점을 회복 못 하는 이유로 추정). 이는 제가 계산할 수 없는 영역 — 코브너처럼 시나리오별 대응만 준비할 뿐이다.
- **베타 0.18의 함정**: 미 증시와 상관성이 낮다는 것은 분산 효과처럼 보이지만, 동시에 "우리가 아는 시장 방향 지표"가 이 종목에는 잘 작동하지 않는다는 뜻이기도 하다. 중국 내수/정책 관련 별도 지표로 시장 방향을 재확인해야 하나 데이터팩에 없음(확인 불가).
- **차기 실적(2026-07-29) 이벤트 리스크**: 컨센서스 EPS $0.46, 매출 $1.47B. 진입 전 이 발표를 통과시킬지, 발표 후 반응을 보고 진입할지 결정 필요.

## 최종 의견
제가 인터뷰한 트레이더 중 누구도 "회사가 좋아 보여서" 진입하지 않았습니다. 마이클 마커스는 펀더멘털과 기술적 신호가 같은 방향을 가리킬 때만 확신을 가졌고, 지금 EDU는 그 조건을 충족하지 못합니다. 실적은 훌륭합니다 — 매출 +19.8%, 영업이익 +44.8%, 가이던스 상향. 하지만 가격은 아직 200일선 아래에 있고, 1년 수익률은 시장 대비 -20%p 이상 뒤처집니다. 애널리스트 목표주가 평균 +42%라는 숫자가 유혹적이지만, 그것은 제 관심사가 아닙니다. 제 관심사는 하나입니다 — "이 거래가 틀렸을 때 어디서 빠져나올 것인가?"

지금 당장 매수한다면 손절폭이 16.9%까지 벌어지고, 이는 단일거래 리스크 상한을 지키려면 포지션을 지나치게 작게 잡아야 한다는 뜻입니다. 반대로 200일선 돌파를 기다리면 손절폭 8%, 손익비 2.5:1의 훨씬 깨끗한 그림이 나옵니다. 기다리는 것도 유효한 포지션입니다. 지금은 감시 리스트에 올려두고 추세 확인 신호를 기다리십시오. 희망은 전략이 아닙니다.
