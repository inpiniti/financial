# AAPL 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-17 | 투자의견: 관망 | 확신도: 4/10

## 한 줄 결론
좋은 기업(ROC 계산조차 필요 없을 만큼 확실히 좋은 사업)이지만, 이익수익률 약 2.7%는 마법 공식이 좋아하는 "싼 가격"과 거리가 멀다 — 지금은 "좋지만 비싼" 쪽에 서 있다.

## 내 철학에서 본 이 기업
얘들아, 껌 가게 얘기부터 하자. 내가 1,000달러를 투자해서 1년에 200달러를 벌면 이익수익률(EY) 20%다. 그런데 같은 껌 가게를 4,910달러에 사야 한다면? 200달러 ÷ 4,910달러 = 4%도 안 된다. 애플이 지금 딱 이 상황이다. 사업 자체는 기가 막히다. 문제는 가격표다.

## 체크리스트 평가

| 항목 | 평가 | 근거 수치 |
|---|---|---|
| 1. 자본수익률(ROC) 높은가 | △ (부분 산출) | 순운전자본이 음수(−$17,674M)라 표준 ROC 공식이 그대로 작동하지 않음. 아래 계산 참고 |
| 2. 이익수익률(EY) 높은가 | × | EY ≈ 2.71% — 마법 공식 유니버스 기준으로 낮은 편(그린블라트가 원래 스크리닝하던 시절 기준 두 자릿수 EY 종목군과 비교하면 현저히 낮음) |
| 3. 규모 필터 통과 | ○ | 시가총액 $4.85조, 유니버스 제외 대상 아님(오히려 초대형주) |
| 4. 좋으면서 동시에 싼가 | × | 좋음은 확실, 쌈은 전혀 아님 |
| 5. 회계적 왜곡 가능성 | △ | 자사주 매입으로 자기자본·ROE가 극단적으로 왜곡됨(ROE 141.5%) — ROC 계산에도 유의 필요 |

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

### 1) EBIT 산출
```
EBIT ≈ 영업이익률 × 매출 (FY2025)
     = 32.0% × $416,161M
     = $133,171M
```

### 2) 투입 유형자본 (ROC 분모)
```
순운전자본(NWC) = 유동자산 - 유동부채 (FY2025)
              = $147,957M - $165,631M
              = -$17,674M  (음수)

순고정자산(Net PP&E): 데이터 팩에 별도 항목 미제공 → 확인 불가
```
그린블라트 공식 그대로면 `ROC = EBIT ÷ (NWC + 순고정자산)`인데, NWC가 음수라 분모가 왜곡된다. 애플은 협력사 대상 매입채무 등으로 운전자본을 마이너스로 운용하는 전형적인 "자본 효율 극대화" 기업이다 — 이는 실제로는 ROC가 극도로 높다는 신호이지, 나쁜 신호가 아니다. 다만 정확한 순고정자산(PP&E) 수치 없이는 정밀한 ROC 숫자를 못 낸다.

**참고 근사치**: PP&E를 대략 $45,000M 내외(Apple 통상 규모, 데이터 팩 미확인이므로 참고용)로 가정하면 투입자본 ≈ $45,000M − $17,674M = 약 $27,000M, ROC ≈ $133,171M ÷ $27,000M ≈ **490%대**. 이 숫자는 확인 불가 항목을 임의 가정한 것이므로 참고치일 뿐, 보고서 본문 판단에는 반영하지 않는다. **정확한 ROC는 확인 불가로 남긴다.**

### 3) EV (기업가치)
```
EV = $4.91조 (데이터 팩 5절 직접 제공, 시가총액 + 부채 - 현금)
```

### 4) 이익수익률(EY) 계산
```
EY = EBIT ÷ EV
   = $133,171M ÷ $4,910,000M
   = 2.71%
```

**비교 기준**: 무위험 이자율(미 국채 장기물 대략 4%대 수준, 데이터 팩에 직접 수치 없음이나 상식적 벤치마크)조차 밑도는 EY다. 그린블라트 식으로 말하면, 이 가격에 애플 전체를 인수한다면 EBIT 기준 자본 회수에 약 37년(1/2.71%)이 걸린다는 뜻이다. 좋은 사업이라도 이 정도면 "싸다"고 부를 수 없다.

### ⚠ 자사주매입에 의한 자본 왜곡 명시
- ROE 141.5% — 이는 사업이 갑자기 초효율화된 게 아니라, 지속적 자사주 매입으로 자기자본이 인위적으로 축소된 결과다(FY2022 자기자본 $50,672M → FY2025 $73,733M, 그 사이 총 자사주매입 규모는 순이익 누적을 크게 상회).
- PBR 45.5x도 같은 이유로 해석 무의미(장부가 자체가 왜곡).
- 마법 공식의 ROC는 ROE보다 이런 왜곡에 상대적으로 덜 취약하지만(분모가 자기자본이 아니라 투입 유형자본이므로), 애플처럼 순운전자본이 마이너스인 극단적 사례에서는 ROC 공식도 액면 그대로 신뢰하기 어렵다. 좋은 기업이라는 결론 자체는 흔들리지 않지만, 정밀한 순위 매기기에는 주의가 필요하다.
- **EY는 EV(시가총액 기반) 대비 EBIT이므로 자사주매입에 의한 자본구조 왜곡의 영향을 상대적으로 덜 받는다** — 다만 자사주매입이 주가를 밀어올려 시가총액(EV의 핵심 구성요소)을 키우는 경로로는 EY를 낮추는 요인이 된다는 점은 유의해야 한다.

## 리스크
- 마법 공식 관점에서 가장 큰 리스크는 "좋은 기업이니까 비싸도 괜찮다"는 직관을 공식보다 우선시하는 것 — 바로 내가 경계하라고 한 그 함정이다.
- 자사주매입에 의한 지표 왜곡: ROE·PBR은 신뢰하지 말 것. ROC도 순운전자본 음수 구조 때문에 정밀 계산에 데이터 추가 확보(순고정자산)가 필요하다.
- 52주 고점 대비 -1.4%, 역대 최고가 근접권에서 EY가 낮다는 것은 "가격이 실적 개선보다 더 빨리 뛰었다"는 신호와 일치한다.
- EV/EBITDA 30.7x, Forward PER 34.3x도 EY 낮음과 같은 방향의 신호 — 여러 지표가 일관되게 "비싸다"를 가리킨다.

## 최종 의견
얘들아, 애플이 나쁜 회사냐고? 절대 아니다. 마이너스 순운전자본으로 사업을 돌린다는 것 자체가 얼마나 자본을 효율적으로 쓰는지 보여주는 증거다. 문제는 그게 아니야. 문제는 "지금 얼마에 살 수 있는가"다. EBIT $1,331억을 EV $4.91조로 나누면 이익수익률이 2.71%밖에 안 나온다. 은행 예금 이자보다 못한 수익률로 사업 전체를 사겠다는 건, 아무리 좋은 사업이라도 마법 공식 관점에서는 통과시킬 수 없다.

공식이 좋아하는 건 "좋으면서 동시에 싼" 기업이다. 애플은 지금 앞부분만 만족하고 뒷부분은 전혀 못 채운다. 나는 뉴스에, 애플 인텔리전스가 중국에 들어간다는 소식에, 사상 최고가 경신에 절대 흔들리지 않는다. 숫자가 말해준다 — 지금은 살 때가 아니라 기다릴 때다. 이 가격에 사는 건 좋은 껌 가게를 시세보다 몇 배 비싸게 사는 것과 같다.

인내심을 가져라. 미스터 마켓은 언젠가 또 비관적인 날을 만들어줄 거다. 그날 애플의 EY가 다시 두 자릿수 근처로 올라오면, 그때 이야기가 완전히 달라진다.
