# NVDA 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-17 | 투자의견: 보유 | 확신도: 6/10

## 한 줄 결론
엔비디아는 껌 가게 비유로 치면 "세상에서 제일 잘되는 껌 가게"입니다 — 자본수익률이 100%를 훌쩍 넘습니다. 문제는 이 가게를 사려면 벌어들이는 돈의 30배에 가까운 값을 치러야 한다는 겁니다. 좋은 기업은 맞는데, 마법 공식이 좋아하는 "싼" 기업은 아닙니다.

## 내 철학에서 본 이 기업
저는 스토리를 안 봅니다. AI가 어떻고, 젠슨 황이 어떻고, 중국 수출 규제가 어떻고 하는 이야기는 숫자로 환산되기 전까지는 소음입니다. 제가 보는 건 딱 둘, ROC(자본수익률)와 EY(이익수익률)입니다. 이 두 숫자를 계산해 보겠습니다.

## 체크리스트 평가

| 항목 | 평가 | 근거 |
|---|---|---|
| ROC (좋은 기업인가) | ○ | 근사 ROC 약 105% — 극히 높음, 압도적 경쟁우위 신호 |
| EY (싼 가격인가) | × | EY 약 3.35% — 국채 금리보다도 낮음, 전혀 싸지 않음 |
| 순위 결합 관점 | △ | ROC는 유니버스 최상위권일 것, 그러나 EY가 최하위권으로 합산 순위를 끌어내림 |
| 규모 필터 | ○ | 시총 4.88조 달러, 문제없음 |
| 회계 왜곡 가능성 | △ | 순고정자산 세부 미확보로 ROC는 근사치, 실제는 다소 차이 있을 수 있음 |

## 밸류에이션 판단 (마법공식 계산)

### 원자료
| 항목 | 값 | 출처 |
|---|---|---|
| TTM EBIT | 1,622.9억 달러 | Q2FY26~Q1FY27 영업이익 합산 (284.4+360.1+443.0+535.4) |
| 시가총액 | 48,840억 달러 | 데이터 팩 §5 |
| 총부채 | 128.1억 달러 | 데이터 팩 §4 (26-04-30) |
| 총현금 | 531.7억 달러 | 데이터 팩 §4 |
| 유동자산 | 1,509.95억 달러 | 데이터 팩 §4 |
| 유동부채 | 438.84억 달러 | 데이터 팩 §4 |
| 자본총계 | 1,954.7억 달러 | 데이터 팩 §4 (26-04-30) |

### 1) 기업가치(EV)
```
EV = 시가총액 + 총부채 − 총현금
   = 48,840 + 128.1 − 531.7
   = 48,436.4억 달러
```

### 2) 이익수익률 (EY = EBIT ÷ EV)
```
EY = 1,622.9억 ÷ 48,436.4억 = 3.35%
```
→ 국채 금리 수준과 비슷하거나 낮음. "싸다"는 기준에서 완전히 탈락. 시장은 이 사업을 인수하려면 벌어들이는 이익의 약 30배를 내라고 요구하고 있습니다(1/EY ≈ 29.9배, Trailing PER 30.88x와 정합).

### 3) 자본수익률 (ROC = EBIT ÷ 투입 유형자본)
정식 산식은 `EBIT ÷ (순운전자본 + 순고정자산)`이나, 데이터 팩에 PP&E(유형자산 순액) 세부 항목이 없어 **확인 불가**. 대안으로 아래 근사식을 사용:
```
투입자본(근사) = 자본총계 + 총부채 − 총현금
              = 1,954.7 + 128.1 − 531.7
              = 1,551.1억 달러

ROC(근사) = EBIT ÷ 투입자본(근사)
          = 1,622.9억 ÷ 1,551.1억
          = 104.6%
```
참고로 순운전자본만 놓고 봐도 `1,509.95억 − 438.84억 = 1,071.11억`으로, 여기에 순고정자산을 더하면 정식 ROC는 근사치보다 다소 낮아질 가능성이 있습니다. 그럼에도 100%에 근접하는 자본수익률은 웬만한 유니버스에서 최상위권입니다.

### 4) 마법공식 결합 판단
- ROC ≈ 105% → "좋은 기업" 축에서는 최상위권 순위 예상.
- EY ≈ 3.35% → "싼 가격" 축에서는 시장 평균 이하, 최하위권 순위 예상.
- 마법공식은 두 순위를 **합산**합니다. ROC 1등이라도 EY가 바닥이면 합산 점수가 나빠져 상위권에 들지 못합니다. 지금 엔비디아는 "좋지만 비싼" 회사 — 제가 회피 목록 1순위로 꼽는 유형입니다.

### 참고: Forward PER 기준 EY
Forward PER 15.72x를 역수 취하면 이익수익률 약 6.4%. 이는 시장이 향후 이익 성장(현재 EBIT 대비 약 2배 성장)을 이미 가격에 반영했다는 뜻입니다. 저는 미래 추정치보다 지금 이 순간의 EBIT과 EV로 계산하는 것을 원칙으로 삼습니다. 성장은 덤이지 전제가 아닙니다.

## 리스크
- 회계적 이익과 실제 현금흐름 괴리: FCF(TTM 근사 966.8억 FY26)와 EBIT 간 큰 괴리는 없어 이 부분은 양호.
- 지표 왜곡 가능성: H20 재고 상각(약 45억 달러)이 일회성으로 최근 분기 EBIT을 낮췄을 수 있어, TTM 합산 EBIT이 다소 보수적으로 잡혔을 가능성 있음 → 실제 ROC/EY는 계산치보다 약간 높을 수 있음.
- 순고정자산 데이터 부재로 ROC는 근사치 — 정밀 계산 시 오차 가능.
- 대중국 수출 규제, 자체 ASIC 확대 등 정성적 리스크는 제가 다루는 영역이 아닙니다. 이런 이슈는 EY가 낮은 이유를 설명해줄 뿐, 제 계산을 바꾸지 않습니다.
- 마법공식 자체의 한계: 이 계산은 개별 종목 매수 근거가 아니라 유니버스 전체 순위 매김의 한 데이터 포인트일 뿐입니다. 엔비디아 하나만 떼어 판단하는 건 제 원래 방법론과 어긋납니다.

## 최종 의견
좋은 기업입니다. 자본수익률 100%가 넘는 사업은 흔치 않습니다. 껌 가게로 치면 껌 한 통 값의 자본으로 매년 껌 통 자체보다 더 많은 이익을 뽑아내는 가게예요. 그런데 그 가게를 사겠다고 줄 선 사람들이 이미 이익의 30배를 부르고 있습니다. 이익수익률 3.35%는 제가 좋아하는 숫자가 아닙니다. 국채 사는 것보다 나을 게 없어 보이죠.

이미 들고 있다면 팔 이유는 없습니다 — ROC가 이렇게 높은 회사는 시장이 계속 프리미엄을 줄 가능성이 있고, 저는 단기 주가를 예측하지 않으니까요. 하지만 마법 공식 관점에서 새로 담을 자리는 아닙니다. 공식이 원하는 건 "좋으면서 싼" 회사입니다. 지금 엔비디아는 "좋지만 비싼" 회사예요. 제 유니버스 순위표라면 ROC 1등 근처, EY는 바닥권 — 합산하면 상위 20~30개 안에 들지 못할 가능성이 큽니다. EY가 6~8%대로 올라올 만큼 주가가 조정되거나 EBIT이 더 커지면 다시 계산해 보겠습니다.
