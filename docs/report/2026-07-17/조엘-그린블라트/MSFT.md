# MSFT 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-17 | 투자의견: 관망 | 확신도: 4/10

## 한 줄 결론

MSFT는 "좋은 기업"인 건 숫자로도 명백합니다(ROC 근사치 약 32%). 문제는 "싼 기업"이 아니라는 점입니다 — 이익수익률(EY)이 4.3% 남짓으로, 마법 공식이 원하는 저가 매수 구간과는 거리가 멉니다.

## 내 철학에서 본 이 기업

여러분, 껌 가게 이야기를 다시 해봅시다. 제이슨의 껌 가게가 2,000원을 투자해서 매년 640원을 번다면 정말 훌륭한 사업입니다. 그런데 그 가게를 통째로 인수하는 데 30,000원을 내야 한다면? 훌륭한 사업이라도 "지금 이 가격에" 사는 건 다른 문제입니다. MSFT가 딱 이 그림입니다. 사업 자체는 나무랄 데 없이 좋습니다. 문제는 가격표입니다.

## 체크리스트 평가

| 항목 | 평가 | 근거 수치 |
|---|---|---|
| 자본수익률(ROC) 높은가 | ○ | EBIT $128,466M ÷ 근사 투하자본 $404,067M ≈ **31.8%** — 우량 기업 기준(통상 20%+) 상회 |
| 이익수익률(EY) 매력적인가 | × | EBIT ÷ EV ≈ **4.31%** — 마법 공식이 노리는 저가 구간(통상 두 자릿수 중반 이상)에 한참 못 미침 |
| 두 지표 모두 상위권인가 | △ | ROC는 상위권이나 EY가 하위권 — "좋지만 싸지 않은" 전형적 회피 유형에 해당 |
| 규모 필터 통과 | ○ | 시총 $2.9조, 필터 대상에서 제외될 이유 없음 |
| 나쁜 뉴스로 값이 눌린 "일시적 저평가"인가 | △ | 6~7월 capex 우려·FTC 조사·Xbox 구조조정으로 -24% 하락했으나, EY 자체는 여전히 낮은 수준 — 비관이 가격을 "싸게" 만들 정도까지는 밀어내지 못함 |

## 밸류에이션 판단 (계산 과정 명시)

### 1) EBIT (FY2025 확정치 기준)
```
EBIT = 매출 × 영업이익률 = $281,724M × 45.6% ≈ $128,466M
```

### 2) 투입 유형자본 (근사)
원 공식은 "순운전자본 + 순고정자산"이나, 데이터 팩에 순고정자산 원자료가 없어 아래로 근사 대체:

| 항목 | 값 |
|---|---:|
| 유동자산(최근분기) | $175,329M |
| 유동부채(최근분기) | $136,661M |
| 순운전자본 | $38,668M |
| 자기자본(FY25) | $343,479M |
| 총부채(FY25, 유이자 기준) | $60,588M |
| **투하자본 근사(자기자본+총부채)** | **$404,067M** |

```
ROC ≈ $128,466M ÷ $404,067M = 31.8%
```
※ 원 공식(순운전자본+순고정자산) 기준으로는 이보다 훨씬 높게(경우에 따라 세 자릿수%) 나올 가능성이 큽니다 — MSFT는 유형자산 비중이 낮은 소프트웨어·플랫폼 기업이기 때문입니다. 정확한 값은 "확인 불가"로 유보합니다.

### 3) EV (기업가치)
```
EV = 시가총액 + 총부채 - 현금
   = $2,918,000M + $60,588M - (현금, 확인 불가)
  ≈ $2,978,588M (현금 미차감 → 보수적으로 EV 과대, EY 과소 방향)
```

### 4) 이익수익률(EY)
```
EY = EBIT ÷ EV = $128,466M ÷ $2,978,588M = 4.31%
```

### 순위 결합 판단
- ROC: 상위권 추정 (약 32%, 우량 기업 기준 충족)
- EY: 하위권 (4.3%는 국채 수익률과 큰 차이 없는 수준 — 마법 공식이 원하는 "인수자가 지불한 값 대비 벌어들이는 몫"으로는 매력 부족)
- **합산 순위는 좋으나 비싸다 쪽으로 크게 기웁니다.** 마법 공식이 최고로 치는 "ROC도 높고 EY도 높은" 교집합에는 들지 못합니다.

### 시나리오: 어느 정도 가격이면 매력적인가
EY가 최소 8% 수준(질 좋은 기업 기준 합리적 진입선)이 되려면:
```
목표 EV = EBIT ÷ 8% = $128,466M ÷ 0.08 ≈ $1,605,825M
→ 시가총액 환산(부채 차감) ≈ $1,545,237M → 현재 시총의 약 53% 수준
→ 현재가 $393.11 대비 주가 약 $209 부근
```
이 정도까지 내려와야 제 공식의 "싸다" 기준에 근접합니다. 지금은 그 절반에도 못 미치는 조정입니다.

## 리스크

- **공식 자체의 한계**: EBIT 산정에 회계연도 확정치를 썼으나, FY26 실적(Azure +40%, AI 매출 급증)이 이어진다면 EY는 개선될 수 있습니다. 다만 capex $190B 가이던스는 미래 EBIT 성장에 대한 시장의 확신을 흔들고 있고, 이는 회계 이익이 실제 현금 창출과 괴리될 수 있다는 제 원칙적 경고와도 맞닿아 있습니다.
- 투하자본 계산이 근사치라 ROC 정확도에 한계가 있습니다(순고정자산 원자료 확인 불가).
- 현금 데이터 미확보로 EV가 다소 과대평가되어 EY가 실제보다 낮게 나왔을 가능성 — 다만 MSFT 시총 규모($2.9조)를 감안하면 현금 차감으로도 EY 결론이 뒤집힐 정도는 아닙니다.
- FTC 반독점 조사, Xbox 구조조정 등은 정성적 노이즈로, 저는 이를 ROC·EY 숫자에 반영되지 않는 한 판단에 넣지 않습니다.

## 최종 의견

여러분, "좋은 기업이니까 사야죠"라는 말에 저는 항상 이렇게 되묻습니다. "그래서 ROC는 얼마고, EY는 얼마입니까?" MSFT는 ROC로 보면 확실히 좋은 기업입니다, 대략 32% — 제이슨의 껌 가게보다 훨씬 남는 장사입니다. 하지만 EY가 4.3%면, 이건 그 좋은 가게를 지나치게 비싼 값에 사겠다는 얘기와 같습니다. 최근 주가가 고점 대비 -24% 빠졌다고 해서 "싸졌다"고 착각하면 안 됩니다. 비싼 것이 덜 비싸진 것과 싼 것은 다릅니다. 저는 이 종목을 지금 당장 제 마법 공식 포트폴리오 상위권에 올리지 않습니다. 좋은 기업 목록에는 적어두되, 가격이 EY 기준으로 훨씬 더 매력적인 구간(대략 주가 $200대 초중반)까지 내려오길 기다리는 것이 제 원칙입니다. 그리고 명심하십시오 — 이 인내심이야말로 공식이 계속 작동하는 이유입니다.
