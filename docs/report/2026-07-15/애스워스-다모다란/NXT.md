# NXT 투자 보고서 — 애스워스 다모다란 (주식 가치평가 완벽 정리)
> 날짜: 2026-07-15 | 투자의견: 매도 | 확신도: 7/10

## 한 줄 결론
낙관 시나리오의 주당가치조차 $82로 현재가 $104.66에 못 미칩니다. 시장은 향후 5년간 FCF 연 30% 이상 성장을 가격에 반영하고 있는데, 이는 제 가장 낙관적인 스토리(연 18% 성장)보다도 훨씬 공격적입니다.

## 내 철학에서 본 이 기업

Nextracker/Nextpower는 스토리가 뚜렷합니다: 태양광 트래커 세계 1위 제조사가 모듈 프레임·AI·로보틱스·전기 BOS·PCS 인수를 통해 "통합 전력기술 기업"으로 확장한다는 것입니다. 수주잔고 $5.25B는 이 스토리를 뒷받침하는 숫자입니다. 하지만 저는 이 스토리를 숫자로 번역할 때 세 가지를 반드시 짚어야 한다고 봅니다.

첫째, 이 기업의 이익 중 상당 부분이 **정책 보조금(IRA 45X 첨단제조 세액공제)** 에서 나옵니다. 분기당 $47~67M, 연 환산 약 $220M — FY2026 순이익 $586M의 약 37~38%에 달합니다. 이는 회사의 경쟁우위가 아니라 정치적 결정에서 나오는 현금흐름이며, 다모다란의 원칙("숫자는 스토리를 반영해야 한다")에서 보면 이 부분은 훨씬 높은 위험 프리미엄으로 할인해야 합니다.

둘째, 관세(tariff) 부담이 분기별 원가율에 200~300bp씩 확대되고 있고, 영업이익률이 FY2024 23.5% → FY2025 21.6% → FY2026 19.6%로 3년 연속 하락 중입니다. 마진이 스토리(고성장+고수익)와 반대 방향으로 가고 있습니다.

셋째, 신사업(PCS)은 아직 매출 기여가 없는 "약속"이지, "현금흐름"이 아닙니다. FY2027 가이던스에 PCS 관련 약 $50M 추가비용이 반영돼 있다는 것 자체가, 신사업이 당분간 이익을 깎아먹는 단계임을 보여줍니다.

## 밸류에이션 판단 (2단계 DCF, 계산 과정)

**공통 입력**: FY2026 FCF $514M, 발행주식 151.65M주, 순현금 $1,093M(현금 1,146 - 부채 53), 5년 고성장기 + 터미널.

### 비관 (IRA 45X 세액공제 축소 반영)
가정: OBBBA로 세액공제 3년 내 50% 감소 + 관세 지속 → FCF 성장률 -10%, -5%, 0%, 3%, 5%. 할인율 14.5%(정책 불확실성 가산), 영구성장률 2.0%.

| 연도 | FCF($M) | PV(14.5%) |
|---|---|---|
| FY27 | 463 | 404 |
| FY28 | 439 | 335 |
| FY29 | 439 | 292 |
| FY30 | 452 | 262 |
| FY31 | 475 | 240 |
| **5년 PV 합계** | | **1,533** |

터미널가치 = 475×1.02/(0.145-0.02) = 3,876 → PV = 3,876/1.145⁵ = **1,972**
기업가치 = 1,533+1,972 = 3,505 + 순현금 1,093 = **자기자본가치 4,598**
**주당가치 = $30.32**

### 기준
FCF 연 10% 성장 5년, 할인율 12.7%(무위험 4.3%+베타1.86×ERP4.5%), 영구성장률 2.5%.

| 연도 | FCF($M) | PV(12.7%) |
|---|---|---|
| FY27~FY31 | 565→828 | 501+490+477+467+455 |
| **5년 PV 합계** | | **2,390** |

터미널가치 = 828×1.025/0.102 = 8,321 → PV = **4,577**
기업가치 = 2,390+4,577 = 6,967 + 순현금 1,093 = **자기자본가치 8,060**
**주당가치 = $53.15**

### 낙관
FCF 연 18% 성장 5년(PCS 안착+backlog 완전실행), 할인율 11.5%, 영구성장률 3.0%.

| 연도 | FCF($M) | PV(11.5%) |
|---|---|---|
| FY27~FY31 | 607→1,177 | 544+576+610+645+684 |
| **5년 PV 합계** | | **3,059** |

터미널가치 = 1,177×1.03/0.085 = 14,259 → PV = **8,285**
기업가치 = 3,059+8,285 = 11,344 + 순현금 1,093 = **자기자본가치 12,437**
**주당가치 = $82.00**

### 역산 DCF — 현재가를 정당화하는 성장률

할인율 12.7%, 영구성장률 2.5% 고정 상태에서 시행착오로 역산한 결과, **5년간 FCF 연평균 약 30~32% 성장**이 있어야 기업가치(순현금 차감 후 시총 수준 약 $15,000M)에 도달합니다.

- g=25% 시도 → 기업가치 약 $12,314M (부족)
- g=32% 시도 → 5년 FCF PV 약 $4,233M + 터미널PV 약 $11,393M = **$15,626M** (목표 근접)

이는 FY2026 매출성장(+20%), FY2027 가이던스 매출성장(+7~15%)보다 훨씬 공격적이며, FCF가 매출성장률을 상회하는 속도로 전환율까지 개선돼야 합니다. 최근 30일 EPS 컨센서스가 하향 우세(9건 down vs 2건 up)인 흐름과도 배치됩니다.

## 리스크

- **정책 리스크(최우선)**: IRA 45X 세액공제가 순이익의 37~38%를 차지. OBBBA발 Section 48E·45Y·45X 축소 일정이 10-K에 명시된 핵심 리스크 — 비관 시나리오에 이미 반영.
- **마진 추세 리스크**: 영업이익률 3년 연속 하락(23.5%→21.6%→19.6%), 관세 부담 분기별 200~300bp 확대 중.
- **밸류에이션 리스크**: PEG 3.91, forward PER 17.8배는 최근 성장 둔화(분기 순이익 QoQ 감소)를 감안하면 낙관적.
- **추정 오차 리스크(제 자신의 리스크)**: 할인율·영구성장률의 작은 변화가 터미널 밸류(전체 가치의 60~80%)를 크게 흔듭니다. 이 추정은 거의 확실히 틀릴 것이나, 방향성 판단의 닻으로는 유효합니다.

## 최종 의견

제 낙관 시나리오조차 $82입니다. 시장은 지금 $104.66을 지불하고 있는데, 이건 제가 세운 어떤 스토리로도 정당화되지 않습니다. 역산해보니 향후 5년간 FCF가 연 30% 이상 복리로 성장해야 한다는 결론이 나왔습니다 — 회사 자신의 가이던스(매출 성장 한 자릿수 후반~십대 초반%)보다도 두 배 이상 공격적인 가정입니다. 그리고 이 회사 이익의 상당 부분은 정책 보조금에서 나옵니다. 정책은 제가 통제할 수 없는 변수이고, 그것이 사라질 때 숫자가 얼마나 취약해지는지는 비관 시나리오가 보여줍니다.

수주잔고 $5.25B, 세계 1위 지위, 순현금 포지션 — 이 회사가 나쁜 회사라는 뜻은 아닙니다. 다만 좋은 회사와 좋은 투자는 다른 질문입니다. 가격이 이미 완벽한 실행과 정책 지속을 전제로 형성돼 있다면, 그 가격에 사는 것은 밸류에이션이 아니라 베팅입니다. 대략적으로 맞는 범위 $30~82를 시장가 $105가 벗어나 있다는 사실이, 지금은 매도 혹은 최소한 신규 매수 보류가 합리적이라는 근거입니다.
