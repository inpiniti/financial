### 손익비
- 리스크(주당) = $41.30 - $38.50 = **$2.80**
- 보상(1차) = $53.00 - $41.30 = $11.70 → **손익비 4.18 : 1**
- 보상(2차) = $55.67 - $41.30 = $14.37 → **손익비 5.13 : 1**
- 데니스의 원칙대로, 승률이 낮아도(예: 40%) 이 손익비면 기댓값은 플러스: 0.4×11.70 - 0.6×2.80 = $4.68 - $1.68 = **+$3.00 (기대값 양수)**

## 리스크
- **소형주 유동성**: 시총 $531.8M, float 1,230만주. 급락 시 슬리피지 위험 — 손절 주문은 역지정가(stop-limit)보다 시장가 손절(stop-market)에 가까운 실행 규율 필요
- **부채 레버리지**: Flatrock 인수로 Debt/Equity 83.9%까지 상승, 순부채 $224.2M. 금리·가스 시추 사이클 하강 시 재무 부담 확대 가능
- **FCF 마이너스 지속**: 성장 CapEx($55-70M 가이던스)가 OCF를 초과하는 구조 — 임대업 특성이나, 가동률 정체 시 자본조달 리스크로 전환될 수 있음
- **공매도 잔고 증가**: 30.2만주 → 44.8만주로 증가 추세 — 신고가 근처에서의 공매도 증가는 상단 저항 신호일 수 있음, 주시 대상
- **이자보상배율 확인 불가**: 데이터팩에 이자비용 항목 미확보 — 부채 부담 정밀 평가에 공백 존재

## 최종 의견
제가 인터뷰한 위저드 중 누구도 "확실한 종목"이라는 표현을 쓰지 않았습니다. 대신 이렇게 물었을 겁니다. "이 거래가 틀렸을 때 어디서 나올 것인가?" 이 종목의 답은 명확합니다 — $38.50, 진입가 대비 -6.8%. 그리고 "얼마를 잃을 수 있는가?" — 계좌의 1.5%, $1,500. 추세와 펀더멘털이 같은 방향을 가리키고(마커스의 조건), 손익비가 4:1을 넘습니다(데니스의 원칙). 이 정도면 진입할 이유는 충분합니다. 다만 float가 얇고 공매도 잔고가 늘고 있다는 것, 부채비율이 84%까지 올라왔다는 것 — 이 두 가지는 절대 잊지 말아야 합니다. 확신도를 8이 아니라 6에 두는 이유는, 시장 전체 방향(섹터 데이터 부재)과 유동성 세부 수치가 확인 불가이기 때문입니다. 규율 없이 확신만 갖고 들어가는 건 트레이딩이 아니라 도박입니다. 손절가를 지키십시오. 희망은 전략이 아닙니다.
