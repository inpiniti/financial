## 한 줄 결론
저는 매크로도 마이크로도 예측하지 않습니다. 지금 이 자리에서 확실히 말할 수 있는 건 하나뿐입니다 — 8일 뒤 실적, 12개월 뒤 완결될 분사, 아직 확인되지 않은 추세 반전, 이 세 가지가 동시에 걸린 자리에서 "손절 폭이 좁은" 진입은 설계하기 어렵습니다. 손익비 계산은 해드리지만, 지금은 트리거를 기다리는 자리입니다.

## 내 철학에서 본 이 기업

"추세가 확인됐는가?" 답은 아직 "아니오"입니다. 52주 고점 $33.76에서 저점 $22.13까지 -34.5% 흘러내렸고, 최근 1년 궤적(2025-07 $32.4 → 2026-07 $23.2)은 완만한 하락이 최근 다시 가속되는 모양입니다. 4월 실적은 컨센서스를 상회했는데도 주가는 -13% 급락했습니다 — 마이클 마커스라면 이걸 "펀더멘털과 기술적 신호가 엇갈린 대표 사례"라고 부를 겁니다. 시장은 서프라이즈보다 조정 EBITDA -16.8% YoY, FCF -28% YoY에 더 크게 반응했습니다. 6월 29일 분사 발표에 +8% 튀었지만 그 상승분도 이후 흡수당했습니다. 이건 "추세가 친구"인 상황이 아니라 "시장이 계속 수업료를 청구하는" 상황입니다.

### 시나리오 A: 즉시 진입 (현재가 매수)
| 항목 | 값 |
|---|---|
| 진입가 | $23.19 |
| 손절가 | $21.50 (52주 저가 $22.13 이탈 확정 + 여유폭) |
| 리스크(주당) | $23.19 − $21.50 = **$1.69** (진입가 대비 −7.3%) |
| 목표가 1 (최근 저항권, 20주 이동평균대 재탈환) | $27.60 |
| 목표가 2 (Morgan Stanley 하향 후 목표가 수준) | $30.00 |
| 손익비 1 | ($27.60−$23.19) / $1.69 = $4.41 / $1.69 = **2.6 : 1** |
| 손익비 2 | ($30.00−$23.19) / $1.69 = $6.81 / $1.69 = **4.0 : 1** |

### 시나리오 B: 지지 재확인 후 진입 (트리거 조건부, 실적 통과 후)
| 항목 | 값 |
|---|---|
| 진입가 | $22.50 (52주 저점 $22.13 근접 지지 테스트 확인 시) |
| 손절가 | $21.50 |
| 리스크(주당) | $22.50 − $21.50 = **$1.00** (진입가 대비 −4.4%) |
| 목표가 1 | $27.60 → 손익비 = $5.10/$1.00 = **5.1 : 1** |
| 목표가 2 | $30.00 → 손익비 = $7.50/$1.00 = **7.5 : 1** |

시나리오 B가 손익비 면에서 압도적으로 우월합니다 — 손절 폭을 좁히는 것만으로 손익비가 2배 이상 개선됩니다. 데니스의 원칙 그대로: "틀렸을 때 빨리 나가고, 맞았을 때 크게 벌어야" 손익비가 만들어집니다.

### 포지션 사이징 (가정: 계좌 자본 $100,000, 단일 거래 리스크 상한 1.5% = $1,500)

| | 시나리오 A | 시나리오 B |
|---|---|---|
| 주당 리스크 | $1.69 | $1.00 |
| 최대 매수 주식수 (리스크 기준) | $1,500 / $1.69 ≈ **887주** | $1,500 / $1.00 = **1,500주** |
| 매입금액 | 887 × $23.19 ≈ $20,570 (자본의 20.6%) | 1,500 × $22.50 = $33,750 (자본의 33.8%) |
| 분산 상한 체크 (단일종목 상한 20~25% 가정) | 통과 | **초과 — 캡 필요** |

래리 하이트라면 시나리오 B에서 리스크금액 기준 사이징 결과가 단일 종목 비중 상한을 넘는 순간 "리스크가 아니라 포지션 크기가 문제"라며 주식수를 줄이라고 했을 겁니다. 손익비가 좋다고 사이징 규율을 깨면 안 됩니다.

## 리스크

- **실적 발표 임박(7/23)**: 4월 사례(서프라이즈+13% 급락 반대 방향 -13%)가 보여주듯, 이 종목은 컨센서스 상회에도 주가가 반대로 갈 수 있는 자리입니다. 실적 통과 전 신규 진입은 변동성 갭 리스크를 그대로 안는 것.
- **분사 실행 리스크(~1년 소요)**: 이사회·규제 승인 필요, 최종 구조·시점 불확실. 이 기간 내내 "재평가 vs 불확실성 프리미엄"이 충돌할 것.
- **추세 미확인**: 52주 -31%, 조정 EBITDA·FCF 역성장세가 지속 중. 현재가가 "싸 보이는" 것과 "바닥을 찍었다"는 것은 다릅니다.
- **배당 매력에 대한 함정 인식**: 5.51% 배당수익률은 매력적이나, 배당성향 25.9%도 낮은 편이라 커버리지 자체는 안전해 보입니다. 다만 이것이 "하락 추세를 무시하고 담아도 된다"는 근거가 되어서는 안 됩니다 — 배당은 손절 규칙을 대체하지 않습니다.

## 최종 의견
제가 인터뷰한 위저드들에게 지금 CMCSA를 보여주면 대체로 같은 반응을 보일 겁니다. "숫자는 흥미롭다. 배당 5.5%, PER 4.7배, 손익비 2.6~7.5:1까지 계산되는 자리다. 하지만 8일 뒤 실적이 있고, 이 종목의 최근 실적 반응 패턴은 서프라이즈해도 급락하는 쪽이었다. 추세도 아직 확인 안 됐다." 마이클 마커스라면 "펀더멘털과 기술적 신호가 같은 방향을 가리킬 때까지 기다리라"고 했을 거고, 폴 튜더 존스라면 "얼마를 벌 수 있는가보다 얼마를 잃을 수 있는가를 먼저 생각하라"고 했을 겁니다. 지금은 진입이 아니라 대기입니다. 손절가($21.50)와 목표가($27.60/$30.00)는 이미 계산해뒀습니다 — 실적을 통과하고 저점 지지가 재확인되면 시나리오 B로 움직이십시오. 기다리는 것도 유효한 포지션입니다.
