# NBIX 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-15 | 투자의견: 보유(관망에 가까운 소극적 보유/제한적 신규 진입) | 확신도: 5/10

## 한 줄 결론

제가 인터뷰한 트레이더들이라면 이 차트를 보고 이렇게 물었을 겁니다 — "추세는 살아있는데, 왜 하필 실적 발표 15일 전에 사이즈를 키우려 하는가?" 추세 자체는 나쁘지 않지만, 사상 최고가 경신 직후 조정과 이벤트 리스크가 겹친 지금은 풀사이즈 진입보다 손익비를 계산해 사이즈를 줄이는 구간입니다.

## 내 철학에서 본 이 기업

NBIX의 펀더멘털(1Q2026 EPS 서프라이즈 +14.4%, INGREZZA·CRENESSITY 동반 성장)과 기술적 신호(50일선 $159.85 위, 200일선 $143.39 위, 최근 사상 최고가 경신)는 마이클 마커스가 말한 "펀더멘털과 기술적 분석이 같은 방향을 가리키는" 상황에 해당합니다. 여기까지는 좋습니다.

문제는 두 가지입니다. 첫째, 현재가($172.04)는 사상 최고가($181.18) 대비 -5.0% 조정 중이며 이것이 건강한 눌림목인지 추세 전환의 초기 신호인지는 아직 확정할 수 없습니다. 둘째, 7/30 실적 발표가 15일 앞으로 다가와 있고, 직전 4개 분기 중 4Q2025는 -18.1% 서프라이즈 미스를 기록한 이력이 있습니다. 폴 튜더 존스라면 "얼마를 벌 수 있는가보다 얼마를 잃을 수 있는가"를 먼저 물었을 것이고, 지금 이 시점의 답은 "실적 발표 갭 리스크로 손절 규칙 자체가 무력화될 수 있다"는 것입니다.

## 밸류에이션 판단 (계산 과정)

슈웨거 방식은 내재가치 산출이 아니라 진입가·손절가·목표가·손익비·사이징의 계산입니다.

### 1) 진입가·손절가

```
현재가(진입 기준): $172.04
50일 이동평균: $159.85 (지지선 참고)
오닐 원칙(매수가 대비 -7~8% 손절):
  $172.04 × (1 - 0.08) = $158.28 ≈ 50일선과 근접
→ 손절가: $159.00 (50일선 이탈 확인 기준)
→ 손절 폭: ($172.04 - $159.00) / $172.04 = -7.6%
```

### 2) 목표가·손익비

```
1차 목표: 애널리스트 목표가 중앙값 $200.00
  상승 여력 = $200.00 - $172.04 = $27.96 (+16.3%)
  손익비 = $27.96 / $13.04 ≈ 2.14 : 1

2차 목표(추세 지속 시, 트레일링 스탑 전제): $250.65(최고 목표가)
  상승 여력 = $250.65 - $172.04 = $78.61 (+45.7%)
  손익비 = $78.61 / $13.04 ≈ 6.03 : 1
```
1차 목표 기준 손익비 2.14:1은 데니스 원칙("맞았을 때 크게, 틀렸을 때 작게")의 최소 기준(통상 2:1)을 겨우 충족하는 수준입니다. 압도적이지 않습니다.

### 3) 포지션 사이징 (자산 $100,000 가정, 비율 환산 가능)

```
단일 거래 리스크 상한 1.5% 적용:
  허용 손실액 = $100,000 × 1.5% = $1,500
  주당 리스크 = $172.04 - $159.00 = $13.04
  매수 가능 주식수 = $1,500 / $13.04 ≈ 115주
  포지션 규모 = 115주 × $172.04 ≈ $19,785 (자산의 19.8%)

보수적 대안(리스크 1.0% 적용, 실적 발표 임박 감안):
  허용 손실액 = $1,000
  매수 가능 주식수 = $1,000 / $13.04 ≈ 77주
  포지션 규모 ≈ $13,247 (자산의 13.2%)
```

### 4) 이벤트 리스크 — 7/30 실적 발표 시나리오 분석

```
차기 실적: 2026-07-30, 컨센서스 EPS $2.17, 매출 $879.6M
직전 4분기 서프라이즈: +9.6% / +4.9% / -18.1%(미스) / +14.4%

시나리오 A (서프라이즈 지속): 갭업 → $181 고점 재돌파 → 추세 추종
시나리오 B (미스 재현, 4Q2025 유형): 갭다운 → 손절가 $159 아래로 갭 이탈 가능
  → 이 경우 실제 손실이 사전 계산된 -7.6%를 초과(슬리피지)할 수 있음
```
갭 리스크는 지정가 손절로 완전히 막을 수 없습니다. 이것이 이번 판단에서 확신도를 낮추는 핵심 이유입니다.

## 리스크

- **실적 발표 갭 리스크(최우선)**: 7/30 실적 발표 전 풀사이즈 보유 시 4Q2025형 미스가 재현되면 손절 규칙이 무력화된다.
- **추세 전환 가능성**: 사상 최고가 대비 -5% 조정이 건강한 눌림목인지 전환 신호인지는 추가 확인 필요 — 50일선($159.85) 이탈 여부가 분기점.
- **파이프라인 리스크**: osavampator·direclidine Phase 3 톱라인이 2027년으로 예정, 시장이 성공을 상당 부분 선반영했다는 지적(데이터 팩 언급) — 트레이딩 기간 내 직접 리스크는 낮으나 밸류에이션 프리미엄의 근거이므로 참고.
- **경쟁 리스크**: Crinetics의 atumelnant가 CRENESSITY 대비 3년 내 경쟁 진입 가능 — 장기 보유 시 고려 요소.

## 최종 의견

제가 인터뷰한 위저드 중 누구도 "실적 발표 15일 전에 사상 최고가 근처에서 풀사이즈로 들어가라"고 말한 적은 없습니다. 추세는 살아있고(50일선, 200일선 모두 위), 손익비도 2.14:1로 최소 기준은 충족합니다. 하지만 4Q2025의 -18.1% 서프라이즈 미스 이력은 "이 종목은 실적 발표에서 크게 흔들릴 수 있다"는 경고입니다. 손절가 $159.00을 정해놓아도 갭다운으로 뚫릴 수 있다는 사실을 받아들여야 합니다.

이미 보유 중이라면 — 유지하되 실적 발표 전 사이즈를 리스크 1% 기준(자산의 13.2%)으로 낮추는 것을 권합니다. 신규 진입을 고려 중이라면 — 7/30 실적 확인 후 진입하거나, 지금 진입한다면 소규모(1% 리스크 기준)로 시작해 실적 이후 추세가 재확인되면 사이즈를 늘리는 방식이 원칙에 맞습니다. 희망은 전략이 아닙니다. "실적이 좋을 것 같아서" 풀사이즈로 들어가는 것은 분석이 아니라 예측이며, 저는 예측보다 대응을 권합니다.
