# EDU 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-15 | 투자의견: 관망 | 확신도: 4/10

## 한 줄 결론
펀더멘털은 나쁘지 않다 — 순현금 40억$, PEG 0.83, 매출 성장 지속 — 그러나 지금 이 종목은 "추세가 확인되지 않은 채, 이벤트 리스크(7/29 실적 + 중국 규제 뉴스)가 코앞에 걸려 있는" 전형적으로 내가 인터뷰한 위저드들이 회피했을 자리다. 손익비 계산은 해놓되, 진입은 실적 발표 이후 방향이 확인된 다음에 하라.

## 내 철학에서 본 이 기업
제가 인터뷰한 마켓 위저드들에게 이 차트를 보여준다면 다들 같은 질문을 먼저 던질 겁니다. "지금 추세가 위인가 아래인가?" 데이터를 보면 답은 명확합니다 — 아니오. 52주 변동률 -4.76%, S&P500 대비 +20.37%p 언더퍼폼, 52주 밴드 하위 30% 구간, 고점 대비 -23.7%. 3월 급등 후 조정, 5월 실적 호조에도 "sell the news"로 하락, 이후 박스권 하락. 윌리엄 오닐이라면 이 차트에 손도 대지 않았을 겁니다 — CAN SLIM의 "M"(시장 방향)과 "N"(신고가 근접)을 둘 다 충족하지 못하니까요.

동시에 마이클 마커스의 조건 — 펀더멘털과 기술적 신호가 같은 방향을 가리키는가 — 도 여기선 어긋나 있습니다. 펀더멘털(매출 YoY +19.8~19.8%, EPS 서프라이즈 +12.6%, PEG 0.83)은 위를 가리키는데 주가는 아래로 갑니다. 이 괴리는 펀더멘털이 틀렸다는 뜻이 아니라 시장이 "정책 리스크"라는 다른 변수를 가격에 반영하고 있다는 뜻입니다. 브루스 코브너라면 여기서 시나리오를 최소 두 갈래로 그렸을 겁니다: (1) 규제가 라이브커머스·성인교육까지 확대되는 시나리오, (2) 규제 우려가 과잉 반응으로 끝나고 실적이 계속 서프라이즈하는 시나리오. 지금 이 두 시나리오의 확률을 저는 판단할 근거(데이터 팩엔 규제 최종 결론 없음)가 없습니다.

## 밸류에이션 판단 — 진입가·손절가·목표가·손익비 계산

저는 내재가치 계산을 하지 않습니다. 대신 "이 거래에서 얼마를 벌 수 있고 얼마를 잃을 수 있는가"를 계산합니다.

### 1) 현재 상황 정리
| 항목 | 값 |
|---|---:|
| 현재가 | $49.59 |
| 52주 저가 | $41.62 |
| 52주 고가 | $64.97 |
| 다음 실적 발표 | 2026-07-29 (컨센서스 EPS $0.46, 매출 14.65억$) |
| 애널리스트 평균 목표가 | $70.61 (최저 $49.00 / 최고 $86.26) |

### 2) 손절가 산정 (오닐 7~8% 룰 적용, 이벤트 리스크 반영해 소폭 확대)
```
기계적 손절 (진입가 -8%)   = $49.59 × 0.92 = $45.62
구조적 지지선 (52주 저가)   = $41.62
채택 손절가                = $45.00 (진입가 대비 -9.3%)
```
7/29 실적과 중국 규제 뉴스라는 두 개의 갭 리스크가 동시에 걸려 있으므로, 순수 8% 룰보다 완충폭을 약간 늘려 $45.00에 둔다. 단, 실적 발표 전날 종가 기준으로 반드시 재점검 — 갭다운 시 $45.00은 무효가 되고 시가 기준 즉시 재평가한다(래리 하이트: "예측하지 말고 대응하라").

### 3) 목표가 산정
```
1차 목표 (기술적 저항선, 3월 급등 고점 부근 추정) = $58~60 (52주 고가 $64.97의 약 90~92%)
2차 목표 (애널리스트 컨센서스 중앙값)           = $72.00
보수적 목표 (진입가 대비 손익비 2:1 기준 역산)    = $49.59 + (49.59-45.00)×2 = $58.77
```

### 4) 손익비 계산 (진입가 $49.59 가정 시)
```
리스크(R) = 진입가 - 손절가 = $49.59 - $45.00 = $4.59 (진입가 대비 -9.3%)
보상(1차, $59 목표) = $59 - $49.59 = $9.41   → 손익비 2.05 : 1
보상(2차, $72 목표) = $72 - $49.59 = $22.41  → 손익비 4.88 : 1
```
1차 목표 기준 손익비 2:1은 리처드 데니스 기준("맞았을 때 수익이 틀렸을 때 손실보다 충분히 커야 한다")을 겨우 충족하는 수준. 2차 목표(애널리스트 컨센서스)까지 도달을 전제로 한 손익비는 매력적이나, 이는 "희망"이지 확정된 추세가 아니다.

### 5) 포지션 사이징 (전체 자산 대비 단일 거래 리스크 1~2% 한도, 하이트 기준)
```
가정: 전체 운용자산 = $100,000, 단일 거래 리스크 한도 = 1.5% = $1,500
1주당 리스크 = $4.59
최대 매수 가능 주식수 = $1,500 / $4.59 ≈ 326주
포지션 규모 = 326주 × $49.59 ≈ $16,167 (전체 자산의 16.2%)
```
이벤트 리스크(실적 갭, 규제 헤드라인 갭)가 걸려 있으므로 통상적 손절 실행이 보장되지 않는다 — 갭다운 시 손실은 $4.59/주를 초과할 수 있다. 따라서 실전에서는 위 사이징의 **50% 이하로 축소**(163주, 자산의 약 8%)하거나, 7/29 실적 발표 이후 진입을 권한다.

## 리스크
- **이벤트 리스크 ①: 7/29 실적 발표** — 직전 분기(FY26 Q1)는 컨센서스 대비 -2.5% 서프라이즈(미스), 직전전 분기(FY26 Q3)는 +12.6% 서프라이즈(비트). 방향성이 일관되지 않아 예측 불가. 5월 실적 발표 때도 "실적은 호조였으나 주가는 하락"(sell the news)한 전례가 있어, 숫자가 좋아도 주가가 오른다는 보장이 없다.
- **이벤트 리스크 ②: 중국 규제 뉴스** — 2026-03-23 라이브커머스·성인교육까지 "쌍감" 확대 가능성 시사. 라이브커머스(East Buy)는 매출의 약 22% 비중. 이 리스크는 갭다운으로 발현되며 손절 규칙이 정상 작동하지 않을 수 있다(슬리피지).
- **애널리스트 의견 극단적 분산**: Goldman·BofA 상향 vs HSBC·China Renaissance·Macquarie·Morgan Stanley 하향 — 이는 "시장 컨센서스가 없다"는 신호이자, 정책 불확실성이 크다는 방증.
- **ADR/HFCAA 지정학 리스크**: 상존. 순현금이 완충 역할은 하나 소거되지 않음.
- Beta 0.18은 통계적 착시다. 실제 변동성은 정책 헤드라인에 크게 좌우되며 베타로 리스크를 가늠할 수 없는 종목이다.

## 최종 의견
제가 인터뷰한 트레이더 중 누구도 "실적 발표 이틀 전, 하락 추세 한복판에서, 정책 리스크까지 겹친 종목"을 신규 매수하지 않았을 겁니다. 손익비 계산 자체는 2:1로 나쁘지 않습니다만, 손익비가 좋다고 진입 타이밍이 좋은 것은 아닙니다. 추세가 확인되지 않았고, 손절 규칙이 갭 리스크 앞에서 무력화될 수 있다는 것이 문제입니다.

기다리는 것도 유효한 포지션입니다. 7/29 실적을 확인하고, 그 반응으로 시장이 이 종목을 어느 방향으로 재평가하는지 본 다음 — 상승 갭 + 거래량 동반이면 그때 위 계산대로($45 손절, 2:1 이상 손익비) 진입을 고려하십시오. 지금 서둘러 들어갈 이유는 없습니다. 놓친 기회에 대한 아쉬움이 지금 이 판단에 개입하고 있지는 않습니까? 그렇다면 그것부터 먼저 점검하십시오.
