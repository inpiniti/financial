# QTTB 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-15 | 투자의견: 관망 | 확신도: 7/10

## 한 줄 결론
톱라인 뉴스는 이미 주가에 44.7배 거래량과 +109%로 반영됐고, 지금 이 순간 시장이 던지는 진짜 질문은 "임상이 성공했는가"가 아니라 "$200M 공모가 어디서 프라이싱되는가"다. 이 질문에 답이 나오기 전에 진입하는 것은 분석이 아니라 도박이다 — 제가 인터뷰한 위저드 중 누구도 이런 조건에서 풀사이즈로 들어가지 않았을 겁니다.

## 내 철학에서 본 이 기업
저는 펀더멘털이나 기술을 판단하는 사람이 아닙니다. 제가 볼 것은 하나입니다 — "이 거래가 틀렸을 때 어디서 빠져나올 것인가, 그리고 최대 얼마를 잃을 수 있는가"입니다. QTTB는 지금 세 가지 위저드적 경고등이 동시에 켜져 있는 종목입니다.

1. **뉴스 갭업 직후 추격매수 금지**: 마이클 마커스는 "포지션이 불리하게 움직이면 빠르게 빠져나와야 한다"고 했지만, 그 이전에 애초에 갭업 직후 뇌동매수를 피하는 것이 먼저입니다. +90~109% 장중 급등, 44.7배 거래량은 전형적인 "이벤트 이미 소화" 신호입니다.
2. **공모(딜루션) 오버행**: $200M 공모가 진행 중이고 공모가·확정 조달액이 확인 불가입니다. 발행주식 1,696만주에 시총 대비 $200M을 추가로 찍는다는 것은 현재 시총($313M)의 64%에 달하는 물량입니다. 공모가가 시장가보다 크게 할인되면 주가는 추가로 짓눌립니다. 이건 예측이 아니라 대응의 영역 — "가격을 알기 전엔 베팅하지 않는다."
3. **극저 float 유동성 함정**: float 259만주. 이런 종목은 스프레드가 넓고 슬리피지가 크며, 손절 주문이 의도한 가격에 체결되지 않을 위험이 있습니다. 래리 하이트라면 이런 유동성 조건에서 포지션 크기를 먼저 깎았을 겁니다.

## 밸류에이션 판단 — 손익비·손절·사이징 계산

슈웨거식 접근에서 밸류에이션은 "얼마가 적정가인가"가 아니라 "이 진입점에서 손익비가 얼마나 유리한가"입니다. 계산 과정을 공개합니다.

### 1) 현재 상황 정리

| 항목 | 값 |
|---|---|
| 현재가(2026-07-14 종가) | $18.46 |
| 톱라인 발표 전날 종가(추정, 52주 최고 -21.7% 역산 아님, 데이터팩 미기재) | 확인 불가 — 단 "+90~109% 급등"이므로 발표 전 약 $9~10 부근으로 역산 |
| 50일 이동평균(지지선 후보) | $10.24 |
| 200일 이동평균(장기 추세선) | $5.55 |
| 52주 최고 | $23.57 |
| 애널리스트 목표주가(평균/중앙값/최저) | $29.67 / $36.00 / $13.00 (n=3, 분산 큼, 참고용) |

### 2) 시나리오별 진입가·손절가·목표가·손익비

시장에 이미 참여 중인 투자자와, 지금 신규 진입을 고민하는 투자자를 구분해서 계산합니다.

**시나리오 A — 지금(급등 직후) 신규 진입할 경우 (비권장, 참고용)**

| 구분 | 가격 | 근거 |
|---|---|---|
| 진입가 | $18.46 | 현재가 |
| 손절가 | $15.50 (-16.0%) | 급등 시작가($9~10대) 갭의 하단 근처이자 심리적 저항이었던 구간. 단 저float 슬리피지 감안 시 실제 체결가는 더 낮을 수 있음 |
| 1차 목표가 | $23.57 (+27.7%) | 52주 최고가(직전 저항) |
| 2차 목표가 | $29.67 (+60.7%) | 애널리스트 평균 목표가(n=3, 참고용) |
| 손익비(1차 목표 기준) | 27.7% / 16.0% = **1.73 : 1** | 데니스 기준 최소선(2:1) 미달 |
| 손익비(2차 목표 기준) | 60.7% / 16.0% = **3.79 : 1** | 2차 목표까지 갈 때만 기준 충족, 그러나 공모 오버행이 상단을 누를 가능성 높음 |

**판정**: 1차 목표 기준 손익비 1.73:1은 리처드 데니스의 "맞았을 때 크게, 틀렸을 때 작게" 원칙에서 최소 기준(통상 2:1 이상)에 못 미칩니다. 게다가 손절가 $15.50 자체가 -16%라는 큰 폭인데, 이는 이미 하루 만에 100% 넘게 뛴 종목의 통상적인 변동성 밴드를 반영한 것으로, 손절폭이 좁지 않다는 것 자체가 "지금 진입은 나쁜 자리"라는 신호입니다.

**시나리오 B — 공모 완료·가격 확정 후 재진입 (권장 경로)**

공모가 발표 후 시장이 딜루션을 소화하고 나면, 통상 이런 소형 바이오텍은 공모가 부근에서 지지선을 형성합니다.

| 구분 | 가격(조건부) | 근거 |
|---|---|---|
| 트리거 | $200M 공모 확정 공모가 발표 + 완료 공시 | 8-K 확인 필요 |
| 예상 진입 구간 | 공모가 ~ 공모가+10% | 공모가가 신뢰할 만한 "바닥"을 제공 |
| 손절가 | 공모가 대비 -10~12% | 공모 참여 기관이 이 가격을 지지선으로 방어하지 않는다는 신호 |
| 목표가 | $23.57(52주고) → $29.67(애널 평균) | 임상 성공 자체는 유효하므로 펀더멘털 상단은 유지 |
| 손익비(보수적) | 목표 대비 상승여력 20~40% ÷ 손절폭 10~12% ≈ **2:1~3.3:1** | 데니스 기준 충족 가능 |

### 3) 포지션 사이징 (래리 하이트 방식 — 단일 거래 손실 1~2% 상한)

가정: 총 투자자산 $100,000, 단일 거래 최대 손실 한도 1.5%(중간값 채택 — 임상 바이오+저float 고변동성이므로 2%가 아닌 보수적 1.5% 적용) = **$1,500**

시나리오 A(현재가 진입, 비권장) 기준:
- 손절폭 = $18.46 - $15.50 = $2.96/주 (16.0%)
- 최대 매수 주식수 = $1,500 ÷ $2.96 ≈ **506주**
- 포지션 금액 = 506주 × $18.46 ≈ **$9,342 (총자산의 9.3%)**

→ 단일 종목 비중 9.3%는 이 정도 변동성(베타 -0.27이지만 개별 이벤트 변동성은 극단적, 최근 1년 수익률 +927.9%)의 임상 바이오텍치고 과도합니다. 마커스의 첫 교훈 "한 번의 거래에 전 재산을 걸지 말라"에 비추어, 저는 **손실 한도 자체를 자산의 1%로 낮추고, 포지션 비중도 5% 이하로 재차 제한**할 것을 권합니다.

재계산(1% 한도, $1,000):
- 최대 매수 주식수 = $1,000 ÷ $2.96 ≈ **337주**
- 포지션 금액 = 337주 × $18.46 ≈ **$6,222 (총자산의 6.2%)**

이마저도 float 259만주의 유동성 함정을 고려하면, 실제 체결 슬리피지를 감안해 계획 물량의 70% 수준(약 $4,300, 자산의 4.3%)으로 시작해 분할 진입하는 것이 하이트식 분산 원칙에 부합합니다.

## 리스크
- **공모가 리스크(최대 변수)**: $200M 신주공모의 최종 공모가·조달액 확인 불가. 시장가보다 대폭 할인된 가격에 확정되면 주가는 공모가 부근까지 즉시 되돌림 가능. 이것이 지금 관망을 권하는 핵심 이유.
- **극저 float 유동성 함정**: float 259만주로 손절 주문 체결가가 계획보다 크게 밀릴 위험. 지정가·분할 주문 필수.
- **임상 바이오 고유 리스크**: Phase 2a는 최종 승인까지 먼 단계. Phase 3(등록목적 프로그램) 진입은 2027년 상반기 계획일 뿐 확정 아님. SIGNAL-AD 프로그램이 1차 endpoint 실패했던 전례가 이미 존재 — 같은 회사, 같은 약물이라도 적응증별 결과는 다를 수 있음을 이미 보여줌.
- **공매도 커버 vs 신규 공매도**: 공매도 비율 8.31%로 전월 대비 감소 중이나, 공모 소식 이후 신규 공매도 유입 가능성 — 추가 조사 필요.
- **현금 자체는 견조**: 유동비율 5.36, 당좌비율 4.81로 단기 지급능력은 우려 없음. 리스크는 성장성/이벤트 변동성이지 파산 위험이 아님.

## 최종 의견
제가 인터뷰한 트레이더들에게 이 차트를 보여주면 다들 같은 질문을 던질 겁니다 — "그 44.7배 거래량 폭발에 올라타지 못한 게 아쉬워서 지금 쫓아가려는 건가, 아니면 새로운 좋은 자리가 보여서인가?" 답이 전자라면 이건 추격매수이고 놓친 기회를 쫓는 전형적인 패턴입니다. 톱라인은 이미 시장에 소화됐고, 지금 주가를 움직일 다음 변수는 $200M 공모가 얼마에 프라이싱되느냐입니다. 그 숫자를 모른 채 베팅하는 것은 예측이지 대응이 아닙니다. 저는 공모가 확정될 때까지 기다리겠습니다 — 기다리는 것도 유효한 포지션입니다. 만약 지금 당장 참여해야 한다면, 손절가 $15.50을 예외 없이 지키고 포지션은 자산의 5% 이내로, 그리고 슬리피지를 감안해 계획 물량의 70%로 시작하십시오. 희망은 전략이 아닙니다.
