# EDU 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-15 | 투자의견: 매수 | 확신도: 6/10

## 한 줄 결론
좋은 기업(ROC 약 22%)을 순현금 덕분에 싸게(EY 약 11.5%) 살 수 있는 자리이지만, 두 지표 모두 "압도적 1등급"은 아니고 중국 ADR 규제 리스크가 공식 밖의 그림자로 남아있다 — 그래도 공식은 공식이다, 산다.

## 내 철학에서 본 이 기업
제이슨의 껌 가게 비유로 말해보자. EDU는 49억 달러를 벌어들이는(FY2025 매출) 사업인데, 그 사업을 지금 시장은 42.7억 달러(EV)에 팔겠다고 한다 — 시총 82.7억 달러라고 하지만 회사 금고 안에 순현금 40억 달러가 들어있으니 실제 사업 자체의 가격표는 그보다 훨씬 싸다. 나는 스토리를 보지 않는다. ROC와 EY, 두 숫자만 본다.

## 체크리스트 평가

| 항목 | 판정 | 근거 |
|---|---|---|
| 자본수익률(ROC) 높은가 | △ | ≈21.9%. 양호하나 마법공식 최상위 데실(30%↑)에는 못 미침. 고정자산 원자료 부재로 근사치 |
| 이익수익률(EY) 높은가 | ○ | ≈11.5%. 순현금 40억$ 덕분에 EV가 시총의 절반 수준으로 축소되어 확보된 수치 |
| 규모 필터 통과 | ○ | 시총 82.7억$, 유니버스 제외 대상 아님 |
| 좋으면서 동시에 싼가 | △ | 두 지표 다 합격선이나 어느 하나도 압도적이지 않음 — "고르게 우수"에는 해당 |
| 스토리·뉴스 개입 여부 | ○ | 중국 규제 리스크, ADR 상장폐지 우려 등 나쁜 뉴스로 가격이 눌린 정황 — 공식이 포착해야 할 바로 그 패턴 |

## 밸류에이션 판단 (마법공식 계산 과정)

**1) EBIT (FY2025 기준)**
```
매출 49.00억$ × 영업이익률 10.0% = EBIT ≈ 4.90억$
```

**2) 투입 유형자본 (순운전자본 + 순고정자산, 근사)**
```
순운전자본 = 유동자산 54.27억 − 유동부채 32.79억 = 21.48억$
순고정자산(근사) = 자기자본 40.86억 + 총부채 8.07억 − 총현금 48.02억 ≈ 0.91억$
투입자본 ≈ 21.48억 + 0.91억 = 22.39억$
```
※ PP&E 원자료 미확인으로 근사치. 순고정자산이 실제로는 더 클 가능성 있음(과소평가 시 ROC 과대계상 가능).

```
ROC = EBIT ÷ 투입자본 = 4.90억 ÷ 22.39억 ≈ 21.9%
```

**3) 기업가치(EV)**
```
EV = 시가총액 82.67억 + 총부채 8.07억 − 총현금 48.02억 = 42.72억$
```
(데이터 팩 §5 제시 EV 41.5억$과 근사 일치, 기준일 차이로 소폭 오차)

**4) 이익수익률(EY)**
```
EY = EBIT ÷ EV = 4.90억 ÷ 42.72억 ≈ 11.5%
```

**5) 순현금 효과 검증 — "만약 순현금이 없었다면?"**
```
EY(시총 기준, 순현금 무시) = 4.90억 ÷ 82.67억 ≈ 5.9%
EY(EV 기준, 순현금 반영)  = 11.5%  → 약 1.9배 확대
```
순현금 40억$이 EY를 5.9%에서 11.5%로 거의 두 배 끌어올렸다. 이것이 마법 공식이 "인수자 관점"을 강조하는 이유다 — 시가총액만 보면 이 회사가 싼지 알 수 없다. 사업 전체를 통째로 살 때 실제 치르는 값은 순현금을 뺀 42.7억$이다.

**6) 종합 판정**

| 지표 | 값 | 벤치마크 대비 | 등급 |
|---|---:|---|---|
| ROC | 21.9% | 30%↑가 최상위권, 20%대는 양호~보통 | 중상 |
| EY | 11.5% | 10%↑면 매력적, 6~8%대는 평범 | 상위 |

두 지표 순위를 합산하면 "좋은 기업 + 싼 가격"의 조합은 성립하되, ROC가 극단적으로 높은 완전체는 아니다. 다만 EY 11.5%는 국채 금리 대비로도 매력적이고, 이게 순전히 순현금 40억$ 덕분에 만들어진 숫자라는 점이 핵심 논거다.

## 리스크
- ROC 계산의 분모(순고정자산)가 원자료 부재로 근사치 — 실제 PP&E가 더 크면 ROC는 낮아질 수 있다.
- 중국 규제("쌍감" 확대 적용 가능성, 라이브커머스 매출 22% 비중)는 EBIT의 미래 지속가능성을 흔들 수 있는 질적 리스크 — 마법 공식 두 지표에는 담기지 않는다.
- HFCAA발 ADR 상장폐지 우려는 낮지만 상존. 다만 순현금 우위가 완충 요인.
- 애널리스트 의견 극단적으로 분산(HSBC·China Renaissance·Macquarie·Morgan Stanley 하향 vs Goldman·BofA 상향) — 시장도 확신을 못 갖고 있다는 신호.

## 최종 의견
숫자로 말하자. ROC 약 22%, EY 약 11.5% — 좋은 기업을, 순현금 40억 달러 덕분에 싸게 살 수 있는 자리다. "경영진이 훌륭하다"거나 "중국 정책이 무섭다"는 얘기는 내 공식에 넣지 않는다. 나쁜 뉴스 때문에 가격이 싸진 것이라면, 바로 그게 공식이 찾는 기회다. ROC가 30%를 넘는 최상위권은 아니라서 확신도를 만점 줄 순 없지만, 두 지표 모두 합격선을 넘었고 순현금이라는 확실한 안전판까지 있다. 나는 이런 종목을 20~30개 바구니에 담아 1년 들고 가는 사람이다 — 개별 종목 하나의 스토리에 겁먹지 않는다. 산다.
