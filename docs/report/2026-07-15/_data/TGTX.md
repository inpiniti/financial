# TGTX (TG Therapeutics, Inc.) 데이터 팩

조회일: 2026-07-15 (Yahoo Finance, SEC EDGAR 기준)

## 1. 기업 개요
- **사업 모델**: 상업단계 바이오제약사. 재발성 다발성경화증(RMS) 치료제 **BRIUMVI(ublituximab)**를 미국·유럽에서 판매 중인 CD20 표적 항체. 단일 주력 제품이 매출의 사실상 전부를 차지.
- **파이프라인 확장**: 2026-07-06 치료저항성 조현병(schizophrenia) 대상 Briumvi Phase 2 임상 개시 — 적응증 확장 시도.
- **경쟁사**: Roche(Ocrevus), Novartis(Kesimpta), Biogen(Tysabri) 등 MS/CD20 계열 치료제. BRIUMVI는 후발주자이나 짧은 투여시간(1시간 유지 투여) 강점.
