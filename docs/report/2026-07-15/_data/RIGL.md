# RIGL (Rigel Pharmaceuticals, Inc.) 공용 데이터 팩

조회일: 2026-07-15 (시세는 2026-07-15 조회 시점 정규장 기준, 재무는 최근 공시/Yahoo 데이터 기준)

## 1. 기업 개요

- **사업 모델**: 혈액질환·종양학 분야 상업화 단계 바이오텍. 자체 개발 저분자 신약을 미국 내 직접 판매(자체 영업조직)하며, 파트너십 계약금·마일스톤·로열티도 수취.
- **주요 제품**: TAVALISSE(성인 만성 면역혈소판감소증, SYK억제제), REZLIDHIA(IDH1변이 재발/불응 급성골수성백혈병), GAVRETO(RET融合 비소세포폐암 등, 2025년 화이자로부터 인수), 2026-06 신규 편입 **VEPPANU**(vepdegestrant, ER+/HER2- 전이성 유방암, 세계 최초 FDA승인 PROTAC, 2026-08 상업화 예정 — Arvinas·Pfizer로부터 라이선스, 업프론트 $70M+조건부 $15M 지급).
- **파이프라인**: R289(IRAK1/4억제제, 2026년 하반기 2상 용량 확정 예정). 2026-04 Eli Lilly가 ocodusertib(RIPK1억제제) 협업 종료 통보(2026-06-15 발효) — 파트너십 축소 이슈.
- **경쟁사**: (ITP) Novartis(Promacta), Amgen(Nplate); (AML/IDH1) Servier(Tibsovo), BMS; (RET+NSCLC) Blueprint Medicines(Gavreto 오리지널 개발사), Eli Lilly(Retevmo); (PROTAC/유방암) Arvinas·Pfizer(자체), AstraZeneca(Faslodex).
- **CIK**: 0001034842 (SEC EDGAR)
- 참고: 2024-06-27 **1:10 역병합(reverse split)** 단행 — 과거 저가주(수$) → 현재 주가·EPS 스케일이 그 이전과 불연속.
