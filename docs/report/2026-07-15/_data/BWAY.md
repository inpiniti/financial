# BWAY (BrainsWay Ltd.) 데이터 팩 — 조회일 2026-07-15

## 1. 기업 개요
- **사업**: 이스라엘(예루살렘) 소재 의료기기 기업. 특허 H-coil 기술 기반 **딥 경두개 자기자극(Deep TMS)** 장비 제조·판매. 비침습 신경자극 치료.
- **적응증**: 주요우울장애(MDD), 강박장애(OCD), 금연보조, (PTSD·조현병·자폐 등은 연구/오프라벨 확장 중)
- **매출 구조**: 장비 판매 + 치료 세션당 소모품/구독(ARR) 모델로 전환 중, 총마진 ~75% 유지
- **주요 고객**: 정신과 병의원, 병원
- **경쟁사**: Neuronetics(STIM, 시장점유율 최대 ~18%), MagVenture(~15%), Nexstim(~14%), BrainsWay(~12%, 2026 추정). 기타: MagStim, Cloud TMS. 대체재: 항우울제, 케타민, ECT/VNS/DBS(Medtronic·LivaNova 등)
- **상장**: NasdaqGM, ADR. 2026-03-04 **2:1 액면분할** 시행(본 팩의 EPS·주가는 분할 반영 기준)
