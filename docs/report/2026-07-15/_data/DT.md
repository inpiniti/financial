# DT (Dynatrace, Inc.) 공용 데이터 팩

조회일: 2026-07-15 (Yahoo Finance, SEC EDGAR 기준) | 회계연도: 4/1~익년 3/31 (FY2026 = 2025-04~2026-03)

## 1. 기업 개요

- **사업 모델**: 클라우드 기반 옵저버빌리티(Observability)·애플리케이션 성능 모니터링(APM)·보안(AppSec) 통합 플랫폼 SaaS 기업. AI 기반 자동 근본원인 분석("Davis AI")이 핵심 차별화.
- **주요 세그먼트**: 단일 플랫폼(Dynatrace Platform) 구독 매출이 대부분(구독 매출 비중 ~96%). Grail 데이터 레이크하우스, 로그 관리, 클라우드 자동화(AI/에이전틱 옵저버빌리티) 신규 모듈 확장 중.
- **경쟁사**: Datadog, Splunk(Cisco 인수), New Relic(비상장 전환), Elastic, AppDynamics(Cisco), Grafana Labs.
- **최근 지배구조 이슈**: 행동주의 헤지펀드 Starboard Value LP가 지분 확보 후 비용절감·자사주매입 압박 → 2026-07-01 이사회에 George Riedel, Dan Streetman 영입(8명→10명), $1B 자사주매입 승인, FY2029 "Rule of 40→50" 목표를 FY2027 Q2 실적 후 Investor Day에서 공개 예정. (출처: [citybiz](https://www.citybiz.co/article/868469/dynatrace-adds-george-riedel-and-dan-streetman-to-board-sets-investor-day-plans/), [StreetInsider](https://www.streetinsider.com/news.php?id=26716433&classic=1), 2026-07-01)
