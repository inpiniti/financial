## 1. 기업 개요

| 항목 | 내용 |
|---|---|
| 사명 | DRDGold Limited (NYSE ADR: DRD, JSE: DRD) |
| 본사 | 남아프리카공화국 요하네스버그 |
| 섹터/산업 | Materials / 금 채굴 — 지표 광미(surface tailings) 재처리 |
| 시가총액 | **$1.80B** (2026-07-14, marketCap $1,798.1M) |
| 발행주식수 | 8,673만 주 |
| 통화 | 사업 실적은 ZAR, ADR 주가는 USD |

**사업 모델**: 신규 채굴이 아닌 **비트바테르스란트(Witwatersrand) 분지 100년치 옛 금광 폐기물(광미 더미) 재처리**가 핵심. Ergo(요하네스버그 인근)와 FWGR(파웨스트 골드 리커버리, West Rand)의 CIL(carbon-in-leach) 공정으로 저품위 광미에서 금을 재추출. 지하 채굴 대비 자본지출·안전 리스크가 낮고 ESG 측면에서 환경 정화 효과(폐기물 처리) 부각. "Vision 2028" 목표: 연간 생산량 20만 온스 이상으로 확대(대형 신규 프로젝트 투자 포함).

**경쟁사**: Harmony Gold(HMY, 남아공 최대 금광사, 광미+지하 채굴 겸업), AngloGold Ashanti(AU), Gold Fields(GFI). 순수 광미 재처리 사업자로는 DRD가 독보적 — 직접 경쟁자가 거의 없는 틈새 사업.
