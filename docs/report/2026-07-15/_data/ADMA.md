# ADMA Biologics (ADMA) 공용 데이터 팩

조회일: 2026-07-15 (주가 기준일: 2026-07-14 종가)

## 1. 기업 개요

- **사업 모델**: 미국 혈장(plasma) 유래 바이오의약품 기업. 자체 FDA 승인 혈장채취소(collection center)와 플로리다 보카레이턴 생산시설을 보유한 수직계열화 구조 (원료 혈장 외부 의존도 축소).
- **주요 제품**: ASCENIV(주력, RSV 항체 함유 차별화 IVIG, 원발성 체액성 면역결핍증 PIDD 치료), BIVIGAM(범용 IVIG), Nabi-HB(B형간염 하이퍼면역글로불린).
- **경쟁사**: CSL Behring, Grifols, Takeda(3대 글로벌 혈장분획 대기업) — 미국 혈장유래제제 4대 생산사 중 하나. Octapharma, BPL/Kedrion도 경쟁.
