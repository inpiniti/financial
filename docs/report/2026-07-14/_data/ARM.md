- **섹터/산업**: Technology / Semiconductors
- **사업 모델**: **반도체 IP 라이선싱** — 직접 칩을 만들지 않는다(팹리스도 아닌 IP 제공자).
  - **라이선스 매출**: 고객사가 Arm의 CPU 아키텍처 설계를 쓸 권리를 사는 대가 (선불)

- **경쟁/대체 위협**: x86(인텔·AMD), **RISC-V**(오픈소스 무료 아키텍처 — 로열티 모델 자체를 위협), 고객사의 자체 설계 내재화
- **지배구조**: **소프트뱅크가 지분 약 88%를 보유**(데이터상 기관 지분 95.94%에 포함). 유통주식(float)이 매우 작다 → 주가 변동성 증폭 요인
  - **내부자 지분율 0.07%** (사실상 없음)
- **최근 전략**: 자체 **AGI CPU**로 AI 인프라에 직접 진출, FY2027~2028 고객 확약 **$2B 이상** 확보 (출처: Motley Fool 2026-07-09)
- **ISS 지배구조 리스크 스코어**: 데이터 없음 — **확인 불가**
