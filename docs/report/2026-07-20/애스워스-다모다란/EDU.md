# EDU 투자 보고서 — 애스워스 다모다란 (주식 가치평가 완벽 정리)
> 날짜: 2026-07-20 | 투자의견: 매수 | 확신도: 7/10

## 한 줄 결론
현재가 $49.15는 제 세 시나리오 중 가장 비관적인 축("규제 리스크 지속 + 순현금 절반 헤어컷")에 근접해 있고, 표준적인 중국 국가위험프리미엄을 반영한 WACC(약 10.1%)로 계산한 Base 시나리오 가치($93.65)와는 거의 두 배 차이가 납니다. 이 갭 자체가 시장이 얼마나 큰 리스크 프리미엄을 EDU에 요구하고 있는지를 보여주는 신호이며, 그 프리미엄이 정당한지 아닌지가 이 보고서의 핵심 질문입니다.

## 내 철학에서 본 이 기업

EDU의 스토리는 명확합니다: 2021년 "쌍감정책"으로 K9 학과 교습이라는 핵심 사업이 붕괴(매출 -90% 수준의 트라우마)한 뒤, 비학과 튜터링·성인교육·해외유학·라이브커머스(둥팡전쉬안/East Buy)로 사업을 재편해 FY2023부터 재성장 국면에 들어섰습니다. 매출은 FY2023 $3.00B → FY2025 $4.90B로 회복했고, 영업이익률도 6.3%→10.0%로 개선 중입니다. 가치 드라이버 파일이 짚은 대로 K-12 비학과 튜터링(+23% YoY, FQ3 FY26)이 재성장의 핵심 엔진이고, 규제 리스크(2026년 3월 베이징의 재차 통제 강화 방침)는 여전히 살아있는 변수입니다.

이 기업을 평가할 때 가장 중요한 것은 **숫자와 스토리의 일치**입니다. "규제 이후 재편에 성공한 성장 기업"이라는 스토리라면, FCF가 안정적으로 늘어나야 하는데 실제로는 FY2024 $839.3M → FY2025 $637.4M로 오히려 줄었습니다. 이는 스토리가 거짓이라기보다, 학습센터 캐파 확장(연 10~14% 가이던스)에 따른 자본적 지출이 현금흐름을 일시적으로 눌렀다는 것으로 해석됩니다 — 재투자 국면의 성숙 기업이 아니라 "재성장을 위해 재투자 중인 회복기 기업"이라는 유형으로 봐야 합니다.

또 하나 특이한 점은 **순현금 $4.0B이 시가총액 $8.19B의 절반에 육박**한다는 것입니다. 이는 밸류에이션에서 두 가지 갈림길을 만듭니다: (1) 이 현금을 액면 그대로 인정하면 EDU는 극단적으로 저평가된 기업이고, (2) VIE 구조·중국 내 규제로 인한 트랩드 캐시(자본 통제·해외송금 제한) 리스크를 감안해 현금에 헤어컷을 적용하면 그림이 달라집니다. 저는 후자를 기본으로 삼되, 헤어컷의 크기를 시나리오별로 달리 적용했습니다.

## 밸류에이션 판단

### 1. 할인율 산정 — 중국 국가위험프리미엄 명시 반영

베타 0.18(Yahoo)은 신뢰하지 않습니다. ADR이 미국 시장과 낮은 상관관계를 가질 뿐, 실제 사업위험을 반영하지 못하는 통계적 허상입니다. 교육서비스 업종의 상향식(bottom-up) 비levered 베타를 약 0.90으로 놓고, EDU의 자본구조로 재levered합니다.

```
D/E = 총부채 $803.8M / 자기자본(시총 기준 근사) ≈ 9.8%
세율(중국 법인세) ≈ 25%
Relevered Beta = 0.90 × (1 + (1-0.25) × 0.098) = 0.90 × 1.0735 ≈ 0.97
```

**자기자본비용(Ke) = Rf + β×성숙시장ERP + λ×중국CRP**

| 변수 | 값 | 근거 |
|---|---|---|
| 무위험이자율(Rf) | 4.3% | 2026-07 시점 미국 10년물 근사 |
| 성숙시장 ERP | 4.5% | 다모다란 방식 미국 내재 ERP 근사 |
| 상향식 베타(재levered) | 0.97 | 교육서비스 업종 비levered 0.90 × 재levered |
| 중국 국가위험프리미엄(CRP) | 2.0% | 중국 국가신용등급 스프레드 × 주식시장 변동성 배수(통상 1.5~2배) 적용 근사 |
| λ(국가위험 노출도) | 1.0 | 매출 거의 전량이 중국 본토(VIE) 발생 |

```
Ke = 4.3% + 0.97×4.5% + 1.0×2.0% = 4.3% + 4.365% + 2.0% = 10.67% ≈ 10.7%
```

**WACC**

```
E/V = 시총 $8.19B / (시총+부채 $8.99B) = 91.1%
D/V = 8.9%
Kd(세후) = 5.0%(중국 회사채 근사) × (1-0.25) = 3.75%
WACC = 0.911×10.7% + 0.089×3.75% = 9.75% + 0.33% ≈ 10.1%
```

이 10.1%가 "표준적인 중국 CRP만 반영한" 순수 WACC입니다. Bear·Bull 시나리오에서는 여기에 규제 리스크 강도에 따라 추가 프리미엄(또는 완화)을 얹었습니다 — "결론을 위해 할인율을 조정"하는 것이 아니라, 리스크의 크기가 다른 시나리오이므로 할인율도 달라야 한다는 원칙에 따른 것입니다.

### 2. 3단계 시나리오 DCF (5개년 명시적 예측 + 터미널가치)

FY2025 매출 $4.90B을 출발점으로, 순현금은 VIE·자본통제로 인한 "트랩드 캐시" 리스크를 반영해 시나리오별로 헤어컷을 달리 적용했습니다(전액 인정 시 저평가가 극단적으로 과장되기 때문).

| 항목 | Bear | Base | Bull |
|---|---|---|---|
| 매출성장(FY26→FY30) | 8/5/3/2/2% | 12/10/8/6/5% | 16/14/12/10/8% |
| FCF마진(정상화) | 10% | 14% | 17% |
| WACC | 12.0% (규제 재강화 반영 프리미엄) | 10.1% (순수 CRP) | 9.0% (규제리스크 완화·재평가) |
| 영구성장률 | 2.0% | 3.0% | 3.5% |
| 순현금 헤어컷 | 50% | 30% | 10% |
| PV(5개년 FCF) | $2,031.7M | $3,365.6M | $4,680.3M |
| PV(터미널가치) | $3,446.6M | $9,111.6M | $17,928.7M |
| 기업가치(EV) | $5,478.3M | $12,477.2M | $22,609.0M |
| 조정 순현금 | $1,997.7M | $2,796.8M | $3,595.9M |
| 자기자본가치 | $7,476.0M | $15,274.0M | $26,204.9M |
| **주당가치**(희석 163.1M주) | **$45.84** | **$93.65** | **$160.67** |
| 주당가치(완전희석 166.7M주 기준) | $44.85 | $91.63 | $157.20 |

터미널가치가 Base 기준 EV의 73%, Bull 기준 79%를 차지합니다 — 제가 항상 경고하는 바로 그 함정입니다. 영구성장률·WACC의 작은 변화가 결과를 크게 흔드니, Bull 시나리오의 $160대는 "가능하지만 낮은 확률의 상방"으로 읽어야지 목표가로 쓸 숫자가 아닙니다.

참고로 별도의 "2021년 규제 재현" 극단 시나리오(매출성장 5/3/2/1/1%, FCF마진 8%, WACC 13%, 영구성장 1%, 현금헤어컷 60%)를 계산하면 주당가치는 **$31 내외**로, 현재가 $49.15보다 낮습니다.

### 3. 역산 DCF — 시장은 무엇을 가격에 반영하고 있는가

현재 시가총액(약 $8.19B)에서 조정 순현금(Base 가정, 30% 헤어컷 $2,796.8M)을 제외하면 시장이 사업가치(EV)에 매기는 값은 약 **$5,393.2M**입니다. Base 시나리오의 5개년 FCF 경로를 그대로 두고, 이 EV를 정당화하는 조건을 역산합니다.

**(a) 영구성장률 역산(WACC 10.1% 고정)**
```
목표 EV $5,393.2M - PV(5개년 FCF) $3,365.6M = PV(TV) 필요값 $2,027.6M
TV(FY30 시점) = $2,027.6M / 0.6182 = $3,280.0M
3,280.0 = 1,015.9×(1+g) / (0.101-g)  →  g ≈ -15.9%
```
FCF가 영구히 매년 -15.9%씩 축소된다는 뜻으로, 문자 그대로는 비현실적입니다. 이는 "고든성장 모형으로는 설명 안 되는 가격"이라는 신호이며, 시장이 사실 정교한 영구성장률 모형이 아니라 확률가중 파국 시나리오(재규제 리스크)를 뭉뚱그려 할인하고 있음을 시사합니다.

**(b) 내재 할인율 역산(영구성장률 2.5% 고정, 반복계산)**
```
r=18%일 때 EV≈$5,693.8M, r=19%일 때 EV≈$5,336.7M → 보간하면 r≈18.7~18.8%
```
시장의 내재 WACC는 약 **18.7%**로, 제가 계산한 순수 중국 CRP 반영 WACC 10.1%보다 8.6%p 높습니다. 이 8.6%p가 바로 "표준 국가위험프리미엄을 초과하는, EDU 고유의 규제 테일리스크 프리미엄"입니다. 이 프리미엄이 과도한지 정당한지가 투자 판단의 핵심입니다 — 2021년 실제로 매출 90%가 증발한 전례가 있었으니, 완전히 근거 없는 공포는 아닙니다. 다만 그 전례 이후 4년간 회사는 비학과·해외·라이브커머스로 규제 노출도를 실질적으로 낮췄습니다.

### 4. 상대가치 교차검증

| 티커 | Fwd PER | EV/EBITDA | 영업이익률 |
|---|---|---|---|
| EDU | 11.43x | 5.21x | 12.7% |
| TAL | 9.85x | 8.01x | 9.0% |
| GOTU | 15.65x | 3.14x | 0.4% |

EDU는 세 회사 중 영업이익률이 가장 높은데도 EV/EBITDA는 TAL보다 낮습니다(5.21x vs 8.01x). 이 배수의 동인은 재투자 필요량·세율·성장인데, EDU의 순현금이 EV를 깎아내려 배수가 인위적으로 낮아 보이는 효과도 있습니다. 그럼에도 이익률 대비 배수가 낮다는 점은 DCF의 저평가 결론과 방향이 일치합니다 — 교차검증 통과입니다.

## 리스크

1. **규제 재현 리스크(최대 리스크)**: 2026년 3월 베이징의 재차 통제 강화 방침. 2021년 전례처럼 서프라이즈성 규제가 나오면 극단 시나리오($31)로 즉각 수렴 가능.
2. **트랩드 캐시 리스크**: 순현금 $4.0B의 상당 부분이 VIE 구조·중국 자본통제 하에 있어, 헤어컷 없이 액면가로 인정하는 것은 위험합니다. 제가 30~50% 헤어컷을 적용한 이유입니다.
3. **중국 학령인구 감소**: 저출산으로 인한 구조적 학생 수 감소는 GDP 성장률 수렴과는 별개의, 업종 고유의 장기 역풍입니다. 영구성장률을 일반 경제성장률(2~3%)보다 더 보수적으로 잡아야 할 근거가 됩니다.
4. **추정 오차**: 이 모든 계산은 제 가정(성장률·마진·할인율·헤어컷)에 전적으로 의존합니다. Bull과 Bear의 격차가 3.5배(주당 $46~$161)라는 것 자체가 불확실성의 크기를 말해줍니다.
5. **ADR/상장폐지 리스크**: 데이터 팩에서 확인 불가로 표시된 항목(공식 환헤지 여부, 정확한 완전희석 EPS)은 추가 확인이 필요합니다.

## 최종 의견

제 가치 추정은 거의 확실히 틀렸을 겁니다. 하지만 범위는 합리적입니다. Bear $46, Base $94, Bull $161 — 이 범위에서 현재가 $49.15는 하단에 바짝 붙어 있습니다. 역산 결과 시장의 내재 할인율은 18.7%로, 제가 표준적인 중국 국가위험프리미엄으로 계산한 10.1%보다 8.6%p나 높습니다. 이 초과분이 정당화되려면 2021년 규모의 규제 충격이 다시 벌어져야 하는데, 지금 회사는 그때와 달리 비학과·해외유학·라이브커머스로 사업을 상당히 다각화한 상태입니다. 스토리가 나아졌는데 시장이 요구하는 리스크 프리미엄은 그대로거나 더 커 보인다면, 이것이 바로 가격과 가치의 괴리가 만드는 기회입니다. 다만 터미널가치가 전체 가치의 70% 이상을 차지하는 모형이라는 것, 그리고 중국 규제라는 진짜 꼬리위험이 남아있다는 것 — 이 두 가지를 인정한 채로 매수합니다. 확신도를 9나 10이 아니라 7로 매기는 이유입니다.
