# 402340 투자 보고서 — 잭 슈웨거 (마켓 위저드)
> 날짜: 2026-07-20 | 투자의견: 관망 (조건부 진입 계획 보유) | 확신도: 6/10

## 한 줄 결론
지금 SK스퀘어는 14배 급등 뒤 고점 대비 -46% 급락 중인 종목이다 — 추세가 아직 확인되지 않았으니 지금 매수는 "떨어지는 칼날을 잡는" 행위다. 손절·목표·사이징을 미리 정해놓고, 추세가 스스로 방향을 알려줄 때까지 기다린다.

## 내 철학에서 본 이 기업
저는 개별 종목이 "싸냐 비싸냐"를 논하기 전에 항상 이렇게 묻습니다. "지금 시장(그리고 이 종목)의 추세는 어느 방향인가?" SK스퀘어는 2025년 7월 153,000원에서 2026년 6월 2,189,000원까지 약 14배 급등했습니다. 폴 튜더 존스라면 이런 포물선형 상승을 보고 "얼마를 벌 수 있는가보다 얼마를 잃을 수 있는가를 먼저 생각하라"고 했을 겁니다. 그리고 정확히 그 우려대로, 7월 들어 -14.98%, -8%대, -7.27% 같은 단일일 급락이 연쇄적으로 터지면서 현재가는 1,177,000원, 고점 대비 -46.2%까지 밀렸습니다.

이건 펀더멘털 이야기가 아닙니다(SK하이닉스 26.1Q 영업이익 +400%, SK스퀘어 지분법이익 폭증 — 펀더멘털은 오히려 개선 중입니다). 문제는 가격 움직임 자체가 "추세가 꺾였다"는 신호를 보내고 있다는 겁니다. 마이클 마커스라면 "펀더멘털과 기술적 신호가 같은 방향을 가리킬 때 확신을 갖는다"고 했는데, 지금은 펀더멘털(개선)과 가격 추세(붕괴)가 정반대 방향입니다 — 바로 이 괴리가 제가 신중해지는 이유입니다.

베타 2.38은 이 종목이 SK하이닉스의 레버리지된 대리 종목(proxy)이라는 것을 숫자로 보여줍니다. 밸류에이션 논리(NAV 할인율 44~46% → 목표 30%, Forward PER 3~4x)는 매력적이지만, 저는 밸류에이션만으로 진입 타이밍을 정하지 않습니다. "싸다"는 이유로 하락 추세 중인 종목을 잡으려는 유혹 — 이게 제가 인터뷰한 트레이더들이 가장 많이 경계한 함정입니다.

## 체크리스트 평가

| 체크리스트 항목 | 평가 | 근거 |
|---|---|---|

| 펀더멘털·기술적 신호가 같은 방향인가 | × | 펀더멘털(SK하이닉스 지분법이익 +419%, 실적 개선)은 상승, 가격 추세는 하락 — 마커스 기준 "확신을 가질 수 없는" 괴리 상태 |

| 최대 손실이 감당 가능한가(1~2% 룰) | ○ | 하이트 기준 단일 거래 리스크 1%로 설계, 포지션 크기로 통제 |
| 여러 시나리오를 그렸는가 | ○ | 관망 후 추세확인 진입 시나리오 + 즉시 역추세 진입 시나리오 이원화(아래 참조) |
| 지금 판단은 분석인가 감정인가 | △ | "14배 오른 종목이 반토막났으니 지금이 저가"라는 생각 자체가 놓친 기회에 대한 아쉬움(FOMO)일 수 있음 — 경계 필요 |

## 밸류에이션 판단 — 트레이딩 계획 (계산 과정 명시)

저는 내재가치를 계산하지 않습니다. 대신 "어디서 사고, 어디서 손절하고, 얼마를 걸 것인가"를 계산합니다.

### 현재 가격 구조
| 항목 | 값 |
|---|---|
| 현재가 | 1,177,000원 |
| 52주(사상) 최고가 | 2,189,000원 (-46.2%) |
| 랠리 시작가(25.7월) | 153,000원 |
| 애널리스트 목표가(저/평균/고) | 1,300,000 / 1,906,000 / 2,700,000원 |
| 최근 단일일 급락 사례 | -14.98%, -8%대, -7.27% (다수) |
| 베타 | 2.38 (고변동) |

### 시나리오 A — 추세 확인 후 진입 (제가 권하는 방식, "기다림도 포지션이다")
- **진입 트리거**: 1,300,000원 이상 종가 마감 + 거래량 동반 (애널리스트 최저 목표가와 일치하는 저항선 돌파 = "시장 자체가 그은 바닥선"을 상향 돌파하는 확인 신호)
- **손절**: 진입가 × (1 - 8%) = 1,300,000 × 0.92 = **1,196,000원** (오닐의 -7~8% 룰)
  - 리스크 폭 = 104,000원/주 (진입가 대비 -8%)
- **목표 1**: 애널리스트 평균 목표가 1,906,000원 → 보상 = 1,906,000 - 1,300,000 = 606,000원
  - **손익비 = 606,000 / 104,000 ≈ 5.83 : 1**
- **목표 2(스트레치)**: 전고점 재도전 2,189,000원 → 보상 = 889,000원
  - **손익비 = 889,000 / 104,000 ≈ 8.55 : 1**
- **사이징**: 하이트 원칙(단일 거래 리스크 계좌의 1%) 적용
  - 포지션 크기 = 리스크 허용치 / 손절 폭% = 1% / 8% = **계좌의 12.5%**
  - (공격적으로 2% 리스크 허용 시 25%까지 가능하나, 베타 2.38의 고변동성을 감안해 1% 기준 12.5%를 상한으로 권고)

### 시나리오 B — 즉시 역추세 진입 (권하지 않음, 참고용)
NAV 할인율 44~46%(목표 30%)와 Forward PER 3~4x라는 밸류에이션에 끌려 "지금 당장" 진입하고 싶다면:
- **진입**: 현재가 1,177,000원
- **손절**: 1,177,000 × 0.92 = **1,083,000원** (-8%, 최근 저점 데이터 부재로 오닐 룰 기계적 적용)
  - 리스크 폭 = 94,000원/주
- **목표**: 애널리스트 평균 1,906,000원 → 보상 = 729,000원
  - **손익비 = 729,000 / 94,000 ≈ 7.76 : 1**
- **사이징**: 추세 미확인 상태이므로 시나리오 A의 절반만 배분 — 1% 리스크 기준 12.5%가 아니라 **계좌의 6% 이내**로 축소 (마커스: "한 번의 거래에 전 재산을 걸지 말 것")

손익비만 보면 두 시나리오 모두 5:1을 훌쩍 넘습니다. 리처드 데니스라면 "승률이 낮아도 이 정도 손익비면 할 만하다"고 했을 겁니다. 문제는 손익비가 아니라 **타이밍**입니다 — 시나리오 A는 추세가 확인된 뒤 들어가므로 승률이 높아지고, 시나리오 B는 손익비는 더 좋아 보이지만 하락 추세 한가운데서 진입하는 것이므로 손절에 걸릴 확률(승률)이 낮습니다.

## 리스크
- **추세 지속 리스크**: 6월 이후 낙폭(-46%)이 단기 조정이 아니라 밸류에이션 정상화의 시작이라면, 1,300,000원 돌파 자체가 오지 않거나 훨씬 더 낮은 가격에서 형성될 수 있음.
- **SK하이닉스 의존 리스크(베타 2.38)**: 이 종목은 사실상 SK하이닉스 레버리지 상품입니다. 메모리 사이클이 반전되면 손절 폭 8%가 하루 만에 뚫릴 수 있습니다(실제로 -14.98% 단일일 급락 전례 있음) — 갭 손실 가능성을 반드시 감안해야 합니다.
- **복수매매 함정**: 전고점 2,189,000원을 본 투자자가 "그때 못 산 것"에 대한 아쉬움으로 지금 추격 매수하는 것은 전형적인 감정적 거래입니다. 데이터 팩에도 "SK하이닉스 평균 매수가 181만원 부근에서 손실구간 목전"이라는 진단이 있는데, 이는 많은 투자자가 고점 부근에서 이미 물려 있다는 뜻 — 반등 시 매물 저항(손실 회피 매도)이 있을 수 있습니다.
- **8/13 실적 발표**: SK스퀘어 2분기 실적 발표(추정) 전후 변동성 확대 가능 — 시나리오 A 진입 트리거가 발표 직전/직후에 걸릴 경우 손절 라인 재점검 필요.

## 최종 의견
제가 인터뷰한 트레이더 중 누구도 "14배 오른 종목이 반토막났으니 이제 바닥이다"라는 이유만으로 매수 버튼을 누르지 않았습니다. 폴 튜더 존스라면 이렇게 물었을 겁니다. "이 거래가 틀렸을 때 얼마를 잃는가? 그리고 지금 이 판단은 분석인가, 놓친 기회에 대한 아쉬움인가?" 펀더멘털(SK하이닉스 지분법이익 급증, NAV 할인율 축소 스토리)은 매력적입니다 — 하지만 펀더멘털이 좋다고 추세를 거스를 이유는 되지 않습니다. 저는 1,300,000원 돌파를 확인할 때까지 기다리겠습니다. 손절은 -8%(1,196,000원), 목표는 애널리스트 평균 1,906,000원, 손익비 5.83:1, 계좌의 12.5% 이내. 그 전까지는 관망입니다 — 기다리는 것도 유효한 포지션입니다.
