## 기업 개요
- 사업모델: 대형 저속 디젤엔진(선박용) 제조·판매, 산업설비/발전용 엔진, 풍력설비, 조선기자재. 구 STX중공업(STX Heavy Industries) → 2023년 HD현대그룹 인수 → 2024.7 사명변경.
- 주요 세그먼트: 엔진(선박용 저속 디젤엔진, 마진 24%대), 부품(예비부품·터보차저 부품).
- 경쟁사: HD현대중공업 엔진기계사업부(비상장 사업부, 대형엔진 세계점유율 36%·1위, 상장 지주 HD한국조선해양 009540/HD현대중공업 329180 참고), 한화엔진(082740, 구HSD엔진, 점유율 약13%·2위). 국내 3사가 세계 선박엔진 공급의 50%+ 차지.
