# CMCSA (Comcast Corporation) 데이터 팩

조회일: 2026-07-20 (미국 동부시간 기준 최근 거래일 종가 2026-07-17)

## 1. 기업 개요
- 사업 모델: 미국 최대 케이블/브로드밴드·모바일 통신사업자 겸 미디어 복합기업. Connectivity & Platforms(브로드밴드·무선·비즈니스서비스) + Content & Experiences(NBCUniversal 스튜디오·방송·Peacock 스트리밍·유니버설 테마파크·Sky) 2대 사업부.
- **2026-06-29 발표**: NBCUniversal+Sky를 별도 상장사로 분할(내년 완료 예정), Comcast 존속법인은 브로드밴드·무선·비즈니스커넥티비티만 보유하는 순수 통신사로 재편. (2026년 1월 Versant Media 스핀오프는 이미 완료.)
- 경쟁사: Charter Communications(CHTR, 케이블), AT&T(T)·Verizon(VZ, 광케이블/파이버·5G FWA), Disney(DIS)·Warner Bros Discovery(WBD, 미디어).
