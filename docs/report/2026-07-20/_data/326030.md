# 326030 SK바이오팜 데이터 팩 (조회일 2026-07-20)

## 기업 개요
- 사업모델: 국내 유일 신약 자체개발·글로벌 직접판매 바이오텍. 뇌전증 치료제 **세노바메이트**(美 제품명 XCOPRI/엑스코프리)가 매출·이익의 핵심(美 시장 직접판매).
- 신규 성장동력: TPD(표적단백질분해), RPT(방사성의약품) 등 차세대 모달리티 확장 — SKL35501/SKL35502(NTSR-1 표적 고형암) 2026-01 한미 FDA 임상1상 IND 승인. 악티늄-225 공급망(테라파워·판테라·에커트앤지글러) 확보.
- 경쟁사: 국내 전통제약(유한양행 000100, 한미약품 128940, 종근당 185750), 바이오시밀러 셀트리온(068270). 글로벌 CNS 신약사(Jazz Pharmaceuticals, UCB)는 직접 비교사이나 Yahoo 무료 데이터로 확인 불가.
