## 기업 개요
- 사업 모델: GPU/AI 가속기 설계·판매 팹리스 반도체사. 데이터센터(AI 학습·추론용 GPU, 네트워킹), 게이밍(GeForce), 프로페셔널 시각화, 자동차/로보틱스 4개 세그먼트. 매출 대부분(약 88%+)이 데이터센터.
- 주요 제품: Blackwell/GB200·GB300 시스템, H20(중국向), NVLink/InfiniBand 네트워킹, CUDA 소프트웨어 생태계.
- 경쟁사: AMD(MI300/MI400 GPU), Broadcom(커스텀 ASIC), Intel(Gaudi), 하이퍼스케일러 자체칩(Google TPU, AWS Trainium, Microsoft Maia), TSMC(파운드리, 공급망 상단).
