# 443060 HD현대마린솔루션 데이터 팩 (조회일 2026-07-20)

## 기업 개요
- 사업모델: 선박·발전용 엔진 애프터마켓(AM) 전문기업. 자체 개발 엔진(HIMSEN) 기반 부품·정비 독점 공급 + 친환경 개조(스크러버/이중연료 전환), 벙커링(선박연료공급), 디지털 솔루션(축발전시스템 등). 2024-05 코스피 상장(신규 상장 종목).
- 최근 성장동력: 데이터센터 전력 수요발 발전용 엔진(AIDC) 신사업, 부유식 해상 데이터센터(FDC), 싱가포르 물류센터 가동(해외 첫 거점).
- 경쟁사(피어그룹, IPO 밸류에이션 기준): HEICO(美 항공 AM), Wärtsilä(핀란드), 한화엔진(082740), STX엔진(077970). AS 매출 비중 기준으로 직접비교 논란 존재(Wärtsilä·Alfa Laval는 AS 비중 26~33%로 순수 AM기업인 HD마린솔루션과 사업구조 상이).
