# 030190 NICE평가정보(NICE Information Service) 데이터 팩 (조회일 2026-07-20)

## 기업 개요
- 사업모델: 1985년 설립, 신용정보법 근거 신용정보업 영위. 개인신용평가(CB)·기업정보·자산관리·빅데이터 4대 사업부. 2021년 마이데이터(신용정보관리업) 허가 취득. 2023년 코스닥→코스피 이전상장.
- 최근 확장: 2026년 마이데이터 서비스 지원사업 최종 선정, 차량정비이력 기반 서비스 "카핀" 출시, 베트남 민간 신용정보서비스 허가 획득(해외진출).
- 경쟁사: 코리아크레딧뷰로(KCB, 비상장·개인CB 양대산맥), 나이스디앤비(130580, 기업CB), 이크레더블(030520, 기업정보), 서울평가정보(구 SCI평가정보, 036120, 기업CB 소형).
- 개인신용평가 시장은 NICE평가정보-KCB 양강 구도.
