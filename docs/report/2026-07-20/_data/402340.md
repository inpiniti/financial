# SK스퀘어 (402340.KS) 공용 데이터 팩

조회일: 2026-07-20 | 통화: KRW | 거래소: KRX(코스피) | 섹터: Technology / Semiconductors(Yahoo 분류, 실질은 투자지주회사)

## 1. 기업 개요
- 사업모델: SK텔레콤에서 2021년 인적분할된 투자전문지주회사. 핵심 자산은 SK하이닉스 지분 20.5%(최대주주)로, 연결손익 대부분이 SK하이닉스 지분법이익에서 발생. 포트폴리오 자회사: 11번가, SK쉴더스(보안), 콘텐츠웨이브, T1(e스포츠), FSK L&S 등.
- 밸류에이션 프레임: SK하이닉스 지분가치 기반 NAV 대비 할인율로 시장에서 거래(현재 할인율 약 44%, 회사 목표 2028년까지 30% 이하로 축소).
- 경쟁/비교군: 국내 투자지주회사(SK Inc, 두산, CJ, 한화 등) — 사업 포트폴리오보다 지분법 자산가치(SK하이닉스)가 주가를 지배하는 구조라 전통적 지주사 비교는 참고용.
- 지배구조: SK Inc.가 최대주주. 내부자 지분 32.2%, 기관 38.6%.
