# TCOM 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-20 | 투자의견: 매수 | 확신도: 8/10

## 한 줄 결론
껌 가게 비유로 말하자면, 이 아이는 자기 자본을 매년 30~40원씩 벌어들이면서(ROC), 가게 문에 걸린 가격표는 이익의 8~13배 수준(EY 9~13%대)에 불과합니다 — 좋은 기업을 싼값에 사는 마법 공식의 정의 그 자체에 들어맞는 종목입니다.

## 내 철학에서 본 이 기업
저는 TCOM이 "중국 OTA"인지 "AI 여행 플랫폼"인지, "반독점 조사가 무섭다"는 뉴스가 진짜인지에는 관심이 없습니다. 제가 묻는 질문은 딱 둘입니다. 이 사업은 투입한 자본 대비 돈을 잘 버는가(ROC), 그리고 지금 가격에 사면 그 이익을 얼마의 수익률로 사는 셈인가(EY). 아래에서 숫자로 답하겠습니다.

## 밸류에이션 판단 (마법 공식 계산)

### 1) ROC (자본수익률) = EBIT ÷ (순운전자본 + 순고정자산)

| 항목 | 값 (FY2025, CNY 백만) |
|---|---|
| EBIT (영업이익) | 17,114 |
| 유동자산 | 121,050 |
| 유동부채 | 78,169 |
| **순운전자본(NWC)** | **42,881** (=121,050−78,169) |
| 순고정자산(Net Fixed Assets) | **데이터팩 미기재, 확인 불가** |

데이터팩에는 PP&E(고정자산) 항목이 없습니다. 저는 숫자를 지어내지 않습니다. 대신 두 가지 시나리오로 범위를 잡습니다.

- **상한 시나리오** (순고정자산 ≈ 0, 즉 완전 asset-light 가정 — 온라인여행 플랫폼은 물리 자산이 적어 현실적으로 가능한 근사): ROC = 17,114 ÷ 42,881 = **39.9%**
- **하한 시나리오** (비유동자산 81,513 = 총자산 202,563 − 유동자산 121,050 중 20%를 순고정자산으로 보수적으로 배분): 투입자본 = 42,881 + 16,303 = 59,184 → ROC = 17,114 ÷ 59,184 = **28.9%**

**결론: ROC ≈ 29~40%.** 어느 가정을 쓰든 제 마법 공식 유니버스 상위 데실(decile)에 들어가는 수준입니다. 매출총이익률 80.6%, 영업이익률 27.4%(FY25) — 적은 자본으로 많이 버는 전형적인 "좋은 기업"입니다.

### 2) EY (이익수익률) = EBIT ÷ EV — **EV 마이너스 문제 처리**

데이터팩(섹션 4)은 EV를 **−$20.5B**로 보고합니다. 시가총액($26.73B)보다 순현금·투자자산이 더 크다는 뜻입니다. 여기서 제 공식이 문자 그대로 깨집니다.

```
EBIT(USD) = 17,114 CNY백만 ÷ 6.78 = $2,525백만
문자 그대로: EY = EBIT / EV = 2,525 / (−20,500) = −12.3%  ← 무의미한 숫자
```

분모가 음수면 분수는 계산할 수 있어도 경제적으로 말이 안 됩니다. EV가 음수라는 건 "이 회사를 통째로 사면 오히려 돈을 돌려받는다"는 뜻인데, 실제로 여러분이 주식을 사려면 여전히 시가총액만큼의 돈을 내야 합니다. 마법 공식은 "인수자가 실제로 치르는 가격"을 분모로 삼는 것이 설계 취지이므로, 저는 이런 순현금 과다 상황에서 EV를 **0이나 음수로 방치하지 않고, 인수자가 실제 지불하는 시가총액으로 바닥 처리(floor)**합니다.

```
EY(floor, 시가총액 기준) = EBIT / 시가총액 = 2,525 / 26,730 = 9.45%
```

교차검증으로, 데이터팩 재무제표에 직접 나온 총부채·총현금성자산만으로 순부채를 재계산하면(단기투자자산 등 광의 현금 제외):

```
순부채(재계산) = 총부채 31,614 − 총현금성자산 39,848 = −8,234 (순현금 CNY 8,234백만)
EV(재계산, CNY) = 시가총액(CNY 181,270) − 순현금(8,234) = 172,036 → EV(재계산, USD) = $25.37B
EY(재계산) = 2,525 / 25,370 = 9.95%
```

이 재계산치와 데이터팩 섹션4의 −$20.5B EV(아마 단기투자상품·재테크 자산까지 포함한 광의 현금 기준)는 서로 크게 어긋납니다 — **데이터 정합성 문제**로 명시해 둡니다. 어느 쪽 현금 정의를 쓰든 결론은 같습니다: **문자 그대로의 EV로 계산하면 EY는 정의상 왜곡되거나 음수가 나오고, 실무적으로 바닥 처리하면 EY는 9~13% 사이입니다.**

**EV 마이너스를 어떻게 처리했는가 — 요약**
1. 문자 그대로 계산하지 않는다 (분모 음수는 공식의 설계 전제 위반).
2. 실제 인수자가 지불하는 가격(시가총액)을 EV의 하한으로 사용해 보수적으로 EY를 산출한다 → 9.45%.
3. 재무제표에 명시된 좁은 의미의 순현금으로 교차검증한다 → 9.95%.
4. 두 방법 모두 EY가 견조한 두 자릿수에 근접 — "순현금 과다 = 계산 오류"가 아니라 "이 회사가 극단적으로 저평가되어 있거나, 시장이 순현금성 자산의 질(중국 내 규제·자본통제 리스크)을 할인하고 있다"는 신호로 해석한다.

### 3) 순위 결합 (개념적)
- ROC ≈ 29~40% → 전체 유니버스에서 상위권 (동종업계 Booking 25.0%, Expedia 7.1%, Tongcheng 19.5% 대비 우월)
- EY ≈ 9.5~13% (보수적 바닥 처리 기준) → 무위험금리(미국 장기채 대비)를 상회하는 매력적 수준, 동종업계 대비 PER 6.37배로 최저
- 두 지표 모두 상위권에 위치 → 그린블라트 공식이 찾는 "좋으면서 동시에 싼" 종목의 교과서적 사례

## 리스크
- **제 공식이 못 잡는 리스크**: SAMR 반독점 조사(2026-01 개시, 벌금·사업구조 변경 가능성 명시), 집단소송 다수 접수. 이건 ROC나 EY 어디에도 안 나옵니다. 정성적으로만 판단 가능한 영역이고, 저는 이걸 "직관으로 종목을 빼는 함정"으로 보되 무시하진 않습니다 — 포트폴리오 20~30개 분산이 바로 이런 개별 리스크를 상쇄하기 위한 장치입니다.
- 2Q2026 가이던스 급감(매출 성장률 17%→3~8%)과 최근 90일 EPS 컨센서스 전방위 하향(FY2026 −10%)은 EBIT의 지속가능성에 대한 경고 신호입니다. 제 ROC·EY는 과거 실적(FY2025) 기반이므로, 미래 EBIT이 꺾이면 EY의 분자가 줄어 밸류에이션 매력이 후퇴합니다.
- 순고정자산 미기재로 ROC는 추정 범위(29~40%)이지 정밀치가 아닙니다. 20-F 원문 대차대조표 확인이 필요합니다.
- EV 계산에 쓰인 현금성자산 정의가 데이터 소스 간 불일치(순부채 −8.2B vs −49.6B) — 실제 EV/EY는 제가 제시한 9~13% 범위 안 어딘가일 가능성이 높지만 정밀치는 아닙니다.
- 중국 ADR/VIE 구조 특유의 리스크(자본통제, 상장폐지 리스크, 회계 접근성)는 제 두 지표에 담기지 않는 구조적 리스크입니다.

## 최종 의견
숫자를 다시 봅시다. ROC는 최소 29%, 아마 40%에 가깝습니다. 껌 가게로 치면 100원 투자해서 매년 30~40원을 버는 셈이니 대단한 사업입니다. EY는 문자 그대로 계산하면 순현금이 너무 많아서 공식이 헛돌지만, 실제로 여러분이 지불하는 가격(시가총액) 기준으로 바닥을 잡으면 9~13% — 나쁘지 않은, 오히려 매력적인 가격입니다. 좋은 기업 + 싼 가격, 이게 제 공식의 전부입니다.

SAMR 조사 뉴스가 무섭게 들리고, 2분기 가이던스가 꺾였다는 것도 압니다. 하지만 바로 그 공포 때문에 가격이 52주 저점권까지 밀린 겁니다. 저라면 이 뉴스를 이유로 공식에서 종목을 빼지 않습니다. 개별 종목 하나가 규제 리스크로 크게 흔들릴 수 있다는 걸 전제로, 저는 20~30개 종목으로 분산합니다. TCOM은 그 포트폴리오에 편입할 자격이 있는 종목입니다 — 다만 전 재산을 걸 종목은 아닙니다. 그게 제 공식이 하는 일이고, 제가 할 일은 그 공식을 흔들림 없이 따르는 것뿐입니다.
