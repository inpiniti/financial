# EDU 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-20 | 투자의견: 매수 | 확신도: 7/10

## 한 줄 결론
껌 가게 소년 제이슨의 눈으로 봐도 답은 나옵니다 — 뉴 오리엔탈은 시가총액의 절반 가까이가 현금이라 실제로 사업을 사는 값(기업가치, EV)은 시가총액의 반값밖에 안 되고, 그 값에 이익수익률(EY) 약 12~13%를 벌어들이는 자본수익률(ROC) 14%대의 준수한 사업입니다. "좋고 싸다"는 두 조건을 웬만큼 충족합니다.

## 내 철학에서 본 이 기업
저는 종목을 "스토리"로 보지 않습니다. 두 개의 숫자 — ROC(좋은 기업인가)와 EY(싼 가격인가) — 로 봅니다. EDU는 학원 사업이라 겉으론 규제 리스크투성이 뉴스로 도배돼 있지만, 저는 뉴스를 안 봅니다. 숫자를 봅니다. 그리고 그 숫자가 말해주는 건 이렇습니다: 학생들이 수업료를 선불로 내는 사업 구조 덕에(이연수익 약 $1.89B), 이 회사는 순운전자본을 거의 필요로 하지 않고, 오히려 현금이 산더미처럼 쌓입니다. 이게 바로 제가 좋아하는 "적은 자본으로 돈을 버는" 사업의 전형입니다.

## 체크리스트 평가

| # | 항목 | 평가 | 근거 수치 |
|---|---|---|---|
| 1 | 자본수익률(ROC) 높은가 | △ | 표준 계산 시 ROC ≈ 14%대(아래 계산 참조). 마법공식 상위권 기준(대개 25%↑)엔 못 미치나, 현금 제외 시 필요 영업자본이 사실상 마이너스(음수) — "자본이 거의 필요 없는 사업"이라는 더 강한 신호 |
| 2 | 이익수익률(EY) 높은가 | ○ | EY ≈ 11.6%~13.5% (아래 계산). 시장 평균 PER 17~20배(EY 5~6%)의 두 배 이상 |
| 3 | 규모 필터 통과 | ○ | 시가총액 $8.19B — 소형주 함정 아님, 유동성·기관 지분율(35.3%) 충분 |
| 4 | 비이성적 비관으로 싸진 것인가 | ○ | 애널리스트 목표주가 평균 $70.61(현재가 대비 +43.6%)인데도 주가는 정체 — "쌍감정책" 트라우마와 규제 뉴스가 계속 가격을 눌러왔다는 정황 |
| 5 | 직관(스토리)이 아니라 숫자로 설명되는가 | ○ | ROC·EY 모두 계산 가능한 하드 넘버로 뒷받침됨 |

## 밸류에이션 판단 (계산 과정 명시)

### 1. 자본수익률(ROC) = EBIT ÷ (순운전자본 + 순고정자산)

**기초 데이터 (2026-02-28 TTM 기준, stockanalysis.com/EDGAR 20-F 교차확인)**
| 항목 | 값 |
|---|---|
| 유동자산 | $5,427.1M |
| 유동부채 | $3,278.6M |
| 순운전자본(NWC, 조정 전) | $5,427.1M − $3,278.6M = **$2,148.5M** |
| 순고정자산(Net PP&E) | $1,693M |
| 총현금 | $4,802.4M |

**(A) 표준(비조정) 투하자본**
```
투하자본 = NWC + 순고정자산 = $2,148.5M + $1,693M = $3,841.5M
```

**(B) 현금 제외 조정(영업용 자본만)** — 그린블라트 원칙: EBIT을 버는 데 실제 필요한 자본만 분모에 넣는다. 학원 수업료 선불(이연수익 $1.89B 포함)이 유동부채에 큰 비중을 차지하므로, 현금을 뺀 "영업 NWC"는:
```
영업 NWC = (유동자산 − 현금) − 유동부채
         = ($5,427.1M − $4,802.4M) − $3,278.6M
         = $624.7M − $3,278.6M = −$2,653.9M (마이너스!)

조정 투하자본 = 영업 NWC + 순고정자산 = −$2,653.9M + $1,693M = −$960.9M
```
→ **영업용 투하자본이 음수.** 학생들이 수업 듣기 전에 돈을 먼저 내는 구조(선수금·이연수익)가 순고정자산보다 크다는 뜻입니다. 껌 가게로 치면, 손님이 껌을 받기도 전에 돈부터 내서 그 돈으로 다음 달 껌을 사입하는 셈 — 사업 확장에 남의 돈(고객 선수금)을 쓰는 구조입니다.

**EBIT (세전영업이익)**
| 기준 | EBIT | 산출 근거 |
|---|---|---|
| FY2025(연간, 확정) | $488.5M (매출의 10.0%) | 데이터 팩 재무제표 |
| TTM(추정) | 약 $552M | FY2025 EBIT/순이익 배수(488.5/371.7≈1.314)를 TTM 순이익($420.1M)에 적용해 역산 — 추정치 |

**ROC 계산 결과**
```
ROC(표준, TTM 추정) = $552M ÷ $3,841.5M ≈ 14.4%
ROC(표준, FY2025)   = $488.5M ÷ $3,514M(=NWC $1,898M+PP&E $1,616M) ≈ 13.9%
ROC(현금조정, 영업자본) = $552M ÷ (−$960.9M) → 계산 불능(분모 음수) = "무한대에 가까운 자본효율"로 해석
```
표준 계산으로는 ROC 14%대 — 마법공식 최상위권(통상 25%↑) 기준엔 못 미칩니다. 하지만 현금을 빼고 보면 영업에 실제 묶인 자본이 마이너스라는 사실이 더 중요합니다. **자본이 필요 없다 못해 남의 돈(선수금)으로 사업을 돌리는 구조**는 ROC 숫자 하나로는 다 담기지 않는 좋은 신호입니다.

### 2. 이익수익률(EY) = EBIT ÷ EV(기업가치)

```
EV = 시가총액 + 총부채 − 총현금
   = $8,190M + $807.0M − $4,802.4M
   = $4,194.6M (≈ $4.19B)
```
(데이터 팩 표기 EV $4.09B와 대체로 정합 — 산출 시점·환산 오차 범위)

```
EY(TTM 추정) = EBIT $552M ÷ EV $4,194.6M ≈ 13.2%
EY(FY2025)   = EBIT $488.5M ÷ EV $4,194.6M ≈ 11.6%
```

**순현금 $4B가 EV에 미치는 영향 — 이게 핵심입니다.**
| 지표 | 값 |
|---|---|
| 시가총액 | $8,190M |
| 순현금 | $3,995.4M (총부채 $807.0M − 총현금 $4,802.4M) |
| 순현금 / 시가총액 비중 | **48.8%** |
| EV / 시가총액 비율 | $4,194.6M / $8,190M ≈ **51.2%** |

→ 시장가격표(시가총액)만 보면 $8.19B짜리 회사지만, 사업 그 자체를 통째로 인수할 때 실제로 치러야 하는 값(EV)은 **시가총액의 절반**밖에 안 됩니다. 나머지 절반은 이미 회사 금고에 들어 있는 현금이기 때문입니다.

이게 왜 중요한가 — 단순 PER로 보면 함정에 빠집니다.
```
단순 이익수익률(순이익/시총, 참고용) = TTM EPS $2.70 ÷ 주가 $49.15 ≈ 5.5%
EV 기반 이익수익률(EY) = 13.2%
```
같은 회사인데 분모를 시가총액으로 두느냐 EV로 두느냐에 따라 이익수익률이 **5.5% vs 13.2%로 2배 이상 차이**가 납니다. 인수자 입장에서 실제 지불하는 값(EV)을 기준으로 보면 이 회사는 훨씬 싸 보입니다. PER만 보고 "18배라 비싸다"고 넘기는 사람은 이 함정에 빠지는 겁니다.

### 3. 마법공식 결론
| 지표 | 값 | 마법공식 상 해석 |
|---|---|---|
| ROC(표준) | 약 14% | 중간 — 톱10% 우량주(25%↑) 아니지만 준수 |
| ROC(현금조정 영업자본) | 음수(=자본 거의 불요) | 매우 강한 신호(정성적 가점) |
| EY | 약 11.6~13.5% | 강함 — 시장 평균 EY(5~6%)의 2배 이상, 마법공식 상위 분위 수준 |

두 지표를 합산 순위로 매기면(가상 유니버스 기준), EY가 상위권을 확실히 끌어올리고 ROC는 중상위권입니다. "위대한 기업이 헐값"은 아니지만, "괜찮은 기업이 확실히 싸다"에는 부합합니다. 이 조합이 제가 찾는 마법공식 최적 지점입니다.

## 리스크
- **분모 문제**: EBIT을 TTM 순이익에서 역산한 추정치입니다. 정확한 개별 이자비용·EBIT raw 데이터가 없어 오차가 있을 수 있습니다(데이터 팩 "확인 불가" 항목).
- **회계 이익 ≠ 현금흐름**: FCF(FY2025 $637.4M)가 순이익($371.7M)보다 크긴 하지만, 규제 이후 사업 재편으로 실적 변동성이 큽니다.
- **제가 인정하는 공식의 한계**: 정성적 리스크(중국 정부의 VIE·사교육 규제, ADR 구조)는 ROC·EY 두 숫자에 전혀 담기지 않습니다. 2021년 "쌍감정책"으로 매출 90% 급락 전례가 있고, 2026년 3월에도 베이징 당국의 통제 강화 방침이 재발표됐습니다(가치 드라이버 #3). 이건 공식이 못 보는 부분이라 별도로 경계해야 합니다.
- 다만 제 원칙대로면: 바로 이 비관적 뉴스 때문에 가격이 싸진 것이고, 공식은 그 점을 이미 포착하고 있습니다. 뉴스가 무섭다고 임의로 빼면 안 됩니다.
- 분산 없이 이 종목 하나에 집중하는 것은 제 방법론과 맞지 않습니다. 20~30개 종목 포트폴리오의 일부로만 접근해야 합니다.

## 최종 의견
숫자로 말하겠습니다. ROC는 표준 계산으로 14%대, 현금 빼고 보면 영업자본이 마이너스라 자본 자체가 거의 필요 없는 사업입니다. EY는 EV 기준 약 12~13%로 시장 평균의 두 배가 넘습니다. 시가총액의 절반이 현금이라는 사실 하나만으로도 이 회사가 겉보기보다 훨씬 싸다는 게 드러납니다. 규제 뉴스가 무섭다는 건 압니다. 그런데 바로 그 공포 때문에 가격이 이렇게 낮아진 겁니다. 저는 이걸 20~30개 종목 포트폴리오의 하나로 담고, 1년 뒤 새 순위표로 재평가하겠습니다. 스토리는 필요 없습니다. ROC 14%, EY 13% — 이게 전부입니다.
