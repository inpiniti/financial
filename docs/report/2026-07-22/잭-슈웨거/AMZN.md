## 내 철학에서 본 이 기업
제가 인터뷰한 위저드들은 하나같이 이렇게 말했습니다. "이 거래가 틀렸을 때 어디서 빠져나올 것인가?"에 답하지 못하면 진입하지 않는다고. ## 밸류에이션 판단 (트레이딩 플랜 — 계산 과정 명시)

슈웨거식 접근에서는 "적정가"를 계산하지 않는다. 대신 진입가·손절가·목표가·사이징·손익비를 계산한다.

### 1. 추세 컨텍스트
- 현재가 $247.55 / 50일선 $251.16 / 200일선 $234.35 / 52주 범위 $196.00~$278.56
- 중기(200일) 상승추세 내부, 단기(50일) 눌림 — "추세가 친구다"는 원칙상 완전 배제 대상은 아니나 신고가 돌파형 진입 조건(오닐)은 아직 미충족

### 2. 진입가 시나리오
| 플랜 | 조건 | 진입가 |
|---|---|---|
| A. 실적 전 선진입(비권장) | 지금 시점 | $247.55 |
| B. 실적 후 확인 진입(권장) | 7/30 발표 후 AWS 성장률 28% 이상 유지 + 갭업 후 첫 눌림목 | 발표 후 결정, 참고 진입 레인지 $247~$260 |

이하 계산은 플랜 A(현재가 기준)로 진행하되, 본 보고서의 결론은 플랜 B(실적 확인 후 진입)를 권고한다.

### 3. 손절가 계산 (오닐 7~8% 규칙)
$247.55 × (1 - 0.075) = **$228.98 ≈ $229.00**

200일 이평($234.35)이 구조적 지지선이나, 그보다 타이트한 오닐식 퍼센트 손절을 우선 적용 — 손절폭을 좁혀 손익비를 개선하기 위함.

### 4. 목표가 및 손익비
| 구간 | 가격 | 진입가 대비 변화 | 리스크/리워드 |
|---|---|---|---|
| 진입가 | $247.55 | - | - |
| 손절가 | $229.00 | -7.5% (-$18.55) | 리스크 = $18.55/주 |
| 1차 목표(52주 전고점 재돌파) | $278.56 | +12.5% (+$31.01) | 손익비 = 31.01/18.55 = **1.67:1** |
| 2차 목표(애널리스트 중앙값) | $315.00 | +27.2% (+$67.45) | 손익비 = 67.45/18.55 = **3.64:1** |

- 1차 목표만으로는 데니스가 요구하는 최소 손익비(2:1 이상)에 미달한다.
- 2차 목표까지 홀드해야 손익비가 유리해진다 — 즉 "틀렸을 때 빨리, 맞았을 때 오래"라는 추세추종 원칙이 그대로 요구되는 자리다. 1차 목표 도달 시 절반 익절 후 나머지는 트레일링 스탑(예: 20일선 이탈 시 청산)으로 전환하는 방식을 권한다.

### 5. 포지션 사이징 (하이트의 1% 룰)
가정: 계좌 자산 $100,000, 단일 거래 리스크 한도 1% = $1,000

```
포지션 주식수 = 리스크 허용액 / 주당 손절폭
            = $1,000 / $18.55
            = 53.9주 ≈ 53주

명목 포지션 금액 = 53주 × $247.55 = $13,120 (계좌의 13.1%)
```

공격적 상한(2% 룰) 적용 시: $2,000 / $18.55 = 107주, 명목 $26,488(26.5%) — 단일 종목 집중도가 지나치게 높아 권장하지 않는다. 1% 룰 기준 53주가 규율에 부합한다.

## 최종 의견
