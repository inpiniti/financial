## 기업 개요
- 사업 모델: GPU/가속컴퓨팅 팹리스 반도체 설계사. Data Center(AI 가속기·네트워킹), Gaming(GeForce), Professional Visualization, Automotive 4개 세그먼트. FY26(2026-01-31 종료) 기준 Data Center가 매출 대부분(약 88~90%) 차지.
