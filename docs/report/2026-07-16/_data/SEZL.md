# SEZL (Sezzle Inc.) 공용 데이터 팩

조회일: 2026-07-16 (주가 스냅샷 기준 2026-07-15 종가 기준, Yahoo Finance)

## 1. 기업 개요

- **사업 모델**: 미국·캐나다 중심 BNPL(선구매후결제) 핀테크. 대표 상품은 무이자 "Pay in 4"(선결제 25%+격주 3회 분납), 장기 할부(최대 $15,000), 가상카드(온라인·오프라인 겸용), 구독형 멤버십("Sezzle Anywhere"), 최근 SezzleCash(예금성 상품)·은행 라이선스 신청·Pagaya 파트너십으로 신용/자금조달 확장 중.
- **주요 세그먼트/지표**: 가맹점 4만1,800개+(Target, GameStop 등), Active Subscriber +48.4% YoY(Q1'26), GMV $1.1B(+37.3% YoY, Q1'26), 구매빈도 7.1x/분기(사상 최고).
- **경쟁사**: Affirm(AFRM), Klarna(비상장/PayPal 제휴), Block의 Afterpay(SQ), PayPal Pay in 4. Sezzle은 무이자 단기 분할 특화, 소형·중신용 소비자층 겨냥.
