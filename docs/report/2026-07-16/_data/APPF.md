# APPF (AppFolio, Inc.) 공용 데이터 팩

조회일: 2026-07-16 | 통화: USD | 거래소: NASDAQ(NGM)

## 1. 기업 개요
- **사업 모델**: 부동산(임대·주거·상업) 관리사를 위한 클라우드 SaaS 플랫폼. 회계·리포팅·마케팅·임대·유지보수·워크플로 자동화·커뮤니케이션 기능 제공.
- **주요 제품**: AppFolio Property Manager (Core/Plus/Max), Realm(AI 어시스턴트 스위트, 고객 99%+ 사용), Value+ 부가서비스(결제처리·보험·유틸리티 정산 등).
- **세그먼트**: 매출 대부분이 구독료(Core) + Value+ 부가서비스(거래 기반, 성장 견인축).
- **경쟁사**: RealPage(Thoma Bravo 비상장), Yardi Systems(비상장), Entrata(비상장), MRI Software.
- **업종**: Technology / Software - Application, 직원 1,702명.
