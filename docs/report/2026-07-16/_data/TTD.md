- **사업 모델**: 독립계(퍼블리셔 미소유) 프로그래매틱 광고 구매 플랫폼(DSP, Demand-Side Platform). 광고주·에이전시가 디스플레이·비디오(CTV 중심)·오디오·모바일 광고를 실시간 입찰로 구매하도록 지원, 매체 지출액 대비 플랫폼 수수료 수익.
