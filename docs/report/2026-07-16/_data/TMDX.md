# TMDX (TransMedics Group, Inc.) 공용 데이터 팩

조회일: 2026-07-16 (미국 동부시간 기준 최신 거래일 2026-07-15/16 시세)

## 1. 기업 개요
- **사업 모델**: 장기이식(organ transplant) 치료 혁신 의료기기 기업. 핵심 제품은 **OCS(Organ Care System)** — 이식용 장기를 체외에서 근생리학적 조건으로 관류·보존·모니터링하는 휴대형 시스템.
- **주요 세그먼트**: OCS Lung(폐), OCS Heart(심장), OCS Liver(간) 3개 제품 라인 + National OCS Program(NOP, 장기 회수·물류 아웃소싱 턴키 서비스, 자체 항공/지상 운송 포함).
- **경쟁사**: Paragonix Technologies(콜드체인 보존), XVIVO Perfusion(스웨덴, 폐 관류), OrganOx(간 관류, metra 시스템), 전통적 저온보관(cold storage) 방식. 아직 시장 초기 단계로 경쟁 강도는 제한적이나 XVIVO·OrganOx가 각 장기별로 경쟁.
- 창립 1998년, 본사 매사추세츠주 앤도버. 직원 수 약 898명(2026-07 기준).
