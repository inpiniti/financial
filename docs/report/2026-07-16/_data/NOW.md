# NOW (ServiceNow, Inc.) 공용 데이터 팩

조회일: 2026-07-16 (미국 동부시간 기준 2026-07-15 정규장 마감가 반영) | 거래소: NYSE

## 1. 기업 개요

- **사업 모델**: 클라우드 기반 워크플로 자동화 플랫폼(Now Platform). IT 서비스 관리(ITSM)에서 출발해 HR·고객서비스(CSM)·보안운영(SecOps)·생성형 AI(Now Assist) 등으로 확장. 구독(subscription) 매출이 전체의 약 97% 이상을 차지하는 SaaS 모델.
- **주요 세그먼트**: 구독(Subscription) / 프로페셔널서비스(기타). 지역별로는 북미 매출 비중이 가장 크며 해외(EMEA·APAC) 성장 가속 중.
- **경쟁사**: Salesforce(CRM), Workday(WDAY), SAP, Atlassian, Microsoft(Power Platform), Appian, Pegasystems.
- **주가 이벤트**: 2025년 12월 17일 5:1 액면분할 실시(2025-12-18부터 분할 조정 거래). 본 팩의 모든 주가·EPS·발행주식수는 분할 반영 기준.

## 8. 주가 흐름 및 최근 이벤트

- 52주 고점 $210.20 → 저점 $81.24로 연중 약 -61% 조정 후 현재 저점 근처($104.73)에서 반등한 상태.
- **2026-07-14 급락**: IBM이 2분기 실적 프리어나운스에서 조정EPS $2.93/매출 $17.2B로 컨센서스($3.01/$17.86B) 하회 발표, CEO가 "기업들이 예산을 하드웨어·보안 쪽으로 급격히 재배분 중"이라 언급. 이에 동반해 소프트웨어·컨설팅주 전반(ADBE, CRM, NOW, WDAY 등) 동반 급락. NOW는 당일 약 -5.8%~-7% 하락(보도별 상이). 반면 사이버보안주(CRWD, OKTA, ZS)는 반사이익으로 상승.
  - 출처: [Yahoo Finance - IBM warning sends software stocks reeling](https://finance.yahoo.com/markets/stocks/articles/ibm-warning-sends-software-consulting-170700828.html) (2026-07-14), [Benzinga](https://www.benzinga.com/trading-ideas/movers/26/07/60438730/servicenow-stock-falls-in-sympathy-with-ibm-after-disappointing-preliminary-q2-results) (2026-07-14)
