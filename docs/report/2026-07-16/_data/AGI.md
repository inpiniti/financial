# AGI (Alamos Gold Inc.) 공용 데이터 팩

조회일: 2026-07-16 | 통화: USD | 거래소: NYSE(NYQ), TSX 이중상장 | 데이터 기준: Yahoo Finance(quoteSummary/chart/fundamentals-timeseries), 2026-07-15 시세, WebSearch 뉴스

## 1. 기업 개요

- **사업 모델**: 캐나다 기반 중형 금 생산업체(gold miner). 온타리오주 Young-Davidson·Island Gold 광산과 멕시코 Mulatos 광산 복합체(Island Gold District는 확장 중인 핵심 성장 자산) 운영.
- **주요 자산**: Young-Davidson(온타리오), Island Gold(온타리오, District 확장 프로젝트 진행 중), Mulatos/La Yaqui Grande(멕시코 소노라주).
- **성장 전략**: 2025년 연산 약 55만 온스 → 2030년 약 100만 온스(+83%) 목표. AISC는 2025년 온스당 $1,524 → 2028년 $1,250로 하락 전망(회사 가이던스, 2026-04-29 Q1 실적발표).
- **경쟁사**: Agnico Eagle Mines(AEM), Kinross Gold(KGC), B2Gold(BTG), Eldorado Gold(EGO), New Gold(NGD) 등 캐나다·중남미 중형 금광 업체.
