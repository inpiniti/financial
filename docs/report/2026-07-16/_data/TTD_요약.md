# TTD — 13인 거장 요약 (2026-07-16)

## 워런 버핏
- **의견 | 확신도**: 관망 | 6/10
- **핵심논거**: 매출총이익률 78.6%로 절대 수준은 훌륭하나 82.2%→78.6%로 4년 연속 하락 추세. 2026Q1 Adjusted EBITDA 마진이 34%→30%로 급락, EPS 컨센서스 12.4% 미스. 최고전략책임자 이직, 대형 에이전시와의 미해결 분쟁까지 겹쳐 해자 훼손 여부가 불확실. 부채 거의 없고 ROE 16%대는 긍정적.
- **적정가/매수가격대**: 매출총이익률 하락이 멈추고 EBITDA 마진이 반등한다는 확인이 나온다면 현재가 $19대는 매력적인 구간. 확인 없이는 가격 불문 매수하지 않음.
- **트리거·재검토조건**: 매출총이익률 하락 추세 정지, 이익잉여금 재증가 확인.
