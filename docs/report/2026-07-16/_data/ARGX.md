# ARGX (argenx SE) 공용 데이터 팩
조회일: 2026-07-16 | 통화: USD | 거래소: NASDAQ(NMS), 원거래소 Euronext Brussels(ENXTBR:ARGX)

## 1. 기업 개요
- **사업 모델**: 벨기에/네덜란드 기반 임상단계를 넘어선 상업화 바이오제약사. 자가면역질환 치료제 개발·판매.
- **주요 제품**: **VYVGART**(efgartigimod, FcRn 억제제) — 중증근무력증(gMG), 만성염증성탈수초성다발신경병증(CIDP), 일본 면역혈소판감소증(ITP) 적응증 승인. 17분기 연속 매출 성장.
- **파이프라인**: empasiprubart(보체 억제제) 등 "Vision 2030" 전략 하 면역질환 다수 파이프라인(근염, 쇼그렌증후군, 그레이브스병 3상 등).
- **경쟁사**: UCB(Rystiggo/CIDP·gMG), Alexion/AstraZeneca(Ultomiris), Roche(neuroimmunology), Immunovant(FcRn 경쟁 후보) 등.
