# Comfort Systems USA, Inc. (NYSE: FIX) — 공용 데이터 팩
조회 기준일: 2026-07-16 (미국 동부시각 종가 기준, 데이터 asOfDate는 각 항목에 별도 표기)

## 1. 기업 개요
- **사업모델**: 상업용·산업용 건물의 기계적(HVAC) 및 전기 설비 설치·시공·유지보수 전문업체(Mechanical & Electrical Contractor, MEP). 미국 전역 40개 주에서 자회사 네트워크로 운영.
- **주요 세그먼트**: Mechanical(공조·배관·플랜트), Electrical(전기 시공). 최근 성장 축은 **데이터센터(하이퍼스케일 AI/클라우드)** 및 반도체·헬스케어·교육 시설 시공.
- **경쟁사**: EMCOR Group(EME), MYR Group(MYRG), IES Holdings(IESC), Limbach Holdings(LMB), APi Group(APG) 등 MEP·전기시공 업계.
- **최근 이슈**: 모듈러(사전제작) 시공 능력 확대 중, 2026년 말까지 모듈러 생산면적 400만 sq ft 목표. COO(Craig Sasser)·CSIO(Briston Blair) 등 경영진 교체(2026-07-01 발효). (출처: Simply Wall St News, 2026-07)
