# OPRA (Opera Limited) 공용 데이터 팩

조회일: 2026-07-16 | 통화: USD | 거래소: NasdaqGS | 데이터 출처: Yahoo Finance(chart/fundamentals-timeseries/quoteSummary), WebSearch(뉴스/IR)

## 1. 기업 개요
- **사업 모델**: 웹 브라우저(Opera, Opera GX, Opera Mini 등) 사용자 기반을 광고·검색 파트너십으로 수익화. 광고(디스플레이·네이티브·비디오)가 매출의 절반 이상, 나머지는 Google 등과의 검색 제휴 수수료.
- **주요 제품/세그먼트**: Opera(PC/모바일 범용), Opera GX(게이머 특화, CPU/RAM/네트워크 리미터), Opera Mini(신흥시장 데이터 절약형), MiniPay(아프리카 지역 모바일 지갑), 최근 AI 브라우저 기능(Browser Connector, Neon) 확대 중.
- **경쟁사**: Google Chrome, Microsoft Edge, Apple Safari(대형 플랫폼 종속 리스크), Brave(프라이버시 특화 신흥 경쟁자). Google 검색 파트너십 의존도가 구조적 리스크.
- **최대주주**: 중국계 컨소시엄(Kunlun Tech 등) 배경, ADR 형태로 나스닥 상장.

## 6. 주가 흐름

- 52주 고가 $21.06 / 저가 $11.71 → 현재가 $20.17은 52주 레인지 상단 96% 지점(고점 근접).
- 올타임 고점 $28.58 / 저점 $3.92.
- 50일 이동평균 $18.63, 200일 이동평균 $15.70 → 단기·장기 이평선 모두 상회(상승 추세).
- 최근 1년 저점(11.71, 2025년 하반기 추정)에서 현재가까지 약 +72% 상승.
- **최근 급등 원인**: 2026-04-28 1Q2026 실적 발표에서 매출·Adj.EBITDA 모두 가이던스 상단 초과, 광고 매출 사상 최대치 기록, MiniPay 지갑 참여 증가, AI 브라우저 기능(Browser Connector, Neon) 확대가 ARPU 개선 기대로 이어지며 랠리 지속 (출처: Opera IR 2026-04-28 보도자료, Simply Wall St 2026-05경 기사).
