# CLBT (Cellebrite DI Ltd.) 공용 데이터 팩

조회일: 2026-07-16 (KST) / 가격 기준: 2026-07-15 정규장 종가

## 1. 기업 개요

- **사업**: 이스라엘 소재 디지털 포렌식/디지털 인텔리전스(DI) 기업. 모바일 기기 데이터 추출·분석 솔루션을 법 집행기관·기업에 구독형(주로 SaaS/ARR) 모델로 판매.
- **주요 제품**: UFED(Universal Forensic Extraction Device, 매출 60~70% 추정), Cellebrite Premium(암호 우회, 20~25%), Cellebrite Government Cloud(FedRAMP High 인증, 2026-05-06), 신규 AI 제품 "Genesis"(2026 출시, 500+ 등록 사용자/15개국).
- **주요 지표(2026-03-31 기준)**: ARR $493M(+21% YoY), Q2 ARR 가이던스 $510~513M.
- **경쟁사**: Magnet Forensics(Grayshift GrayKey 통합, 북미 강세), MSAB(XRY, 유럽/GDPR), Oxygen Forensics.
- **거래소**: NASDAQ(NMS), 통화 USD.
