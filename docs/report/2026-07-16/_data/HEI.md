# HEICO Corporation (HEI) 공용 데이터 팩

조회일: 2026-07-16 (미국 동부 기준 2026-07-15 종가 데이터)
회계연도(FY) 종료월: 10월 말

## 1. 기업 개요

- **사업 모델**: 항공우주·방위산업용 부품 제조·수리. FAA 인증 PMA(제조사 대체) 항공부품 시장의 최대 독립 플레이어.
- **주요 세그먼트**:
  - Flight Support Group(FSG): 엔진·기체 교체부품 제조/수리/유통, 매출 비중 최대. Wencor 인수(2023) 이후 규모 확대.
  - Electronic Technologies Group(ETG): 방산·우주·통신용 전자부품, 상대적으로 장주기·고성능 제품.
- **경쟁사**: TransDigm(가장 유사한 비즈니스모델·애프터마켓 마진 비교대상), AAR Corp, GE/Rolls-Royce(OEM), Boeing Distribution, Magellan Aerospace, General Atomics, Radiall, Hanwha Aerospace(ETG).
- 최근 5년 매출 CAGR 약 23.7%(WebSearch, StockStory 인용).
