# HEI.A (HEICO Corporation — Class A Common Stock) 데이터 팩

조회일: 2026-07-16 | 통화: USD | 거래소: NYSE
주: HEICO는 보통주(HEI)와 Class A(HEI.A, 의결권 1/10)로 구분 상장. 재무제표는 연결 기준으로 양 클래스 동일, 주가·거래량·기관 보유율만 클래스별 상이.

## 1. 기업 개요
- **사업 모델**: 항공우주/방위산업 부품 및 전자부품 제조·MRO(정비·수리·오버홀) 전문 기업. OEM 대비 저가의 PMA(Parts Manufacturer Approval) 인증 대체부품이 핵심 경쟁력.
- **주요 세그먼트** (FY2026 Q2 기준):
  - Flight Support Group(FSG): 항공기·엔진 교체부품, 수리·오버홀·유통. 매출 비중 최대, 영업이익률 26.2%.
  - Electronic Technologies Group(ETG): 항공우주·방위·우주·의료·통신용 고신뢰성 전자부품/서브시스템. 영업이익률 26.5%.
- **경쟁사**: TransDigm(가장 근접한 비즈니스모델 비교군), Teledyne Technologies, Amphenol(일부 커넥터/전자 영역), FSG 부문은 GE·Rolls-Royce 등 OEM 및 독립 애프터마켓 업체와도 경쟁.
- **회계연도**: 10월 31일 종료 (FY2025 = 2024-11~2025-10).
