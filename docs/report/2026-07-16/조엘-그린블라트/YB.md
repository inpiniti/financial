# YB 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 보유(관심 유니버스 편입, 소형 분산 비중) | 확신도: 5/10

## 한 줄 결론
자본수익률(ROC)은 35~47%로 마법 공식이 좋아하는 최상위권 수준이지만, 이익수익률(EY)의 분모인 EV(기업가치)가 순현금 과잉으로 사실상 0에 근접하거나 음수가 되는 이례적 구간에 있어 — "좋은 기업"임은 숫자가 증명해도 "얼마나 싸게 샀는가"는 표준 공식으로 안정적으로 답하기 어렵다.

## 내 철학에서 본 이 기업
나는 스토리를 안 본다. "AI 보험중개 플랫폼이라 성장성이 좋다"는 말에는 관심 없다. 숫자로 말한다 — ROC는 얼마고 EY는 얼마인가. 껌 가게 비유로 치면 YB는 가게 운영에 큰 자본이 안 들어가면서(운전자본 회전이 빠른 중개업) 100원 넣으면 35~47원을 버는 좋은 가게다. 문제는 이 가게 금고 안에 있는 현금이 가게를 사는 값(시가총액)과 맞먹거나 그보다 많다는 것 — 공짜보다 싸게 사는 셈인지, 아니면 시장이 그 현금과 이익의 지속가능성을 못 믿는 것인지, 공식만으로는 결론이 안 난다.

## 체크리스트 평가

| 항목 | 평가 | 근거 |
|---|---|---|
| ① 자본수익률(ROC) | ○ | 35.8%~46.8%(연환산 참고) — 자산경량형 사업, 유형자본 대비 이익 창출력 우수 |
| ② 이익수익률(EY) | △ | EV가 순현금 과잉으로 0 근접~음수 → 표준 EY 산출 불안정. 참고 E/P(PER역수) = 30.8% |
| ③ 순위 결합 | △ | ROC는 확실한 상위권, EY는 계산 자체가 이례적이라 순위 매기기 곤란 |
| ④ 규모 필터 | △ | 시총 $625.8M은 통과하나 float 4.84M주로 극히 작음 — 유동성 리스크 유의 |
| ⑤ 나쁜 뉴스로 싸진 멀쩡한 기업인가 | ○ | 배당락+저유동성 변동성으로 주가 하락, 실적(YoY 매출+35.6%, 순이익+31.4%)은 견조 — 비관이 다소 과도해 보임 |

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

### 1) 자본수익률(ROC)

```
ROC = EBIT ÷ 투입 유형자본(순운전자본 NWC + 순고정자산)
```

| 항목 | 값 (CNY 백만, 26Q1 스냅샷) |
|---|---:|
| EBIT(FY2025 영업이익) | 1,337 |
| 유동자산 | 5,240 |
| 유동부채 | 1,504 |
| 순운전자본(NWC) | 3,736 |
| 순고정자산 | 확인 불가(데이터팩 미제공, 보험중개 플랫폼 특성상 미미 추정) |
| **투입 유형자본 ≈ NWC** | **3,736** |
| **ROC (FY2025 기준)** | **1,337 / 3,736 ≈ 35.8%** |
| ROC (26Q1 영업이익 437 연환산=1,748) | 1,748 / 3,736 ≈ 46.8% |

→ 어느 쪽으로 계산해도 35~47% 수준의 매우 높은 자본수익률. "100원 넣고 35~47원 버는" 사업 — 좋은 기업의 조건을 확실히 충족한다.

### 2) 이익수익률(EY)

```
EY = EBIT ÷ EV(EV = 시가총액 + 부채 - 현금)
```

| 항목 | 값 |
|---|---:|
| 시가총액(희석, USD) | $636.8M |
| 총부채(CNY 4.25M, 무시 가능) | ≈ $0.6M |
| 총현금(CNY 4,668M, 데이터팩 명시 환산) | ≈ $658M |
| **EV = 636.8 + 0.6 − 658** | **≈ -$20.6M (음수)** |
| EBIT(FY2025, CNY 1,337M) | ≈ $188M (환율 ≈7.1 적용) |
| **EY = EBIT / EV** | **정의상 계산 불가 (분모 음수)** |

- 데이터 팩 원문도 "EV/EBITDA -2.72, 순현금이 시총 초과해 EV 음수 — 해석 주의"라고 명시한다. 마법 공식이 전제하는 "정상적으로 양(+)인 EV" 상황을 벗어난 이례적 케이스다.
- 참고 대체지표로 **PER(trailing) 3.25의 역수인 E/P = 30.8%**를 실질 이익수익률의 근사 하한선으로만 참고한다. (단, 이 역시 EV가 아닌 시가총액 기준이라 마법 공식 정의와는 다르다.)

### 3) 마법공식 순위 결합
ROC 35~47%는 유니버스 상위권에 들 것이 확실하다. 그러나 EY는 표준 정의로는 산출되지 않는 이례적 케이스이므로, 두 지표를 단순 합산해 "최고의 기회"라고 기계적으로 결론 내리는 것은 내 원칙과 어긋난다. 마법 공식은 EV가 정상 범위(양수)인 유니버스를 전제로 설계된 시스템이며, 이런 예외 케이스는 별도 처리 규칙(제외 또는 최상위 EY 티어 임의 배정) 없이는 순위표에 안정적으로 편입할 수 없다.

### 적정가 판단
나는 그레이엄식 목표주가나 DCF는 계산하지 않는다. 마법 공식은 "얼마가 적정가인가"가 아니라 "이 가격에 사도 좋은가"를 답하는 시스템이다. ROC는 확실한 매수 신호를 주지만, EY 계산이 불안정한 이상 "이 가격이 싸다"는 결론까지 기계적으로 내리기는 어렵다. 현재가 $13.58을 단독 매수 근거로 삼기보다, 20~30개 분산 포트폴리오의 작은 한 조각으로만 담는 것이 내 원칙에 맞다.

## 리스크
- **EV 음수(순현금 초과)의 함정**: 종종 "시장이 보고된 이익·현금을 온전히 신뢰하지 않는다"는 신호와 동반된다. 중국 역외 상장기업 특유의 회계 신뢰도·자본 역외송금 리스크를 배제할 수 없다.
- **float 4.84M주로 극히 작음** — 개별 종목 리스크가 커서, 내 방식이면 이런 종목은 20~30개 중 하나로만 담아야지 몰빵하면 안 된다.
- **애널리스트 커버리지 1~2명**뿐이라 정보 비대칭 위험, 회계 이슈나 중국 소형주 특유의 리스크는 두 지표에 담기지 않는다.
- **성장의 지속성**: 영업이익률이 4년 만에 -1.9%→30.6%로 급변 — 마법 공식은 최근 실적을 기계적으로 쓰지만, 이런 급변 기업은 평균회귀 리스크가 크다.
- 최근 주가 -55.5%(52주 고점 대비) 하락 — 나쁜 뉴스처럼 보이지만, 실적은 오히려 개선(매출·순이익 YoY 두 자릿수 증가)되고 있다는 괴리가 흥미롭다. 다만 이 괴리가 "기회"인지 "함정"인지는 EV 왜곡 탓에 공식만으로 단정하기 어렵다.

## 최종 의견
좋은 껌 가게를 찾았다. ROC 35~47%, 100원 넣고 35~47원 버는 사업이다. 그런데 EY를 계산하려니 분모인 EV가 마이너스가 나온다. "가게 금고 안 현금이 가게 값보다 많다"는 뜻인데, 이건 순진하게 "공짜보다 싸게 산다"고 좋아할 일이 아니라 왜 시장이 이 가게를 이렇게 헐값에 넘기려 하는지 의심해봐야 하는 신호이기도 하다. 나는 개별 종목의 뉴스나 직관에 흔들리지 않는다고 늘 말하지만, EV가 음수라는 건 스토리가 아니라 숫자 자체가 공식의 전제를 벗어났다는 뜻이다. 이럴 땐 공식을 억지로 우겨넣기보다, 이 종목을 20~30개 분산 바스켓의 아주 작은 한 조각으로만 담고, 회계 신뢰도와 유동성이 몇 분기 더 검증되는 걸 인내심 있게 지켜보는 게 맞다. 단독 베팅의 대상은 아니다.
