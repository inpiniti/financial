# YB 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 매수 | 확신도: 7/10

## 한 줄 결론
껌 가게 비유로 말하자면, YB는 100원 넣고 매년 30~50원을 벌어들이는 좋은 껌 가게인데, 시장은 그 가게를 "현금 곳간보다도 싸게" 팔고 있다. ROC도 높고 EY(사실상 EV 음수라 무한대에 가까운)도 높다 — 마법 공식이 좋아하는 정확히 그 조합이다.

## 내 철학에서 본 이 기업
나는 스토리를 안 본다. "경영진이 훌륭하다", "AI 보험중개 플랫폼이라 성장성이 좋다" 같은 말에는 관심 없다. 숫자로 말하겠다. ROC는 얼마고 EY는 얼마인가 — 이 두 질문에만 답하면 된다.

## 체크리스트 평가

| 항목 | 평가 | 근거 |
|---|---|---|
| ① 자본수익률(ROC) | ○ | 35.8%~46.8%(연환산) — 자본경량형 사업, 유형자본 대비 이익 창출력 우수 |
| ② 이익수익률(EY) | ○ | EV가 음수(순현금 과잉) → 사실상 초고수익률. 참고 E/P(PER역수) 30.8% |
| ③ 순위 결합 | ○ | 두 지표 모두 상위권 — "좋으면서 동시에 싼" 기업의 전형 |
| ④ 규모 필터 | △ | 시총 $625.8M, float 4.84M주로 매우 작음 — 유동성 리스크 유의, 포트폴리오 편입 비중 축소 필요 |
| ⑤ 나쁜 뉴스로 싸진 멀쩡한 기업인가 | ○ | 배당락+저유동성 변동성으로 하락, 실적(YoY 매출+35.6%, 순이익+31.4%)은 견조 — 비관이 과도해 보임 |

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

### 1) 자본수익률(ROC)

```
ROC = EBIT ÷ 투입 유형자본(NWC + 순고정자산)
```

| 항목 | 값 (CNY 백만, 26Q1 스냅샷) |
|---|---:|
| EBIT(FY2025 영업이익) | 1,337 |
| 유동자산 | 5,240 |
| 유동부채 | 1,504 |
| 순운전자본(NWC) | 3,736 |
| 순고정자산 | 데이터 팩 미제공(자산경량형 사업, ≈0으로 근사) |
| **투입 유형자본 ≈ NWC** | **3,736** |
| **ROC (FY2025 기준)** | **1,337 / 3,736 ≈ 35.8%** |
| ROC (26Q1 연환산: EBIT 437×4=1,748) | 1,748 / 3,736 ≈ 46.8% |

→ 어느 쪽으로 계산해도 35~47% 수준의 매우 높은 자본수익률. "100원 넣고 35~47원 버는" 사업 — 좋은 기업의 조건을 확실히 충족.

### 2) 이익수익률(EY)

```
EY = EBIT ÷ EV(=시가총액+부채-현금)
```

| 항목 | 값 |
|---|---:|
| 시가총액(희석) | $636.8M |
| 총부채 | CNY 4.25M (≈ $0.5M, 무시 가능) |
| 총현금 | CNY 4,668M |
| CNY→USD 환율(역산: BPS $12.28×32.39M주=$397.7M ÷ 자기자본 CNY 3,346M) | ≈ 8.415 |
| 총현금(USD 환산) | 4,668 / 8.415 ≈ $554.7M |
| **EV = 636.8 + 0.5 − 554.7** | **≈ $82.6M** (데이터 팩 원문 서술상 순현금 초과 시나리오도 존재 — 아래 참고) |
| EBIT(FY2025, USD 환산) | 1,337 / 8.415 ≈ $158.9M |
| **EY = 158.9 / 82.6** | **≈ 192%** (환율 역산 오차 감안 시 최소 30%~최대 무한대 구간) |

- 데이터 팩 원문은 "EV/EBITDA -2.72, 순현금이 시총 초과해 EV 음수"라고 명시한다. 환율 역산 방식에 따라 EV가 근소한 플러스~음수 사이를 오간다는 뜻이며, 어느 쪽이든 **EV가 시총 대비 극히 작거나 음수** → EY는 정상적 범위(10~20%)를 훨씬 상회하는 이상치.
- 보수적 참고치로 **PER(trailing) 3.25의 역수 = E/P 30.8%**를 실질 이익수익률의 하한선으로 채택한다.

### 3) 마법공식 순위 결합
ROC 35~47% (높은 상위권), EY 30%+(현금과잉으로 사실상 EV 왜곡, 상위권 확실) → **두 지표 모두 유니버스 최상위권**에 해당할 조합. 합산 순위 기준으로는 "최고의 기회" 범주.

### 적정가 판단
나는 그레이엄식 목표주가나 DCF는 계산하지 않는다. 마법 공식은 "얼마가 적정가인가"가 아니라 "이 가격에 사도 좋은가"를 답한다. ROC 35%+, EY 30%+ 조합은 공식이 원하는 정확한 조합이므로 **현재가 $13.58은 매수 가격대로 판단**한다. 다만 EV 계산이 통화·시점 불일치로 다소 불안정하므로(확인 불가 항목 참조) 확신도를 10이 아닌 7로 낮춘다.

## 리스크
- **float 4.84M주로 극히 작음** — 개별 종목 리스크가 커서, 내 방식이면 이런 종목은 20~30개 중 하나로만 담아야지 몰빵하면 안 된다.
- **환율·시점 불일치**: CNY 재무제표와 USD 시가총액을 교차 환산하는 과정에서 EV·EY 수치에 오차 가능성 — "확인 불가" 항목으로 남겨둔다.
- **애널리스트 커버리지 1~2명**뿐이라 정보 비대칭 위험, 회계 이슈나 중국 소형주 특유의 리스크(감사 신뢰성 등)는 두 지표에 담기지 않는다.
- 최근 주가 -55.5%(52주 고점 대비) 하락 — 나쁜 뉴스처럼 보이지만, 실적은 오히려 개선(매출·순이익 YoY 두 자릿수 증가)되고 있다는 점이 핵심. 이 괴리가 바로 마법 공식이 포착하려는 지점이다.

## 최종 의견
좋은 껌 가게를 찾았는데, 그 가게 금고 안에 있는 현금만 해도 가게 전체 가격과 맞먹거나 더 많다면 어떻게 하겠는가? 사야 한다. YB는 ROC 35~47%로 자본을 매우 효율적으로 굴리는 사업이면서, EV가 시총 대비 왜소하거나 음수에 가까울 정도로 싸게 거래되고 있다. 배당락과 저유동성 변동성 탓에 주가가 눌렸을 뿐, 실적은 YoY로 계속 좋아지고 있다. 이런 뉴스발 비관이야말로 공식이 가격을 싸게 만드는 정확한 메커니즘이다. 다만 시총 $625M, float 5백만 주 남짓의 소형주이니 20~30개 분산 포트폴리오의 한 조각으로만 담아라. 이 한 종목에 몰빵하는 순간 나는 더 이상 그린블라트가 아니다.
