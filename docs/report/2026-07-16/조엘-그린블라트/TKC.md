# TKC 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 2/10

## 한 줄 결론
마법 공식이 요구하는 ROC(자본수익률)·EY(이익수익률) 계산에 필요한 핵심 원자료(순고정자산, EBIT 정밀치, TRY/USD 환율, EV)가 데이터 팩에서 대거 "확인 불가"이므로, 공식을 정직하게 적용할 수 없다. 억지로 숫자를 끼워 맞춰 순위를 매기느니 관망한다.

## 내 철학에서 본 이 기업
나는 이 사업이 좋은지 나쁜지, 싼지 비싼지를 오직 두 숫자로 판단한다. 스토리("5G 스펙트럼 25% 더 많다", "디지털 사업 +64% 성장")는 참고만 할 뿐 판단에 넣지 않는다. TKC는 터키 1위 통신사(시장점유율 41%, 3사 과점)로 사업 자체는 안정적으로 보이지만, 초인플레이션 국가의 TRY 재무제표와 USD 표시 ADR 주가가 뒤섞여 있어 "숫자로 말하기"가 매우 어렵다. 이 통화 불일치 자체가 이미 하나의 경고 신호다.

## 체크리스트 평가

| 항목 | 평가 | 근거 |
|---|---|---|
| ROC 계산 가능 여부 | × | 순운전자본은 산출 가능(75.41조 TRY = 유동자산 184.36조 − 유동부채 108.95조)하나, 순고정자산 원자료 미제공으로 분모 완성 불가 |
| EY 계산 가능 여부 | △ | EBIT(TTM 근사) 41.67조 TRY는 산출 가능하나, TRY/USD 환율 미제공으로 USD EV 환산 불가 → 신뢰 가능한 EY 산출 실패 |
| 재무 안전성 (이자보상배율) | × | 이자비용 항목 미확보, "확인 불가" |
| 규모 필터 | ○ | 시가총액 $51.28억 — 마법 공식 유니버스 하한선(통상 $50M~) 상회, 규모는 통과 |
| 회계 신뢰성 | △ | 2025 연간 EPS(20.2 TRY) 확정치 여부 불명, 2026Q1 FCF -28.86조 TRY 급감 원인 미분해 |
| 부채 추세 | × | 총부채 2026-03 206.35조, 2024말 대비 +51% 급증. 부채/자본 71%로 악화 |

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

**1) 이익수익률 EY = EBIT ÷ EV**

- EBIT(TTM) ≈ 매출 243.70조 TRY × 영업이익률 17.1% = **41.67조 TRY**
- EV(USD) 산출 시도:
  - 시가총액 = $51.28억
  - 순부채(TRY) = 총부채 206.35조 − 현금 95.77조 = **110.58조 TRY**
  - TRY→USD 환산 환율: 데이터 팩 미제공, **확인 불가**
  - → EV(USD) 정식 산출 **불가**

  참고용 概算(데이터 팩 외부 가정, 신뢰 불가 명시): 1USD≈40TRY 가정 시
  - 순부채 ≈ $27.6억, EV ≈ $78.9억
  - EBIT(USD 환산) ≈ $10.4억
  - **EY(참고치) ≈ 13.2%** ← 매매 판단에 사용 금지, 원자료 확보 전까지 무효

**2) 자본수익률 ROC = EBIT ÷ (순운전자본 + 순고정자산)**

- 순운전자본 = 184.36조 − 108.95조 = **75.41조 TRY** (산출 가능)
- 순고정자산 = **확인 불가** (원자료 미제공)
- → ROC 정식 산출 **불가**

- 참고 근사(ROC 대용 불가, 단순 비율): EBIT(TTM) 41.67조 ÷ 자기자본 290.91조 ≈ **14.3%** — 이것은 ROC 정의와 다르므로 순위 매김에 사용 불가

**3) 결합 순위**
두 지표 중 어느 하나도 원칙대로 산출되지 않았으므로 마법 공식 순위표 자체를 만들 수 없다. "적정가"는 산출하지 않는다.

## 리스크
- TRY 초인플레이션·리라 약세로 인한 환위험 — 부채·이자비용·USD환산매출 변동성 확대
- 2026Q1 총부채 급증(+51%)의 원인 미상 — 5G·파이버 CapEx 조달용인지 다른 이유인지 불명확
- FCF -28.86조 TRY 급감, 세부 분해 불가 — 현금창출력 왜곡 가능성
- 애널리스트 컨센서스 표본이 극소(Yahoo 기준 2명)이거나 소스별 상이(2~10명) — 신뢰도 낮음
- 데이터 결측 자체가 리스크: ROIC, 이자보상배율, EV/EBITDA, 정밀 PBR/PSR 모두 확인 불가

## 최종 의견
좋은 기업인지 물으면 나는 항상 되묻습니다. "그래서 ROC는 얼마입니까?" 이 회사는 순고정자산 자료가 없어 답할 수가 없습니다. 싸냐고 물으면 "EY는 얼마입니까?" 물어야 하는데, 환율 자료가 없어서 EV조차 제대로 못 구합니다. 억지로 환율을 가정해서 EY 13%쯤 나온다고 말할 수는 있지만, 그건 제가 여러분에게 늘 경고하는 바로 그 함정입니다 — 없는 숫자를 끼워 맞춰 스토리를 만드는 것. 저는 스토리를 팔지 않습니다. 숫자가 갖춰질 때까지는, 이 종목은 제 20~30개 포트폴리오에 들어올 자격이 없습니다. 자료가 다 모이면 그때 다시 계산해봅시다.
