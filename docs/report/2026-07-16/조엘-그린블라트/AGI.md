# AGI 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 보유 | 확신도: 5/10

## 한 줄 결론
좋은 기업(ROC 21%)이지만 아주 싸진 않은 기업(EY 7.5%)이다 — 마법 공식이 말해주는 건 "훌륭하지만 최상급 매수 기회는 아니다"이다.

## 내 철학에서 본 이 기업
저는 껌 가게 이야기를 자주 합니다. AGI는 온타리오와 멕시코에서 금이라는 "제품"을 캐서 파는 사업체입니다. 최근 3년간 영업이익률이 18.2%→31.1%→37.5%→48.6%로 극적으로 개선됐죠. 이건 스토리가 아니라 숫자입니다. 문제는 "이 좋은 사업체를 지금 얼마에 살 수 있는가"입니다. 6월 지진·정전 사고로 주가가 52주 고점 대비 -47.6% 빠졌는데, 이게 진짜 싸진 건지 EBIT도 같이 훼손된 건지를 숫자로 확인해야 합니다.

## 체크리스트 평가

| 항목 | 평가 | 근거 수치 |
|---|---|---|
| ROC (자본수익률) | ○ | 21.1% — 채굴업 치고 우수, ROE 25.9%·ROA 12.0%와 정합 |
| EY (이익수익률) | △ | 7.51% — "싸다"고 부를 10%+ 구간에는 못 미침 |
| 순위 결합 (좋으면서 싼가) | △ | 좋은 쪽은 확실하나 싼 쪽은 보통 수준 |
| 규모 필터 | ○ | 시총 $12.19B, 유니버스 제외 대상 아님 |
| 재무 안전성(참고) | ○ | 유동비율 2.01, 순현금 포지션(부채 $220M vs 현금 $660~700M) |

## 밸류에이션 판단 (계산 과정)

**Step 1. EBIT 산출**
```
EBIT = 매출 × 영업이익률 = $1,808.8M × 48.6% ≈ $879.1M  (2025 회계연도)
```

**Step 2. 투입자본 산출 (ROC 분모)**
```
순운전자본 = 유동자산 $1,169.3M − 유동부채 $580.8M = $588.5M
순고정자산: PP&E 원자료 확인 불가 → 대체 추정치 사용
투하자본(대체) = 자기자본 $4,607.8M + 총부채 $219.7M − 현금 $659.5M ≈ $4,168.0M
```

**Step 3. ROC 계산**
```
ROC = EBIT ÷ 투하자본 = $879.1M ÷ $4,168.0M ≈ 21.1%
```

**Step 4. EV 산출 (EY 분모)**
```
EV = 시가총액 $12,190M + 총부채 $219.7M − 총현금 $703.9M = $11,705.8M
```

**Step 5. EY 계산**
```
EY = EBIT ÷ EV = $879.1M ÷ $11,705.8M ≈ 7.51%
```

**결론**: ROC 21.1%는 우수, EY 7.51%는 보통. 두 지표를 합산 순위로 매길 전체 유니버스 데이터는 없으나, 그린블라트 기준으로 "EY 10%대"가 눈에 띄게 싼 구간임을 감안하면 AGI는 "좋은 기업을 적정가에" 사는 상황에 가깝지, "좋은 기업을 헐값에" 사는 상황은 아니다. 마법 공식은 스토리를 보지 않으므로 지진·정전 뉴스는 판단에 넣지 않되, 그 뉴스가 EBIT 전망(FY2026 컨센서스 90일 전 $2.99 → 현재 $2.37, -20.7%)을 낮추고 있다는 점은 EY의 분자가 앞으로 낮아질 수 있음을 시사한다 — 이는 공식이 반영할 미래 데이터일 뿐, 지금 개입할 사안은 아니다.

## 리스크
- EBIT가 2025년 급증한 이유(마진 확대)가 일시적(금값 급등 $4,200~4,800/oz 구간)일 가능성 — 금값이 정상화되면 EBIT와 EY 모두 재계산 필요.
- 2025년 순이익 Yahoo 원자료 내부 불일치($885.8M vs $587.1M) — 확인 불가, EBIT 추정의 정확도에 유의.
- 순고정자산 원자료 미확보로 ROC 분모를 대체 추정 — 실제 유형자산 기준 ROC와 오차 가능.
- 개별 종목 리스크(지진·정전 등 운영 사고)는 마법 공식이 담지 못하는 부분 — 저라면 이 종목 하나에 집중하지 않고 20~30개 분산 포트폴리오의 일부로만 취급.

## 최종 의견
자, 숫자로 봅시다. ROC 21%는 훌륭합니다. 100원 투자해서 21원을 버는 사업체는 흔치 않아요. 하지만 EY가 7.5%라는 건, 인수자 입장에서 이 사업 전체를 사면 매년 7.5%를 번다는 뜻입니다. 나쁘지 않지만 제가 찾는 "비이성적으로 싼" 구간은 아닙니다. 주가가 52주 고점에서 반토막에 가깝게 빠졌다고 해서 무조건 싸진 게 아니에요 — EBIT 전망도 같이 깎이고 있으니까요. 저는 이 종목을 뉴스로 판단하지 않습니다. 공식이 말해주는 건 "좋은 기업, 적정 가격"입니다. 20~30개 분산 포트폴리오의 한 조각으로는 나쁘지 않지만, 지금 이 한 종목에 확신을 걸 자리는 아닙니다.
