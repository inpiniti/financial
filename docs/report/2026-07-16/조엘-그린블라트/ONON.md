# ONON 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 6/10

## 한 줄 결론
ROC는 확실히 좋은 기업(근사 27%대)임을 보여주지만, EY 3.5%는 "싸다"는 조건을 충족하지 못한다 — 좋은 기업이지 싼 기업이 아니다.

## 내 철학에서 본 이 기업
제이슨의 껌 가게로 치면, ONON은 껌 한 통 팔 때마다 62.8%를 남기는(매출총이익률) 아주 잘나가는 가게입니다. 손님도 계속 늘어(매출 FY2022 1,222M→FY2025 3,014M, 3년새 2.5배) 좋은 사업이라는 건 숫자가 말해줍니다. 문제는 이 가게를 지금 얼마에 사야 하냐는 것입니다. 시가총액 $12.49B, PER 40.7배를 지불해야 한다면, 그건 이미 동네 사람들이 이 가게가 좋다는 걸 다 알고 줄 서서 웃돈 얹어 사려는 상황과 같습니다.

## 체크리스트 평가
| 항목 | 평가 | 근거 |
|---|---|---|
| ROC 높음 ("좋은 기업인가") | ○ | 근사 ROC ≈ 27.6% (순운전자본 기준, 순고정자산 확인 불가로 보수적 하한). 매출총이익률 62.8%, 영업이익률 12.5%로 개선 추세 |
| EY 높음 ("싼 가격인가") | × | EY ≈ 3.5% (환율 근사). PER 40.7x trailing, EV/EBITDA 25.6x — 시장 평균 대비 명백히 비쌈 |
| 두 지표 동시 충족 | × | ROC는 상위권 후보, EY는 하위권. 마법공식 합산 순위에서 EY가 발목을 잡음 |
| 규모 필터 | ○ | 시총 $12.49B로 충분히 크고 유동성 있음 |
| 비관으로 인한 저평가 신호 | △ | CEO 사임 급락(-11.2%), 소송 리스크, 관세 우려로 52주 고점 대비 -31% 하락 — 그린블라트식 "나쁜 뉴스로 싸진 좋은 기업" 패턴과 유사해 보이나, EY 계산상 여전히 절대적으로 비쌈 |

## 밸류에이션 판단 (계산 과정)

**1) EBIT (FY2025, CHF)**: 영업이익 = 377.0M

**2) ROC 계산**
```
투입 유형자본 ≈ 순운전자본 = 유동자산(2,053.3) − 유동부채(688.6) = CHF 1,364.7M (26Q1 기준)
※ 순고정자산(PP&E) 데이터 팩 미확보 → 확인 불가, 분모 과소평가 가능성 있어 아래 ROC는 상한 근사치
ROC ≈ 377.0 / 1,364.7 = 27.6%
```

**3) EV 계산 (USD, 환율 CHF/USD 확인 불가 → 근사 1.12 가정)**
```
시가총액           = $12.49B
+ 총부채(CHF 541.9M ≈ $0.61B, 근사)
− 현금성자산(CHF 1,020.4M ≈ $1.18B, 근사)
= EV ≈ $11.92B
```

**4) EY 계산**
```
EBIT(CHF 377.0M ≈ $422M, 근사)
EY = EBIT / EV ≈ 422 / 11,920 ≈ 3.5%
```

**5) 마법공식 판정**
- ROC 27.6% (근사) → 상위권 후보, "좋은 기업" 조건 충족 가능성 높음
- EY 3.5% (근사) → 하위권, "싼 가격" 조건 미충족. 국채 수익률 대비도 우위 없음
- **결론: 한 지표만 우수 → 그린블라트가 회피하라고 명시한 "좋지만 싸지 않은 기업" 패턴에 해당**

## 리스크
- 환율 근사 적용(CHF→USD)으로 EV·EY 계산에 오차 존재 — 방향성은 훼손되지 않을 정도의 마진
- 순고정자산 미확보로 ROC가 상한 편향(실제는 이보다 낮을 가능성)
- CEO 교체발 증권소송 리스크, 관세 우려 — 다만 마법공식은 이런 정성적 리스크를 지표에 반영하지 않는 게 원칙
- 베타 2.12로 고변동성 — 포트폴리오 20~30종목 분산 없이 단독 보유 시 개별 리스크 노출

## 최종 의견
제이슨이 껌 가게를 하나 더 열까 고민할 때 저는 항상 두 가지만 물어봅니다. "얼마나 잘 버니?"와 "얼마에 살 수 있니?" ONON은 첫 번째 질문엔 자신 있게 대답합니다 — ROC 근사 27.6%, 훌륭합니다. 그런데 두 번째 질문에서 머뭇거립니다. EY 3.5%, PER 40.7배. 이건 껌 가게를 사는 게 아니라 이미 유명해진 껌 가게에 웃돈 주고 줄 서는 겁니다. CEO 사임에 놀란 시장이 -31% 할인을 해줬다고는 하나, 아직 "비이성적으로 싸다"고 부를 수준은 아닙니다. 마법 공식은 좋으면서 동시에 싼 기업을 찾으라고 말합니다. 지금 ONON은 절반만 통과했습니다. 저는 스토리에 혹하지 않습니다. EY가 더 매력적인 가격까지 내려오거나, 이익이 더 늘어 EY가 자연히 올라올 때까지 기다리겠습니다.
