# PGY 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 4/10

## 한 줄 결론
EY(이익수익률)는 17.8%로 훌륭한 가격 신호를 주지만, ROC(자본수익률)는 마법 공식의 표준 정의로 계산할 원자료(순운전자본·순고정자산)가 없어 산출 불가 — 절반의 공식만으로는 매수 버튼을 누를 수 없다.

## 내 철학에서 본 이 기업
나는 개별 종목 스토리를 안 봅니다. "이 사업체는 얼마나 돈을 잘 버는가?"와 "그 사업체를 지금 얼마에 살 수 있는가?" 이 두 질문에 숫자로만 답합니다. PGY(Pagaya)는 대출 자산을 자기 대차대조표에 최소한만 올리고 수수료로 버는 AI 신용평가 플랫폼입니다. 문제는 이런 "자산경량형(asset-light)" 금융 플랫폼은 전통 제조업처럼 순운전자본·순고정자산이 뚜렷하지 않아서, 제가 켐프의 껌 가게에 적용하던 방식 그대로 ROC를 뽑아내기가 까다롭다는 겁니다. 데이터 팩도 정직하게 "ROIC 확인 불가"라고 못 박아뒀습니다.

## 체크리스트 평가

| 항목 | 평가 | 근거 |
|---|---|---|
| 1. 자본수익률(ROC) 산출 가능 | × | 순운전자본/순고정자산 원자료 데이터 팩에 없음. ROE 16.3%, ROA 12.6%는 참고치일 뿐 마법공식 정의 아님 |
| 2. 이익수익률(EY) 산출 가능 | ○ | EBIT $223.8M(FY2025) / EV $1.26B = 17.76% — 계산 가능, 수치 매력적 |
| 3. 두 지표 모두 상위권(合산 순위 우수) | × | ROC 부재로 합산 순위 자체를 매길 수 없음 |
| 4. 규모 필터 통과 | ○ | 시총 $1.49B, 유니버스 제외 대상 아님 |
| 5. 회계 이익≒실질 현금흐름 정합성 | △ | FY2025 영업이익 $223.8M vs FCF $224.7M — 거의 일치(양호). 다만 FY2022~2024는 적자·전환기였고 흑자 이력이 4~5분기로 짧음(2Q25~1Q26) |

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

### 1) 이익수익률(EY) — FY2025 기준
```
EBIT(FY2025 영업이익)     = $223,808천
EV(현재, 2026-07-15 기준)  = $1,260,000천
EY = EBIT / EV = 223,808 / 1,260,000 = 17.76%
```

### 2) 이익수익률(EY) — TTM(최근 4개분기) 기준, 정밀도 보완
```
2Q2025 영업이익 = $47,785천
3Q2025 영업이익 = $69,680천
4Q2025 영업이익 = $65,943천
1Q2026 영업이익 = $61,052천
TTM EBIT 합계   = $244,460천

TTM EY = 244,460 / 1,260,000 = 19.40%
```
→ 두 기준 모두 EY 17~19% 구간. 국채수익률(참고: 통상 4%대) 대비 압도적으로 높은 이익수익률 — "싼 가격" 조건은 강하게 충족.

### 3) 자본수익률(ROC) — 산출 시도 및 한계
```
ROC = EBIT ÷ (순운전자본 + 순고정자산)
```
- 데이터 팩에 유동자산($1,544,326천)·유동부채($238,148천)는 있으나, 이는 순운전자본이 아니라 유동성 지표(유동비율 6.48x)로만 제시됨. 여기서 "영업 관련" 유동자산·부채만 분리한 순운전자본을 추출할 원자료가 없음.
- 순고정자산(PP&E) 수치 자체가 데이터 팩에 부재.
- 대체 지표(ROE 16.3%, ROA 12.6%, financialData 기준)는 참고는 되나 마법 공식이 정의한 ROC와 다른 개념(부채·세금 효과가 섞여 있어 그린블라트가 애초에 회피하려던 왜곡을 그대로 안고 있음).
- **결론: ROC 확인 불가 → 마법 공식의 "합산 순위" 자체를 매길 수 없음.**

### 4) 참고 배수 (마법공식 대체 확인용)
| 지표 | 값 |
|---|---|
| EV/EBITDA | 4.16x |
| PER(trailing) | 16.33x |
| PER(forward) | 5.08x |
| PBR | 2.81x |

EV/EBITDA 4.16x는 마법 공식 스크리너에서 상위권에 걸릴 법한 낮은 배수입니다. 이는 EY가 좋다는 계산과 방향이 일치합니다. 다만 저는 이런 대체 배수로 ROC를 "짐작"하지 않습니다. 짐작이 시작되는 순간 공식이 아니라 제 직관이 개입하게 되고, 그것이야말로 제가 가장 경계하는 함정입니다.

## 리스크
- **공식 자체의 한계가 이 종목에서 정확히 발현**: 회계 이익이 실제 사업 구조(자산경량형 플랫폼)를 온전히 반영하는지 불확실. 2022~2024년 3년 연속 영업적자였고, 흑자 전환은 5개 분기에 불과 — 지속가능성 검증 기간이 짧음.
- 2026-02-09 4Q2025 실적 발표 당일 -24% 급락(네트워크 볼륨 성장 둔화) — 실적 변동성이 크고, 이는 마법 공식이 "포트폴리오 20~30종목 분산"으로 흡수하도록 설계된 개별 종목 리스크의 전형.
- 증권소송 조사(Iceberg Research 공매도 리포트發), Klarna 소송 — 세부 청구 내용 확인 불가. 정성적 리스크는 제 지표에 담기지 않으므로 별도로 인지만 해둠.
- Beta 5.34로 매우 높은 변동성 — 분산 없이 단일 종목으로 접근하면 심리적으로 버티기 어려운 유형.

## 최종 의견
숫자로 말하죠. EY는 17.8%(FY2025 기준) 혹은 TTM 19.4% — 이건 "싼 가격"입니다. 국채 이자의 4배가 넘습니다. 문제는 제 공식이 두 다리로 서 있다는 겁니다. "싼가?"라는 질문엔 답했지만 "좋은 기업인가?"라는 질문, 즉 ROC를 계산할 재료가 이 데이터 팩엔 없습니다. 저는 EV/EBITDA 4배가 낮다고 해서, 혹은 ROE 16%가 괜찮아 보인다고 해서 그걸 ROC 대용으로 슬쩍 끼워 넣지 않습니다. 그렇게 시작하면 다음엔 "경영진이 훌륭해서요", "AI가 대세라서요" 같은 스토리가 기어들어 옵니다. 공식이 반이라도 없으면 공식이 아닙니다. 재무제표에서 순운전자본과 순고정자산을 직접 확인하기 전까지는 관망입니다. 다만 EY 수치 자체는 충분히 매력적이니, ROC를 확인할 수 있는 원자료(10-K 대차대조표 세부 항목)를 구하는 순간 재평가할 가치는 있습니다.
