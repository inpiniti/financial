# HMY 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 4/10

## 한 줄 결론
ROC는 좋은 기업의 신호를 보이지만(약 30%), 이익수익률(EY) 계산에 필요한 통화 환산·EV 구성요소가 데이터 팩에서 확인 불가라 "싼가"를 자신 있게 말할 수 없다. 계산이 안 되면 사지 않는다.

## 내 철학에서 본 이 기업
껌 가게 얘기로 시작해봅시다. 제이슨의 껌 가게가 100원을 투자해서 매년 30원을 번다면 좋은 사업입니다. 하몬디골드(HMY)는 FY2024 기준 그 비슷한 수익률을 냅니다. 문제는 "그 가게를 지금 얼마에 살 수 있는가"입니다. 이 회사는 남아공 랜드(ZAR)로 장부를 쓰고 미국에서 달러(USD)로 거래됩니다. 환율 자료가 데이터 팩에 없으니, 제가 EV를 정확히 계산할 수 없습니다. 계산 못 하는 걸 아는 척 사는 건 마법 공식이 아니라 도박입니다.

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

### 1) ROC (자본수익률)

```
ROC = EBIT ÷ (순운전자본 + 순고정자산)
```

- EBIT (FY2024, 영업이익) = 11,533 ZAR백만
- 순운전자본 = 유동자산(11,497) − 유동부채(7,359) = 4,138
- 순고정자산: 데이터 팩에 PP&E 별도 항목 없음 → **확인 불가**
- **근사 대체**: 유형투하자본 ≈ 자본총계(40,774) − 현금(4,693) + 총부채(2,300) = 38,381
  (그린블라트 원 산식의 정확한 대체가 아님을 명시 — 순수 순운전자본+순고정자산 분해가 안 되어 근사치로만 사용)

```
ROC(근사) = 11,533 / 38,381 = 30.0%
```

참고 대조: Yahoo ROE 33.49%, ROA 17.30% — 방향성 정합적, 우량 기업 신호로 판단.

### 2) EY (이익수익률)

```
EY = EBIT ÷ EV,  EV = 시가총액 + 총부채 − 현금
```

- 시가총액(USD) = $9,565M
- 총부채(ZAR, Q2FY26) = 12,894 → USD 환산 환율 데이터 팩 미기재 → **확인 불가**, 부득이 상식적 근사 환율(ZAR/USD ≈ 18, 데이터 팩 밖 추정치이므로 신뢰도 낮음) 적용
  - 총부채(USD 근사) ≈ 716
  - 현금(USD 근사, 7,350÷18) ≈ 408
- EV(근사) = 9,565 + 716 − 408 = **9,873M**
- EBIT(USD 근사, 11,533÷18) ≈ $641M

```
EY(근사) = 641 / 9,873 = 6.5%
```

**이 수치는 신뢰도가 낮다.** 환율을 데이터 팩 밖에서 임의로 가정했고, EV/EBITDA(0.511)·PEG(0.03) 등 팩 내 관련 지표도 "비정상적"이라고 이미 경고돼 있다. 이익수익률이 6% 안팎이라면 마법 공식 기준으로 특별히 싸다고 할 정도는 아니다(상위권 후보는 통상 두 자릿수 EY를 보인다).

### 순위 결합 판단

두 지표 중 ROC는 상위권 후보(30%대)로 보이나, EY는 계산 신뢰도가 낮고 값 자체도 눈에 띄게 높지 않다. 마법 공식은 "좋으면서 동시에 싼" 기업을 찾는 것인데, 지금은 "좋아 보이지만 싼지는 계산이 불확실한" 상태다.

## 리스크

- **계산 리스크(가장 중요)**: EV 산출에 필요한 환율, 순고정자산 세부 항목이 확인 불가. 근사치로 낸 EY(6.5%)는 실제와 크게 다를 수 있다.
- 데이터 팩 자체가 EV/EBITDA·PEG를 "비정상적 신뢰도 낮음"으로 명시 — 소스 데이터 품질 문제.
- 재무 안전성: 유동비율이 1.56(FY2024)에서 0.54(Q2FY26)로 급락, 원인 불명.
- 운영 리스크: AISC +25% YoY, 생산량 -9% YoY — 사업 자체의 이익창출력(EBIT)이 최근 분기 악화 추세일 가능성. 마법 공식은 과거 회계 수치를 쓰므로 이런 최근 악화를 즉시 반영하지 못하는 한계가 있다.
- FY2025 연간 실적 전체가 확인 불가 — ROC·EY 계산이 FY2024 구 데이터에 의존.

## 최종 의견

제이슨이라면 이렇게 물을 겁니다. "이 가게, 얼마 투자해서 얼마 버는데요?" ROC는 대답이 나옵니다, 30% 근처로 나쁘지 않습니다. 그런데 "그럼 지금 얼마에 살 수 있는데요?"라고 물으면 저는 정확한 답을 못 합니다. 랜드와 달러를 섞어 쓰는 회사인데 환율 자료가 없고, EV/EBITDA는 데이터 자체가 "말도 안 되는 숫자"라고 팩이 스스로 경고하고 있습니다.

저는 스토리로 사지 않습니다. 숫자로 삽니다. ROC 숫자는 있는데 EY 숫자를 믿을 수 없다면, 그건 마법 공식의 절반만 계산한 것이지 마법 공식이 아닙니다. 공식이 완성되지 않으면 순위를 매길 수 없고, 순위를 매길 수 없으면 사지 않습니다. 이건 비관도 낙관도 아니고, 그냥 계산이 안 끝난 겁니다. 환율과 최신 연간 실적이 확인되면 다시 계산하겠습니다.
