# PSTL 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 매도 | 확신도: 7/10

## 한 줄 결론
좋은 기업일 수는 있어도(그건 REIT 전용 잣대로 따져야 할 문제다) 내 두 가지 잣대 — 자본수익률과 이익수익률 — 로는 "좋지도 싸지도 않은" 전형적인 마법공식 하위권 종목이다.

## 내 철학에서 본 이 기업
껌 가게 이야기를 다시 해보자. 제이슨의 껌 가게가 좋은 사업인 이유는 적은 돈을 넣고 많은 돈을 뽑아내기 때문이었다. PSTL은 우체국 건물을 사서 임대료를 받는 사업이다. 나쁜 사업은 아니다 — USPS라는 정부 신용의 임차인, 니치 시장 독점적 지위, 연간 임대료 escalator까지 갖췄다. 하지만 내 공식은 스토리를 안 듣는다. "이 사업에 들어간 자본 대비 얼마를 버는가", "지금 얼마에 살 수 있는가" 이 두 숫자만 본다. 그리고 이 두 숫자가 말해주는 답은 신통치 않다.

## 밸류에이션 판단 (마법공식 계산 과정)

### 1) 자본수익률 (ROC = EBIT ÷ (순운전자본 + 순고정자산))

| 항목 | 값(천$) | 비고 |
|---|---|---|
| EBIT (FY2025 영업이익) | 33,612 | |
| 유동자산(1Q26) | 13,675 | |
| 유동부채(1Q26) | 64,137 | |
| 순운전자본(NWC) = 유동자산−유동부채 | −50,462 | 마이너스 |
| 총자산 근사 = 총부채(386,135)+자기자본(292,163) | 678,298 | |
| 순고정자산 ≈ 총자산 − 유동자산 | 664,623 | 정밀 net PP&E 미확보, 근사치 |
| 투입 유형자본 = NWC + 순고정자산 | 614,161 | |
| **ROC = 33,612 ÷ 614,161** | **≈ 5.5%** | **낮음** |

주의: 이 계산은 net PP&E 정밀치가 아니라 총자산 근사로 대체한 것이라 오차가 있다. 다만 방향성은 명확하다 — REIT는 부동산 자체가 자본집약적 자산이라 마법공식이 원래 찾는 "적은 자본으로 큰 이익을 내는" 유형과는 구조적으로 안 맞는다. 5%대 ROC는 결코 "경쟁우위"를 말해주는 숫자가 아니다.

### 2) 이익수익률 (EY = EBIT ÷ EV)

| 항목 | 값 | 비고 |
|---|---|---|
| 시가총액 | $898.6M | |
| 총부채(1Q26) | $386.1M | |
| 현금(유동자산으로 근사) | $13.7M | 별도 현금 항목 미확보 |
| **EV = 898.6+386.1−13.7** | **$1,271.0M** | |
| EBIT(FY2025) | $33.6M | |
| **EY = 33.6 ÷ 1,271.0** | **≈ 2.6%** | **매우 낮음** |

PER 46배(trailing), EV/EBITDA 19.7배와 정확히 같은 이야기를 하고 있다. "싸다"는 말은 여기에 안 붙는다.

### 3) 종합 순위 판단

| 지표 | 값 | 판정 |
|---|---|---|
| ROC | 약 5.5% (근사) | 하위권 |
| EY | 약 2.6% | 하위권 |

두 지표 모두 하위권 — 내가 가장 경계하는 조합이다. "좋지만 비싼" 것도 "싸지만 나쁜" 것도 아니고, 그냥 둘 다 평범하거나 비싸다. 20~30개 종목으로 포트폴리오를 구성한다면 이 종목은 내 순위표 상위권에 들어올 자리가 없다.

## 리스크

- 계산의 신뢰도: EBIT/EV·ROC 계산 모두 데이터 팩상 근사치다(net PP&E, 현금 항목 정밀치 미확보). 실제 10-K/10-Q 대조 시 숫자가 다소 달라질 수 있으나, 방향성(둘 다 저조)이 뒤집힐 정도는 아니라고 본다.
- 애초에 REIT는 마법공식의 설계 대상이 아니다. GAAP 이익이 감가상각으로 크게 눌려 EBIT/EY가 구조적으로 낮게 나오고, 시장은 AFFO(주당 $1.40~1.42)로 이 종목을 프라이싱한다. 내 공식이 이 업종에 대해 "싸다/비싸다"를 제대로 판별하는지 자체가 의문이다.
- 그럼에도 원칙은 원칙이다. 예외를 인정하기 시작하면 공식은 무너진다.

## 최종 의견
자, 숫자로 말해봅시다. ROC 5.5%, EY 2.6%. "경영진이 훌륭하다", "우체국이 안 망한다"는 말은 제가 좋아하지 않습니다 — 그래서 ROC는 얼마고 EY는 얼마입니까? 라고 되물어야죠. 답은 이미 나왔습니다. 좋지도 싸지도 않습니다. 껌 가게로 치면, 껌 한 통 팔아서 남는 돈도 적은데 가게 인수 가격은 비싸게 부르는 셈입니다. USPS 임대라는 스토리는 매력적이지만 제 포트폴리오 20~30개 종목에 들어갈 자리는 없습니다. REIT 전용 지표(AFFO, cap rate)로 보면 다른 결론이 날 수도 있겠지만, 그건 제 공식이 아닙니다. 제 공식은 이 종목을 하위권에 놓고, 저는 공식을 흔들림 없이 따릅니다.
