# INOD 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 6/10

## 한 줄 결론
INOD는 마법공식의 두 기둥 중 하나(ROC)는 최상위권, 다른 하나(EY)는 최하위권인 "좋지만 싸지 않은" 전형적 회피 유형이다.

## 내 철학에서 본 이 기업
투자의 전부는 좋은 기업을 싼 가격에 사는 것입니다. INOD는 껌 가게 제이슨의 비유로 치면 "장사는 기가 막히게 잘하는데, 가게를 사는 값이 너무 비싼" 케이스입니다. AI 데이터 엔지니어링이라는 사업 자체는 자산을 거의 안 쓰고도(서비스업, 재고 없음, PP&E 미미) 많은 돈을 버는 구조라서 ROC가 매우 높게 나옵니다. 문제는 시장이 이미 그 사실을 알고 주가에 반영해버렸다는 것입니다.

## 체크리스트 평가

| 항목 | 평가 | 근거 수치 |
|---|---|---|
| 1. 자본수익률(ROC) | ○ | EBIT $39,873천 ÷ NWC $84,862천 ≈ 47.0% (또는 EBIT÷(자기자본+부채) ≈ 35.7%) — 두 근사 모두 최상위권 |
| 2. 이익수익률(EY) | × | EBIT $39,873천 ÷ EV $2.02B ≈ 1.97% (Q1 FY26 연환산 시 3.35%) — 무위험금리 이하, 최하위권 |
| 3. 순위 결합 | × | ROC 순위 최상위 + EY 순위 최하위 → 합산 점수 중간~하위, "좋지만 비싼" 함정 |
| 4. 규모 필터 | ○ | 시총 $2.13B, 유니버스 편입 가능 규모 |
| 5. 비이성적 저가 여부 | △ | 52주 고점 대비 -48% 급락했으나, 여전히 PER(TTM) 59.9x, EV/EBITDA 38.9x로 절대 밸류는 비쌈 |

## 밸류에이션 판단 (계산 과정)

**ROC 계산**
```
EBIT (FY2025 영업이익) = $39,873천
순운전자본(NWC) = 유동자산 $135,394천 − 유동부채 $50,532천 = $84,862천
ROC(근사 A, NWC 기준) = 39,873 / 84,862 = 47.0%

투하자본(근사, 자기자본+총부채) = $107,145천 + $4,430천 = $111,575천
ROC(근사 B) = 39,873 / 111,575 = 35.7%
```
※ 순고정자산 데이터 미제공("확인 불가")으로 두 가지 근사 병행 제시. 서비스업 특성상 PP&E 미미 → 실제 ROC는 근사 A(47%)에 더 근접할 가능성.

**EY 계산**
```
EV = $2.02B (시가총액 $2.13B + 부채 $4.43M − 현금 $117.4M 근사)
EY (FY2025 EBIT 기준) = 39,873 / 2,020,000 = 1.97%
EY (Q1 FY26 연환산: 16,900 × 4 = 67,600) = 67,600 / 2,020,000 = 3.35%
```

**역산 매력 구간**
```
EY 8% 도달 필요 EV = EBIT 39,873 / 0.08 = $498,413천(≈$0.5B)
현재 EV 대비 비율 = 0.5B / 2.02B ≈ 0.25
근사 주가 = $65.31 × 0.25 ≈ $16
```
또는 현재 EV·주가를 유지한 채 EBIT가 3~4배(40%+ 성장 가이던스로 3~4년 내 가능권)로 커지면 자연 도달.

## 리스크
- 데이터팩상 순고정자산·이자보상배율·ROIC 직접 산출 데이터 "확인 불가" → ROC는 두 근사치의 범위로만 제시
- EBIT 성장이 가이던스(40%+)에 못 미치면 EY 개선 시나리오가 지연됨
- 마법공식은 개별 종목 스토리 예측 도구가 아니라 유니버스 전체 반복 매매 전략 — 단일 종목 판단에는 원래 한계가 있음을 전제

## 최종 의견
좋은 기업이라는 것과 싼 기업이라는 것은 전혀 다른 이야기입니다. INOD의 ROC 35~47%는 유니버스 최상위권이고, 이건 부인할 수 없는 사실입니다. 하지만 EY가 2~3%대라면, 이건 마법공식이 찾는 "싼 가격"이 절대 아닙니다. 국채가 4%대를 주는데 이 사업 전체를 인수해서 2~3%밖에 못 번다면, 그 값을 치를 이유가 없습니다. 52주 고점 대비 48% 빠졌다고 해서 "쌀 것"이라고 넘겨짚는 것도 제가 가장 경계하는 함정입니다 — 비쌌던 것이 덜 비싸진 것과, 실제로 싼 것은 다릅니다. EBIT가 3~4배로 자라거나 주가가 지금의 4분의 1 수준까지 조정되어야 제 공식표에 상위권으로 올라옵니다. 그때까지는 지켜만 보겠습니다.
