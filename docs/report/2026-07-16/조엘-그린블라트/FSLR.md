# FSLR 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 보유 | 확신도: 6/10

## 한 줄 결론
First Solar는 자본수익률(ROC 약 20.6%)이 준수한 "좋은 기업"이지만, 이익수익률(EY 약 7.2%)은 마법 공식이 원하는 "아주 싼 가격" 수준에는 못 미친다 — 좋지만 헐값은 아니다.

## 내 철학에서 본 이 기업
껌 가게 비유로 말하자면, First Solar는 100원을 넣으면 약 20원을 벌어오는 제법 괜찮은 가게다. 문제는 지금 이 가게를 사려면 얼마를 내야 하느냐다. EV 기준으로 계산해보니 EBIT 대비 약 7%의 수익률이 나온다. 나쁘지 않지만, 내가 진짜 좋아하는 "미스터 마켓이 패닉에 빠져 헐값을 부르는" 수준은 아니다. 4분기 실적 쇼크와 집단소송 뉴스로 시장이 겁을 먹은 건 사실이지만, 숫자가 그 겁먹음을 충분히 보상해줄 만큼 싸지지는 않았다.

## 체크리스트 평가

| 항목 | 평가 | 근거 수치 |
|---|---|---|
| 자본수익률(ROC) | ○ | EBIT $1,596.9M ÷ 투하자본(자기자본+부채-현금) $7,766.7M ≈ **20.6%** — 준수한 수준 |
| 이익수익률(EY) | △ | EBIT $1,596.9M ÷ EV $22,278.9M ≈ **7.17%** — 보통 수준, 압도적 저가 아님 |
| 규모 필터 | ○ | 시가총액 $24,050M, 유니버스 제외 대상 아님 |
| 정성적 스토리 개입 여부 | - | 4Q25 쇼크·집단소송·정책 리스크는 "숫자"로 이미 EV/EBIT에 반영된 것으로 간주, 별도 가중 안 함 |
| 분산 포트폴리오 적합성 | △ | 개별 종목 베팅이 아닌 20~30개 분산의 한 조각으로만 편입 검토 대상 |

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

**1) ROC 계산**
```
투하자본 = 자기자본 + 총부채 - 현금
        = 9,538.0M + 655.3M - 2,426.6M(1Q26 근사)
        = 7,766.7M

ROC = EBIT(FY2025) ÷ 투하자본
    = 1,596.9M ÷ 7,766.7M
    = 20.6%
```
(참고: 데이터 팩 ROIC 15.0%(순이익 기준), ROE 16.0%와 궤를 같이 함. EBIT 기준이라 세전 효과로 다소 높게 산출)

**2) EY 계산**
```
EV = 시가총액 + 총부채 - 현금
   = 24,050.2M + 655.3M - 2,426.6M
   = 22,278.9M

EY = EBIT(FY2025) ÷ EV
   = 1,596.9M ÷ 22,278.9M
   = 7.17%
```

**3) 마법 공식 종합**

| 지표 | 값 | 판단 |
|---|---|---|
| ROC | 20.6% | 좋은 기업 (○) |
| EY | 7.17% | 보통 가격 (△, 10%+가 이상적) |

유니버스 전체 순위표가 없어 그린블라트식 "합산 순위"는 산출 불가(확인 불가). 다만 개별 지표만 놓고 보면 "좋으면서 압도적으로 싼" 조합은 아니고, "좋지만 적당히 싼" 조합에 해당한다. Forward PER 9.55x, PEG 0.60은 성장 프리미엄을 감안한 참고 지표일 뿐 공식 계산에는 포함하지 않았다.

## 리스크
- 4Q2025 가이던스 쇼크(-13.6% 단일일 급락), 증권 집단소송(2026-06-23 제기, 청구마감 2026-08-24) — EBIT의 향후 지속가능성에 불확실성.
- IRA(Section 45X) 세액공제 등 정책 의존도가 높아 EBIT 자체가 정책 변화에 취약. Bernstein은 이 이유로 Underperform(목표가 $217) 제시.
- 회계상 EBIT과 실제 현금흐름의 괴리: 1Q2026 FCF가 -$333.4M로 전환(재고 급증 영향) — 공식이 놓치는 질적 리스크.
- 개별 종목으로 집중 매수하는 것은 내 원칙에 어긋난다. 20~30개 분산의 일부로만 접근해야 한다.

## 최종 의견
좋은 가게인 건 분명합니다, ROC 20%면 나쁘지 않아요. 하지만 지금 가격에 사려면 이익수익률이 7% 남짓 — 국채보다는 낫지만 제가 좋아하는 "거저 줍는" 수준은 아닙니다. 4분기 실적 쇼크에 시장이 겁먹은 건 이해하지만, 그 겁먹음이 숫자에 완전히 반영되어 진짜 헐값을 만들었느냐 하면, 아직은 아닙니다. 저라면 이 종목 하나에 베팅하지 않고, 마법 공식 전체 순위표에서 이보다 EY가 더 높은 종목들과 함께 분산해서 담겠습니다. 개별 종목의 뉴스에 휘둘리지 마세요 — 숫자가 말해주는 대로만 움직이십시오.
