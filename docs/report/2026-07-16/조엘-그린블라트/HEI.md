# HEI 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 7/10

## 한 줄 결론
껌 가게로 치면 장사는 잘하는데(ROC 중간~양호), 가게를 팔 때 부르는 값이 너무 비싸다(EY 약 2%) — 좋은 기업이지만 싼 기업이 아니니 지금은 산다.

## 내 철학에서 본 이 기업
저는 스토리를 안 봅니다. "항공부품 시장 최대 독립 플레이어"라거나 "5년 매출 CAGR 23.7%"라는 말은 듣기 좋지만, 저에게 중요한 건 딱 두 숫자입니다. 이 사업체가 투입한 돈 대비 얼마나 버는가(ROC), 그리고 지금 그 사업체를 통째로 사려면 벌어들이는 이익 대비 얼마를 치러야 하는가(EY). HEI는 첫 번째 질문에는 합격점을 주지만, 두 번째 질문에서 완전히 무너집니다.

## 체크리스트 평가

| 항목 | 평가 | 근거 수치 |

| 규모 필터 통과 | ○ | 시총 $49.13B, 유동성·유니버스 편입에 문제 없음 |

## 밸류에이션 판단 (계산 과정)

### 1) 자본수익률(ROC)
```
데이터팩 제공 ROIC(NOPAT/투하자본, FY2025) = 12.8%
참고: ROE = 순이익/자기자본 = 690.4 / 4,305.3 = 16.0%
참고(거친 근사, 순고정자산 데이터 부재로 완전한 원 공식 대체 불가):
  EBIT / (순운전자본 + 총부채 - 현금)
  = 1,019.0 / (1,524.4 + 2,193.7 - 210.3)
  = 1,019.0 / 3,507.8 = 29.1% (참고용, 신뢰도 낮음)
```
→ ROC는 그레이엄류 안전자산이 아닌 "괜찮은 사업"임을 시사. 다만 마법공식 상위 그룹(통상 20~30%대 ROC)에 확실히 든다고 단언하기엔 데이터가 부족합니다.

### 2) 이익수익률(EY = EBIT/EV)
```
시가총액 = $49,130M
+ 총부채(최신분기) = $2,587.3M
- 현금(최신분기) = $210.3M
= EV = $51,507.0M

EBIT(FY2025 확정) = $1,019.0M
EBIT(TTM 근사, FY26 Q1+Q2 + FY25 Q3+Q4) = 259.9+350.4+248.2+265.0 = $1,123.5M

EY(FY2025 기준) = 1,019.0 / 51,507.0 = 1.98%
EY(TTM 근사)    = 1,123.5 / 51,507.0 = 2.18%
```
→ 이익수익률 약 2.0~2.2%는 제가 사는 기업들의 EY(통상 두 자릿수, 8~15%+)와는 비교 자체가 안 되는 구간입니다. 미국채 장기물 수익률에도 못 미칩니다.

### 3) 순위 결합 (마법공식 스코어)
```
ROC 순위: 중간~양호 (12.8~16% 구간, 최상위권 아님)
EY 순위: 최하위권 (2.0% 내외)
합산 점수 = 중간 순위 + 최하위 순위 = 나쁨
```
→ 두 지표 합산 시 상위권에 들지 못합니다. "좋은 기업을 싸게" 원칙에서 앞쪽 절반만 만족합니다.

## 리스크
- 데이터 한계: 순고정자산 세부치 부재로 ROC 원 공식 정밀 계산 불가 — 위 수치는 대체·근사치임을 밝힙니다.
- 밸류에이션 리스크: PER 61.7x, EV/EBITDA 37.9x — 실적 미스 시 하방 변동성 큼.
- 이중클래스 주식구조(HEI/HEI.A) — 의결권 희석 이슈 세부 확인 불가.
- 참고로 이건 제 리스크 관점이 아니라 그레이엄·버핏류 안전마진 리스크입니다. 제 진짜 리스크는 "이런 종목을 스토리에 흘려 편입하고 싶은 유혹"입니다 — 저는 그 유혹에 넘어가지 않습니다.

## 최종 의견
제이슨의 껌 가게로 설명하죠. 이 가게는 장사가 꽤 잘 됩니다. 하지만 지금 이 가게를 사려면 1년에 버는 돈의 46~50배를 내야 합니다. 좋은 가게라도 그 값에 사면 좋은 투자가 안 됩니다. ROC는 준수한데 EY가 2% 근처라는 건, "좋지만 비싼 기업"이라는 뜻이고, 이건 제가 명시적으로 회피하는 유형입니다.

최근 실적 서프라이즈(+24.8%)에 시장이 환호하며 주가가 급등했다는 뉴스, 애널리스트 목표주가 평균 $385라는 이야기 — 다 흥미롭지만 제 공식에는 안 들어갑니다. 저는 스토리로 사지 않습니다. 숫자로 삽니다. 지금 HEI의 숫자는 "매수하라"고 말하지 않습니다.

그렇다고 "매도하라"도 아닙니다. 제 공식은 보유 중인 종목을 스토리로 파는 것도 금지합니다. 저는 이 종목을 애초에 제 20~30개 포트폴리오에 편입하지 않았을 뿐입니다. 지금 이 가격대라면 관망이 맞는 답입니다. EV/EBIT가 정상화되어 EY가 두 자릿수에 근접하기 전까지는요.
