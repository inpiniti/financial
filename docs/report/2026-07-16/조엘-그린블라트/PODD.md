# PODD 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 4/10

## 한 줄 결론
공식이 요구하는 두 숫자 중 하나(EY)가 낮게 나와서, 좋은 기업일 수는 있어도 "지금 이 가격에" 살 만큼 싸지는 않습니다.

## 내 철학에서 본 이 기업
껌 가게 소년 제이슨의 질문으로 돌아가 봅시다. "이 사업은 얼마나 돈을 잘 버는가?" 그리고 "지금 얼마에 살 수 있는가?" 이 두 질문 외에는 관심 없습니다. Omnipod 리콜 뉴스, 소송 보도, 애널리스트의 "Strong Buy" 컨센서스 — 전부 노이즈입니다. 저는 스토리를 사지 않고 숫자를 삽니다.

## 체크리스트 평가

| 항목 | 평가 | 근거 수치 |
|---|---|---|
| 자본수익률(ROC) 계산 가능성 | △ | 순고정자산(Net Fixed Assets) 데이터 미확보로 정식 ROC 산출 불가. 데이터 팩 8번 항목도 동일하게 "확인 불가" 명시 |
| 이익수익률(EY) | △ | EY ≈ 4.1% — 마법 공식 유니버스(대개 미국 대형주 상위 랭커는 8~15% 이상)에서 상위권이라 보기 어려움 |
| 규모 필터 | ○ | 시가총액 $11.03B, 필터 통과 |
| "좋으면서 싼가" | × | 좋을 가능성은 있으나(성장률 30%대, 매출총이익률 71.6%), 지금 가격은 그 좋음에 이미 상당 부분 값을 치른 상태 |
| 나쁜 뉴스로 인한 저평가 기회 여부 | △ | Class I 리콜로 52주 고점 대비 -55.1% 하락 — 공식이 좋아하는 "비관으로 싸진 멀쩡한 기업" 패턴과 유사하지만, EY 숫자 자체가 아직 저를 설득하지 못함 |

## 밸류에이션 판단 (계산 과정)

### 1) EBIT 추정
```
FY2025 영업이익률 17.5% × 매출 $2,708.1M = EBIT(TTM 근사) ≈ $473.9M
```
(분기 데이터 부재로 정밀 TTM 대신 FY2025 연간 실적을 사용. 26Q1 영업이익 $122.2M은 참고치로 병기)

### 2) EV 계산
```
EV = 시가총액 + 총부채 - 현금
   = $11,030M + $948.1M - $480.4M
   = $11,497.7M
```

### 3) 이익수익률(EY)
```
EY = EBIT ÷ EV = 473.9 / 11,497.7 = 4.12%
```
→ 국채 금리보다 낮은 수준. 마법 공식 관점에서 "싸다"고 말하기 어렵습니다. 인수자가 이 회사를 통째로 살 때, 투자원금 대비 매년 벌어들이는 세전영업이익이 4%대라는 뜻입니다. 이 가격에 회사 전체를 인수한다면 저라면 주저하겠습니다.

### 4) 자본수익률(ROC) — 근사치 (정식 산출 불가)
정식 공식: `ROC = EBIT ÷ (순운전자본 + 순고정자산)`

- 순운전자본(NWC, 26Q1) = 유동자산 $1,709.5M - 유동부채 $687.5M = $1,022.0M
- 순고정자산: **데이터 팩에 항목 없음 → 확인 불가**

부득이 대체 근사치(투하자본 = 자기자본 + 총부채 - 현금)를 참고용으로 계산:
```
투하자본(근사) = 1,302.6 + 948.1 - 480.4 = $1,770.3M
ROC(근사) = 473.9 / 1,770.3 ≈ 26.8%
```
이 근사치가 사실이라면 "좋은 기업"의 문턱은 넘습니다(마법 공식 상위권 기업들은 대개 ROC 25% 이상). 하지만 이건 제 공식이 아니라 제 공식을 흉내 낸 것에 불과합니다. 순고정자산 원자료 없이는 정직하게 "ROC 얼마"라고 말할 수 없습니다.

### 5) 순위 결합 판단
두 지표 중 하나(ROC)는 근사적으로 양호해 보이지만 정식 산출이 불가능하고, 다른 하나(EY 4.1%)는 명백히 낮습니다. 마법 공식은 "좋으면서 동시에 싼" 기업을 찾는 것이지, "좋아 보이는데 비싼" 기업을 찾는 게 아닙니다. Trailing PER 37.2x, EV/EBITDA 19.2x — 이 숫자들이 EY 계산과 정확히 같은 이야기를 하고 있습니다: 지금은 싸지 않습니다.

## 리스크
- 제 공식의 첫 번째 한계가 정확히 여기서 드러납니다: 순고정자산 데이터 부재로 ROC를 온전히 신뢰할 수 없습니다. 데이터 없이 판단하지 않는 것도 규율입니다.
- Class I 리콜로 인한 소송 리스크는 EBIT에 아직 완전히 반영되지 않았을 수 있습니다. 향후 분기 실적에 일회성 비용(소송충당금 등)이 반영되면 EBIT이 낮아지고 EY는 더 나빠질 수 있습니다.
- FY2025 순이익률이 20.2%→9.1%로 급감한 원인이 확인 불가 — 회계상 이익의 질을 완전히 신뢰하기 어렵습니다.

## 최종 의견
제이슨의 껌 가게로 돌아가 봅시다. 아무리 좋은 가게라도 장부가치의 10배를 부르면 저는 지갑을 닫습니다. PODD는 매출총이익률 71.6%, 성장률 30%대로 분명 "좋은 사업"의 냄새가 납니다. 문제는 시장도 그걸 알고 있고, 이미 상당한 프리미엄을 요구하고 있다는 겁니다. EY 4.1%는 제 기준으로 상위권 후보가 되기엔 부족합니다.

리콜 뉴스로 주가가 반토막 났다고 해서 무조건 사라는 게 제 공식이 아닙니다. 비관 때문에 가격이 "충분히" 싸졌는지를 숫자가 확인해줘야 합니다. 지금 이 숫자는 아직 "그렇다"고 말하지 않습니다. 순고정자산 데이터가 확보되어 정식 ROC가 계산되고, 주가가 더 하락해 EY가 8% 근처까지 올라온다면 그때 다시 계산기를 두드려 보겠습니다. 지금은 관망입니다 — 공식이 사라고 말하지 않는데 제가 왜 사겠습니까.
