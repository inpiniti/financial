# TGS 투자 보고서 — 조엘 그린블라트 (주식시장을 이기는 작은 책)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 4/10

## 한 줄 결론
마법 공식의 두 지표 중 하나(ROC)는 훌륭한 숫자가 나오지만, EY(이익수익률)를 구하는 데 필요한 EV(기업가치) 원자료가 통화 혼재 오류로 신뢰할 수 없다 — 공식이 요구하는 "숫자로 순위를 매기는" 절차 자체가 이 종목에서는 완전하게 성립하지 않는다. 껌 가게 소년 제이슨의 원칙대로, 계산이 안 되면 사지 않는다.

## 내 철학에서 본 이 기업
투자의 전부는 "좋은 기업을 싼 가격에 사는 것"이고, 그걸 숫자로 확인하는 두 개의 지표가 ROC와 EY다. TGS는 아르헨티나 규제 파이프라인 운송사로, 매출총이익률 53.9%(FY2025), 영업이익률 44.9%라는 인상적인 마진을 보여준다. 얼핏 "좋은 기업"의 신호다. 하지만 재무제표는 ARS, 주가·EV 관련 지표는 USD로 섞여 있고, 데이터 팩 스스로도 EV/EBITDA·PSR을 "통화 단위 혼재 오류로 사용 불가"라고 적어놨다. 나는 스토리를 믿지 않는다. 숫자를 믿는다. 그런데 그 숫자의 절반(EY)을 못 믿는 상황이다.

## 체크리스트 평가

| 항목 | 평가 | 근거 |
|---|---|---|
| ROC (자본수익률) 계산 가능한가 | ○ | EBIT·투하자본 모두 ARS로 일관되게 구할 수 있음 |
| ROC 수준이 높은가 | ○ | 아래 계산 결과 ROC ≈ 21.8%, 규제 인프라 기업치고 매우 높음 |
| EY (이익수익률) 계산 가능한가 | × | EV 원자료(Yahoo `enterpriseValue`)가 음수로 나오는 통화 혼재 오류, 데이터 팩이 "확인 불가" 명시 |
| 대체 EY 추정치 신뢰도 | △ | 직접 재구성 시도했으나 환율 역산에 의존한 근사치일 뿐, 공식이 요구하는 정밀도에 못 미침 |
| 규모 필터 통과 | ○ | 시총 $47억, 유동주식 3,134만주 — 마법 공식 유니버스에 들어갈 규모 |
| 두 지표 모두 상위권인가 | 확인 불가 | EY를 신뢰할 수 없으니 순위 합산 자체가 불가능 |

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

### 1) ROC (자본수익률) 계산 — ARS 기준

```
ROC = EBIT ÷ (순운전자본 + 순고정자산)
```

투입 유형자본을 데이터 팩에서 직접 구하기 어려워, 그린블라트 원 공식의 대체 근사치로 "순운전자본 + 자기자본" 대신 **(유동자산-유동부채) + (총자산 근사 = 자기자본+총부채 - 현금)** 방식을 시도했으나, 데이터 팩에 총자산·순고정자산 항목이 없다. 따라서 통상 실무에서 쓰는 근사식으로 대체한다:

```
근사 ROC = EBIT ÷ (순운전자본 + 유형자산 근사치)
```

데이터 팩에 유형자산(PP&E) 수치가 없어 완전한 근사도 불가능하다. 대신 자기자본+부채-현금(투하자본 근사, IC)으로 계산한다.

| 항목 (FY2025, ARS 백만) | 값 |
|---|---:|
| EBIT (영업이익) | 766,932 |
| 자기자본 | 3,127,863 |
| 총부채 | 1,705,606 |
| 현금성자산 | 804,107 |
| 투하자본(IC) = 자기자본+총부채-현금 | 4,029,362 |

```
ROC(근사) = 766,932 ÷ 4,029,362 = 19.0%
```

순운전자본만 따로 보면 (유동자산-유동부채) = 2,236,945 - 447,234 = 1,789,711. 이걸 그대로 IC로 쓰면:

```
ROC(순운전자본만 기준) = 766,932 ÷ 1,789,711 = 42.9% (과대 추정, 고정자산 미포함 오류)
```

두 근사치의 폭(19.0%~42.9%)을 볼 때, **보수적으로 19~20%대**가 실제 ROC에 가깝다고 판단한다. 이는 규제 인프라 사업치고 상당히 우수한 자본효율이다. 다만 이 수치는 순고정자산(PP&E) 원자료 부재로 인한 근사치이며, 정밀한 마법 공식 계산에는 부족하다.

### 2) EY (이익수익률) 계산 — USD 기준, EV 재구성 시도

원 공식: `EY = EBIT ÷ EV(=시가총액+부채-현금)`. Yahoo 제공 EV는 ARS·USD 혼재로 음수(-2,300억)가 나와 사용 불가하다. 직접 재구성을 시도한다.

환율 역산: 1Q26 매출 Ps.484,201M = $350.2M → 환율 ≈ 1,383 ARS/USD (2026년 초 기준).

| 항목 | ARS 백만 | USD 환산(÷1,383) |
|---|---:|---:|
| FY2025 EBIT | 766,932 | $554.5M |
| FY2025 총부채 | 1,705,606 | $1,233.3M |
| FY2025 현금 | 804,107 | $581.6M |
| 시가총액(실측 USD) | — | $4,707.8M |

```
EV(재구성) = 시가총액 + 부채 - 현금 = 4,707.8 + 1,233.3 - 581.6 = $5,359.5M

EY = EBIT ÷ EV = 554.5 ÷ 5,359.5 = 10.3%
```

**주의**: 이 환율은 1Q26 시점 매출로 역산한 단일 시점 근사치이며, 초인플레이션 국가 통화라 연중 변동성이 매우 크다(FY2025 EBIT는 1년 평균 환율을 써야 정확한데, 여기선 최근 환율을 그대로 적용했다). 데이터 팩이 EV/EBITDA를 "확인 불가"로 명시적으로 배제한 이유가 바로 이 통화 문제이며, 나의 재구성치도 같은 리스크를 안고 있다.

### 3) 마법 공식 순위 결합

이 두 지표(ROC ≈ 19~20%, EY(근사) ≈ 10.3%)만 놓고 보면:
- ROC 19~20%: 준수한 수준. S&P500 평균 ROC(대략 15~20%대)와 비슷하거나 약간 상회.
- EY 10.3%(근사): PER 약 9.7배에 해당하는 수준으로 나쁘지 않으나, 데이터 팩의 실측 Trailing PER 15.87x, Forward PER 11.31x와 비교하면 상당한 괴리가 있다(EBIT 기준이라 순이익 기준 PER보다 높게 나오는 게 정상인데, 오히려 낮게 나온 것은 환율 근사 오차 가능성을 시사).

**결론**: ROC는 신뢰할 만한 근사치로 "좋은 기업" 신호를 보내지만, EY는 원자료 오류로 정밀 계산이 불가능하다. 마법 공식은 두 지표의 "정확한 순위 결합"이 생명인데, 근사치 하나로는 유니버스 내 순위를 매길 수 없다. 나는 확신 없는 숫자로 순위를 매기지 않는다.

## 리스크
- **데이터 리스크(가장 큼)**: EV/EBITDA, PEG, PSR, 이자보상배율, ROIC 원자료 모두 "확인 불가" — 마법 공식 계산의 절반(EY)이 근본적으로 흔들린다.
- **거시 리스크**: 아르헨티나 페소 환율·인플레이션·ENARGAS 요금 규제 지연 — 초인플레이션 회계(IAS 29) 특성상 명목가 자체가 왜곡될 수 있다.
- **1Q26 실적 미스**: EPS $0.75 vs 컨센서스 $0.88(-14.9%) — 최근 4개 분기 어닝 서프라이즈가 -40.7%, +44.3%, +30.2%, -14.9%로 극도로 혼조. 이런 변동성은 EBIT의 "정상화 수준"을 판단하기 어렵게 만든다.
- **개별 종목 리스크**: 마법 공식은 애초에 개별 종목이 아니라 20~30개 분산 포트폴리오로 평가하는 전략이다. TGS 하나만 떼어서 "매수하라 말라" 판단하는 것 자체가 내 원래 방법론의 취지에서 벗어난다.

## 최종 의견
숫자로 말하겠습니다. ROC는 대략 19~20%로 꽤 훌륭합니다 — 파이프라인 하나 깔아놓고 규제 요금 받아가는 사업치고는 자본을 효율적으로 굴리고 있다는 뜻이죠. 문제는 EY입니다. 이 회사 EV를 제대로 구하려고 했더니, 재무제표는 아르헨티나 페소, 주가는 미국 달러로 표시돼 있고, Yahoo가 자동 계산한 EV/EBITDA는 마이너스로 나옵니다. 마이너스 기업가치라니, 이 회사를 사면 오히려 돈을 받는다는 뜻인가요? 당연히 오류입니다. 제가 환율을 역산해서 EY를 대략 10.3%로 추정해봤지만, 이건 어디까지나 근사치입니다. 저는 좋은 기업을 싸게 사라고 말하지만, "싸다"는 것도 정확한 숫자로 확인해야 합니다. 근사치로 대충 순위를 매기는 건 제 원칙이 아닙니다. 게다가 마법 공식은 애초에 한 종목을 콕 집어 사고파는 전략이 아니라 20~30개를 분산해서 기계적으로 반복하는 전략입니다. TGS 하나만 놓고 "사라 말라" 하는 것 자체가 제 방법론을 오용하는 겁니다. 데이터가 정리되기 전까지는 관망입니다. 공식이 부진하다고 포기하지 말라고 늘 말하지만, 애초에 공식을 돌릴 수 없는 데이터로 억지로 결론을 내는 것도 규율이 아닙니다.
