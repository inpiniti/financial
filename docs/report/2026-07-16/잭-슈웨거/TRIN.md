# TRIN 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-16 | 투자의견: 보유(관망에 가까운 소극적 보유) | 확신도: 5/10

## 한 줄 결론
추세와 펀더멘털은 같은 방향을 가리키지만 손익비가 1.34:1로 얇고 52주 밴드 상위 91% 지점에서 추격하는 모양새라, 신규 진입은 눌림목까지 기다리고 기존 보유자는 손절선을 명확히 그어놓고 배당 캐리를 태우는 것이 합리적이다.

## 내 철학에서 본 이 기업
제가 인터뷰한 트레이더들이라면 TRIN을 두고 이렇게 물었을 겁니다. "이 거래가 틀렸을 때 어디서 빠져나올 것인가?" TRIN은 BDC라는 업종 특성상 배당수익률 14.44%가 매력적이지만, 이것이 손절 규칙을 느슨하게 만들 이유는 되지 않습니다. 폴 튜더 존스라면 "얼마를 벌 수 있는가"보다 "얼마를 잃을 수 있는가"부터 물었을 것이고, 답은 명확합니다 — 오닐의 7.5% 규칙을 적용하면 주당 $1.33, 계좌 1% 한도 기준 751주가 상한선입니다.

추세는 확인됩니다. 2025년 11월 저점 $14.60 이후 완만하게 우상향해 현재 $17.72까지 왔고, 2026Q1 실적(NII YoY +37.4%)이 펀더멘털을, NYSE 이전과 신규 커밋 확대가 모멘텀을 뒷받침합니다. 마이클 마커스라면 펀더멘털과 기술적 신호가 같은 방향을 가리킨다는 점에서 어느 정도 확신을 가졌을 겁니다. 다만 52주 밴드 상위 91% 지점, 사상 최고가($20.26) 대비 아직 갭이 있다는 것은 "신고가를 사서 더 오르길 기다리는" 추격 매매의 함정일 수도 있습니다.

## 밸류에이션 판단 (계산 과정)

### 진입·손절·목표·손익비

| 항목 | 값 | 산출식 |
|---|---|---|
| 진입가 | $17.72 | 현재가(2026-07-15 종가) |
| 손절가 | $16.39 | 17.72 × (1 − 0.075) = 16.39 (오닐 7~8% 규칙) |
| 1차 목표 | $18.25 | 애널리스트 목표주가 중앙값 (상승여력 +3.0%) |
| 실전 목표 | $19.50 | 애널리스트 최고치($19.00)~사상최고가($20.26) 중간 근사 (상승여력 +10.0%) |
| 2차(스트레치) 목표 | $20.26 | 사상 최고가 (상승여력 +14.3%) |

**손익비 계산**
```
리스크(주당) = 17.72 − 16.39 = 1.33  (−7.5%)
리워드(주당) = 19.50 − 17.72 = 1.78  (+10.0%)
손익비 = 1.78 / 1.33 = 1.34 : 1
```
데니스 기준(맞았을 때 크게, 틀렸을 때 작게)으로 보면 1.34:1은 합격선(2:1 이상)에 못 미친다. 다만 배당수익률 14.44%가 보유 중 캐리로 더해지므로 총기대수익(가격+배당) 관점에서는 다소 개선되나, 손절 판단 자체를 완화할 근거는 아니다.

### 포지션 사이징 (하이트의 1% 규칙, 가정 계좌 $100,000)

```
최대 손실 허용액 = 100,000 × 0.01 = $1,000
주당 리스크 = 17.72 − 16.39 = $1.33
매수 가능 주식수 = 1,000 / 1.33 = 751주
포지션 규모 = 751 × 17.72 = $13,308  (계좌의 약 13.3%)
```
참고로 2% 규칙 적용 시 1,503주, $26,633(26.6%)로 단일 종목 편중이 과도해지므로 1% 규칙을 권고한다.

## 리스크
- 밴드 상위 91% 지점 진입은 추격매수 성격 — 조정 시 되돌림 폭이 클 수 있음
- NYSE 이전(2026-07-24/27) 전후 단기 변동성 확대 가능성 — 결과 방향은 확인 불가
- Payout Ratio 116.9%로 100% 초과 — 원본소득 대부분 배당이라는 BDC 특성이나, 이익 성장 둔화 시 배당 지속성 리스크
- PEG 5.14로 성장 대비 밸류에이션 부담 존재
- BDC 레버리지(D/E 116.4%) — 금리·신용 사이클 악화 시 포트폴리오 부실 위험

## 최종 의견
제가 인터뷰한 위저드 중 누구도 "손절 지점 없이 진입하라"고 말한 사람은 없습니다. TRIN은 추세와 펀더멘털이 같은 방향을 가리키는, 나쁘지 않은 그림입니다. 하지만 손익비가 1.34:1에 그치고 52주 밴드 최상단 근처에서 사는 셈이라, 이 자리에서 신규로 크게 베팅할 자리는 아닙니다. 기존 보유자라면 $16.39를 손절선으로 못박고 배당 캐리를 태우며 $19.50까지 홀드하고, 신규 진입을 고려한다면 조정 시 $16.5~17 부근 눌림목을 기다리는 것이 손익비를 개선하는 길입니다. 희망은 전략이 아닙니다 — "더 오를 것 같다"는 느낌으로 신고가 근처를 추격하지 마십시오.
