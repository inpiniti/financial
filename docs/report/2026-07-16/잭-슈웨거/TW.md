# TW 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-16 | 투자의견: 관망 (조건부 소액 진입 시나리오 병기) | 확신도: 4/10

## 한 줄 결론
펀더멘털은 견조하지만(1Q26 매출 +21% YoY) 주가는 아직 하락추세 안에 있다 — "싸다"는 이유로 하락 중인 종목을 잡지 말라는 것이 내가 인터뷰한 모든 위저드의 공통 경고다. 6월 저점($94~96)에서 반등이 시작된 건 사실이나, 추세 전환이 확인된 것은 아니다. 진입한다면 손절 폭이 좁은 지점, 즉 최근 스윙 저점 바로 아래에서만 해야 한다.

## 내 철학에서 본 이 기업
나는 기업의 비즈니스 모델을 평가하는 사람이 아니다. 내가 인터뷰한 마이클 마커스는 항상 "펀더멘털과 기술적 신호가 같은 방향을 가리킬 때 가장 큰 확신을 가졌다"고 말했다. TW의 경우 이 둘이 어긋나 있다:

- **펀더멘털**: 매출 FY2022→FY2025 1,188.8M→2,052.4M(연평균 +20%), 1Q26 매출 +21% YoY, 순이익률 39.6%, D/E 2.1%로 재무 매우 건전. 상승 신호.
- **기술적**: 52주 고점 대비 -32%, 5년 중앙값 PER 54.7x → 현재 24.3x로 밸류에이션 디레이팅이 진행 중이며 최근 다수 애널리스트(Morgan Stanley, Barclays)가 목표가를 하향. 하락 추세 신호.

윌리엄 오닐의 CAN SLIM 중 M(시장 방향)과 개별 추세가 정렬되지 않은 상태에서, "펀더멘털이 좋으니 언젠가 돌아올 것"이라는 생각으로 물타기하는 것이 바로 내가 가장 경계하는 패턴이다. 희망은 전략이 아니다.

## 밸류에이션 판단 — 나는 적정가를 계산하지 않는다

나는 애널리스트나 밸류에이션 전문가가 아니다. 목표주가는 다른 15명의 애널리스트가 이미 계산했다(평균 $129.86, 중앙값 $122.50, 최저 $105, 최고 $201). 나에게 중요한 것은 "얼마가 적정한가"가 아니라 "이 거래의 손익비가 얼마인가"다. 대신 진입가·손절가·목표가·손익비를 계산한다.

### 시나리오 A: 지금 즉시 진입 (비추천, 참고용)

| 항목 | 값 | 근거 |
|---|---:|---|
| 진입가 | $100.68 | 현재가 |
| 손절가 | $93.50 | 6월 저점($94.25) 하회 확정 + 약간의 버퍼 |
| 손절 폭 | -7.1% ($7.18) | |
| 1차 목표가 | $114.85 | 애널리스트 최저 목표가 $105와 중앙값 $122.50의 중간값 근사 |
| 목표 상승폭 | +14.1% ($14.17) | |
| **손익비** | **약 1.97 : 1** | 14.17 / 7.18 |

손익비 자체는 나쁘지 않다(2:1 근접). 하지만 이 시나리오의 문제는 **추세 확인이 안 된 상태에서 진입**한다는 것 — 데니스라면 "틀렸을 때 빨리 손절"은 가능해도, "맞을 확률"에 대한 근거(추세)가 없다고 지적할 것이다.

### 시나리오 B: 추세 전환 확인 후 진입 (권장)

| 항목 | 값 | 근거 |
|---|---:|---|
| 트리거 조건 | 종가 기준 $105 돌파 + 거래량 동반 | 애널리스트 최저 목표가($105) 상회 = 하락추세 저항선 돌파 신호로 활용 |
| 진입가(가정) | $105.50 | 돌파 확인 후 소폭 프리미엄 |
| 손절가 | $99.50 | 직전 스윙 저점(반등 시작점 $97~100 구간) 하회 |
| 손절 폭 | -5.7% ($6.00) | |
| 1차 목표가 | $122.50 | 애널리스트 중앙값 |
| 목표 상승폭 | +16.1% ($17.00) | |
| **손익비** | **약 2.83 : 1** | 17.00 / 6.00 |
| 2차 목표가(추세 지속 시) | $138 | 직전 애널리스트 상향 전 목표가(Morgan Stanley 구 목표가) 부근, 트레일링 스탑으로 관리 |

이 시나리오의 손익비(2.83:1)가 A안(1.97:1)보다 명확히 유리하다. 오닐 원칙대로 "신고가·돌파 부근에서 매수"하는 것이 하락 중인 종목을 미리 잡는 것보다 승률과 손익비 모두에서 우월하다.

### 포지션 사이징 (래리 하이트 1% 룰 적용)

가정: 계좌 자본 $100,000 기준(예시, 실제 계좌 규모에 비례 조정)

- 단일 거래 최대 손실 허용액 = 자본의 1% = **$1,000**
- 시나리오 B 기준 손절 폭 = 진입가 $105.50 − 손절가 $99.50 = $6.00/주
- **포지션 크기 = $1,000 / $6.00 = 약 166주**
- 투입 자본 = 166주 × $105.50 = 약 $17,513 (계좌의 약 17.5%)
- 최대 손실 발생 시: 166주 × $6.00 = $996 ≈ 자본의 1.0% (룰 충족)

시나리오 A로 즉시 진입할 경우: 손절 폭 $7.18/주 → 포지션 크기 = $1,000/$7.18 ≈ 139주, 투입 자본 약 $13,995(계좌의 14%). 손절 폭이 넓을수록 포지션은 작아진다 — 이것이 손익비와 별개로 사이징이 항상 리스크(1%)를 우선한다는 원칙이다.

## 리스크

- **추세 미확인 리스크**: 현재 반등이 진짜 바닥 형성인지, 하락추세 중 데드캣 바운스인지 구분이 안 된다. 최근 1개월 -11.4% 급락 직후의 반등이라는 점이 특히 경계 대상.
- **애널리스트 목표가 하향 러시**: Morgan Stanley($138→$122), Barclays($118→$114) 등 최근 목표가가 잇달아 하향 — 기관 컨센서스가 약세로 기울고 있다는 신호. 이는 "펀더멘털은 좋은데 시장 심리가 나쁜" 전형적 불일치 국면.
- **신용시장 점유율 정체**: Raymond James가 "Outperform 유지"하면서도 신용시장 점유율 확대 정체를 우려로 명시 — 성장 스토리의 핵심 축이 흔들릴 가능성.
- **실적 발표 리스크(2026-07-30)**: 2Q26 컨센서스 EPS $0.95, 매출 $560.5M. 실적 발표 전 진입 시 갭 리스크(손절선을 무력화하는 갭다운) 존재 — 실적 발표를 관통하는 포지션은 원칙적으로 회피하거나 사이즈를 절반으로 줄여야 한다.
- **베타 0.62로 개별 촉매 의존도 높음**: 시장 전체 방향과 무관하게 개별 뉴스(목표가 조정, 실적)에 더 민감하게 반응할 수 있음.
- **거시 환경 확인 불가**: 금리 변동성, 채권시장 거래량 등 TW 비즈니스 핵심 동인에 대한 최신 데이터가 팩에 없어 "시장 전체 방향" 체크리스트 항목을 완전히 충족할 수 없음.

## 최종 의견

내가 인터뷰한 위저드들에게 이 차트를 보여준다면 다들 같은 질문을 던질 겁니다. "지금 이 매수는 추세를 따르는 겁니까, 아니면 싸 보여서 사고 싶은 겁니까?" 매출이 21% 성장하고 이익률이 40%에 육박하는 좋은 회사라는 건 압니다. 하지만 마이클 마커스는 좋은 회사와 좋은 거래를 구분하라고 했습니다. 지금 TW는 52주 고점에서 32% 빠진 채 6월 저점을 딛고 반등하는 중이지만, 애널리스트들은 여전히 목표가를 낮추고 있습니다. 펀더멘털과 기술적 신호가 어긋나 있는 이 상태에서는, 확신을 가지고 들어갈 이유가 없습니다.

지금 당장 들어가고 싶다면 손절가 $93.50을 절대 타협하지 말고, 포지션은 계좌의 1% 리스크 룰 안에 가두세요. 하지만 내 조언은 기다리는 것도 하나의 포지션이라는 겁니다. $105 돌파를 거래량과 함께 확인한 뒤 들어가면 손익비가 1.97:1에서 2.83:1로 개선됩니다. 서두를 이유가 없습니다. 시장은 언제나 다음 기회를 줍니다 — 지금 이 종목을 놓친다고 해서 손해 볼 건 없습니다. 하지만 추세 없이 들어가서 물타기하게 되면, 그건 손해로 끝납니다.
