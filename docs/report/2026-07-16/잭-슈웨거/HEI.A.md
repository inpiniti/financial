# HEI.A 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-16 | 투자의견: 매수(추세추종, 손절 규칙 전제) | 확신도: 6/10

## 한 줄 결론
추세와 펀더멘털이 같은 방향(마커스 조건)을 가리키고 있지만, 밸류에이션이 이미 비싸고 클래스A 데이터 신뢰도가 낮다 — "얼마를 벌 수 있는가"보다 "어디서 틀렸다고 인정할 것인가"를 먼저 정하고 들어가야 하는 자리다.

## 내 철학에서 본 이 기업
제가 인터뷰한 위저드들이라면 이 종목을 두고 이렇게 물었을 겁니다. "추세는 확인됐는가?" — 현재가 $253.20이 50일선($240.26)과 200일선($241.54) 모두 위에 있고, 두 이평선이 근접해 골든크로스 근처에서 정배열로 가는 구간이니 답은 예스입니다. "펀더멘털도 같은 방향을 가리키는가?" — FQ2 FY26 EPS $1.66이 컨센서스 $1.33을 24.8% 상회했고, 매출·순이익이 각각 YoY +25%, +49% 증가했습니다. 마이클 마커스라면 이 정렬을 보고 "가장 확신을 가질 수 있는 조건"이라 했을 겁니다.

하지만 PER(trailing) 44.7x, EV/EBITDA 27.9x는 싼 종목이 아닙니다. 저는 밸류에이션 전문가가 아니지만, 위저드들의 공통 원칙은 "비싸도 추세가 있으면 올라타되, 손절 규칙을 반드시 세운다"는 것입니다. 밸류에이션은 진입을 막는 이유가 아니라 손절을 더 엄격하게 지켜야 하는 이유입니다.

## 밸류에이션 판단 (계산 과정)

저는 내재가치를 계산하지 않습니다. 대신 진입·손절·목표·사이징·손익비를 계산합니다.

**1) 진입가/손절가/목표가**

| 항목 | 값 | 산출 근거 |
|---|---|---|
| 진입가 | $253.20 | 현재가(2026-07-16 종가) |
| 손절가 | $240.00 | 200일 이동평균($241.54) 하회 확정 종가 기준. 참고: 오닐 7~8% 룰 적용 시 $234.2~$233.0 |
| 손절 폭 | -5.21% ($13.20) | (253.20-240.00)/253.20 |
| 1차 목표가 | $279.66 | 직전 52주 고점(저항선) |
| 2차 목표가 | $320.00 | HEI.A 표시 목표주가(분석대상 1인, 신뢰도 낮음 — 참고용) |

**2) 손익비**

- 1차 목표 기준: (279.66 − 253.20) / (253.20 − 240.00) = 26.46 / 13.20 ≈ **2.00R**
- 2차 목표 기준: (320.00 − 253.20) / 13.20 = 66.80 / 13.20 ≈ **5.06R**
- 데니스의 원칙("틀렸을 때 빨리 손절, 맞았을 때 오래 보유")대로면 1차 목표 도달 시 절반 청산 후 잔여 물량은 트레일링 스탑(예: 20일 저가 이탈)으로 2차 목표까지 끌고 가는 방식을 권장.

**3) 포지션 사이징 (래리 하이트 1% 룰, 계좌 $100,000 가정)**

```
허용 최대 손실 = 계좌자산 × 1% = $100,000 × 0.01 = $1,000
주당 리스크    = 진입가 - 손절가 = $253.20 - $240.00 = $13.20
매수 가능 수량 = 허용 최대 손실 / 주당 리스크 = $1,000 / $13.20 ≈ 75주
포지션 금액    = 75주 × $253.20 ≈ $18,990 (계좌 대비 약 19.0%)
```

계좌 규모에 관계없이 이 비율(리스크 1%, 종목 비중 약 19%)로 스케일 가능. 다만 단일 종목 비중 19%는 위저드들의 일반적 분산 원칙(하이트) 상 상한에 가까우므로, 1차 50%·눌림 시 2차 50% 분할 진입을 권장.

## 리스크

- **밸류에이션 리스크**: PER 44.7x, EV/EBITDA 27.9x — 성장 둔화 신호 시 배수 압축 폭이 클 수 있음. 손절 규칙이 이를 방어하는 유일한 장치.
- **데이터 신뢰도 리스크**: HEI.A는 애널리스트 커버리지가 사실상 없고(recommendationTrend 1건), 목표주가($320)의 신뢰도가 낮음. 보통주(HEI) 기준 UBS $390 목표와 클래스 간 프리미엄 구조 확인 필요 — 확인 불가 항목.
- **밸류에이션 급등 이력**: HEI 보통주가 2026-07-06 사상최고가($365.31)를 찍은 직후라 단기 과열 가능성. 추격매수보다 눌림목 재진입이 유리.
- **거시/섹터 리스크**: 시장 전체 방향(체크리스트 M) 데이터 확인 불가 — 항공우주/방위 섹터 전반의 방향성을 별도 확인 없이는 확신도를 더 높이기 어려움.
- **회계연도 특성**: 차기 실적 발표 2026-08-25(FQ3 FY26) 예정 — 발표 전후 변동성 확대 가능, 실적 발표 직전 신규 진입은 이벤트 리스크 가중.

## 최종 의견

여러분이 저에게 이 종목을 가져왔다면, 저는 목표주가나 PER 배수 얘기를 하기 전에 이렇게 물었을 겁니다. "이 거래가 틀렸을 때 어디서 빠져나올 겁니까?" 이 보고서에서는 그 답을 $240.00으로 정했습니다. 추세와 펀더멘털이 같은 방향을 가리키는 지금, 진입 자체를 막을 이유는 없습니다. 하지만 손익비 2.0R(1차 목표 기준)은 특출나게 매력적인 수준은 아니며, "훌륭한 회사"라는 이유로 손절 없이 물타기를 하는 순간 이 거래는 실패로 끝납니다. 희망은 전략이 아닙니다. 손절가에 도달하면 예외 없이 나가고, 목표가에 도달하면 절반은 챙기고 나머지는 추세가 꺾일 때까지 태우십시오. 그것이 제가 만난 모든 위저드가 반복해서 말한 유일한 공통 규칙입니다.
