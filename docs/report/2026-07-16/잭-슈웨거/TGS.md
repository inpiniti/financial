# TGS 투자 보고서 — 잭 슈웨거 (마켓 위저드)
> 날짜: 2026-07-16 | 투자의견: 보유(신규 진입은 조건부 관망) | 확신도: 5/10

## 한 줄 결론
추세는 살아있지만 최근 실적 미스로 펀더멘털·기술적 신호가 완전히 정렬되지 않았다. 손절선과 손익비를 미리 확정한 상태에서만, 그것도 축소된 포지션으로만 접근할 자리다.

## 내 철학에서 본 이 기업
제가 인터뷰한 트레이더들이라면 TGS를 보고 먼저 이렇게 물었을 겁니다 — "이 거래가 틀렸을 때 어디서 나올 것인가?" TGS는 아르헨티나 규제 파이프라인 사업에 액체(NGL)·미드스트림 비규제 사업이 빠르게 성장하며(1Q26 영업이익 +55%/+57%) 스토리는 좋습니다. S&P의 신용등급 상향(B-→B, 06/11), RIGI 승인, $3B NGL 프로젝트 상업계약까지 촉매가 겹쳤고, 52주 저점 $19.74에서 고점 $36.35까지 +84% 상승한 뒤 현재 $31.27로 조정 중입니다. 마이클 마커스라면 "펀더멘털과 기술적 신호가 같은 방향을 가리킬 때 확신이 커진다"고 했겠지만, 1Q26 EPS $0.75가 컨센서스 $0.88 대비 -14.9% 미스했고 최근 4개 분기 서프라이즈가 -40.7%/+44.3%/+30.2%/-14.9%로 들쭉날쭉합니다. 신호가 완전히 정렬되지 않았다는 뜻입니다. 게다가 아르헨티나 페소·ENARGAS 규제·정치 리스크는 정량화가 불가능한 영역 — 폴 튜더 존스라면 "얼마를 잃을 수 있는가"부터 물었을 겁니다.

## 밸류에이션 판단 (계산 과정)

**추세·촉매**
- 52주 밴드: 저점 $19.74 ~ 고점 $36.35, 현재 $31.27 (밴드 내 약 69% 지점)
- 촉매: S&P 등급 상향(06/11), RIGI 승인($550M Perito Moreno 확장), $3B NGL 프로젝트(06월 계약)
- 역풍: 1Q26 EPS 컨센서스 미스(-14.9%), 최근 4분기 서프라이즈 혼조

**진입가**
- 기준 진입가: $31.50 (현재가 부근, 반등 확인 시)
- 대안 진입가(눌림목): $29.00 (조정 저점 지지 확인 시)

**손절가 (오닐 7~8% 규칙)**
```
손절가 = $31.50 × (1 - 0.075) = $29.14 → $29.10
손절 폭 = $31.50 - $29.10 = $2.40 (-7.6%)
```

**목표가**
- 1차 목표(52주 고점 재도전): $36.35
- 2차 목표(컨센서스 중앙값): $41.00
- 3차 목표(BofA 상향치): $46.00

**손익비**
```
1차 목표 기준: ($36.35 - $31.50) / ($31.50 - $29.10) = $4.85 / $2.40 = 2.02 : 1
2차 목표 기준: ($41.00 - $31.50) / $2.40 = $9.50 / $2.40 = 3.96 : 1
```
→ 데니스 원칙에 따라 1차 목표 도달 시 손절을 진입가로 트레일링, 잔여 물량은 2차 목표까지 보유.

**포지션 사이징 (하이트 방식, 단일 거래 리스크 1% 상한)**
```
가정 계좌: $100,000
리스크 한도(1%): $1,000
주당 손실폭: $2.40
매수 수량 = $1,000 / $2.40 = 약 416 ADS
포지션 금액 = 416주 × $31.50 = 약 $13,104 (계좌의 13.1%)
```
아르헨티나 페소·규제 리스크(정량화 불가)를 감안, 실제 상한은 0.5~1%로 보수화 권장. 0.5% 적용 시 약 208주, 포지션 금액 약 $6,552(계좌의 6.6%).
유동성 체크: 일평균 거래량 340,217주 대비 416주는 미미 — 슬리피지 문제 없음.

- 아르헨티나 페소 환율·인플레이션·ENARGAS 요금 규제 지연 — 정량화 불가, 상시 거시 리스크
- 최근 4분기 EPS 서프라이즈 혼조(-40.7%~+44.3%) — 실적 예측가능성 낮음, 다음 실적(8/3 혹은 8/12, 소스 불일치)까지 불확실성 지속
- 기관 지분율 14.31%로 낮음 — 수급 기반이 약해 조정폭이 커질 수 있음
- EV/EBITDA, PSR, 이자보상배율 등 핵심 밸류에이션 지표 다수가 통화 혼재 오류로 "확인 불가" — 밸류에이션 크로스체크 제한적

## 최종 의견
제가 인터뷰한 위저드들의 목소리를 빌리면 이렇습니다. 마이클 마커스는 "포지션이 불리하게 움직이면 빠르게 인정하고 빠져나와야 한다. 희망은 전략이 아니다"라고 했고, 이 원칙은 TGS에도 예외 없이 적용됩니다. 지금 이 종목은 스토리와 촉매는 훌륭하지만 최근 실적 미스로 신호가 완전히 정렬되지 않았습니다. 신규 진입을 하려는 사람에게는 이렇게 말하겠습니다 — 손절가 $29.10을 진입 전에 확정하고, 포지션 크기를 계좌의 1% 리스크 이내(약 416주 또는 그 절반)로 제한하고, 52주 고점 $36.35 돌파를 추세 재개의 확인 신호로 삼으십시오. 이미 보유 중이라면, 급등 이후의 자만을 경계하며 손절선을 지키고 1차 목표에서 물량 일부를 정리해 이익을 실현하는 것을 권합니다. "'얼마를 벌 수 있는가'보다 '얼마를 잃을 수 있는가'를 먼저 생각한다"는 폴 튜더 존스의 원칙, 아르헨티나 리스크가 있는 이 종목에는 특히 절실합니다.
