## 한 줄 결론
추세와 펀더멘털은 같은 방향을 가리키지만, 이미 52주 고점 근처에서 손익비가 2:1을 넘지 못한다 — 오닐이라면 "지금은 아니다"라고 말할 자리다.

## 내 철학에서 본 이 기업
제가 인터뷰한 위저드들은 하나같이 "얼마를 벌 수 있는가"보다 "얼마를 잃을 수 있는가"를 먼저 물었습니다. PSTL은 좋은 이야기를 가진 종목입니다 — USPS 임대라는 니치 독점, 1Q2026 EPS 서프라이즈 +9.6%, AFFO 가이던스 상향. 마커스식으로 말하면 펀더멘털과 기술적 신호(52주 저점 대비 +76% 상승 추세)가 같은 방향입니다. 하지만 지금은 52주 고점 대비 -5.2% 지점, 즉 이미 크게 달린 뒤의 되돌림 구간입니다. 여기서 진입하면 손절 폭 대비 목표 여유가 좁습니다.

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

**추세/진입 구조**
- 현재가: $23.91 / 최근 1개월 박스권: $22.54~$24.98 / 52주 고점: $25.22

**손절가 산출 (오닐식 -7~8% 룰)**
```
손절가 = $23.91 × (1 - 0.08) = $22.00 (박스권 하단 근접)
손절 폭 = $23.91 - $22.00 = $1.91 (-8.0%)
```

**목표가**
- 1차 목표(52주 고점 재도전): $25.22
- 2차 목표(애널리스트 최고가): $27.00

**손익비 계산**
```
1차: 보상 $25.22-$23.91=$1.31 / 리스크 $1.91 = 0.69 : 1  → 데니스 기준(승률<손익비) 미달
2차: 보상 $27.00-$23.91=$3.09 / 리스크 $1.91 = 1.62 : 1  → 여전히 2:1 미달
```

**포지션 사이징 (하이트식 단일거래 손실 한도 1%)**
```
가정 자본 = $100,000
리스크 한도(1%) = $1,000
주당 리스크 = $1.91
매수 가능 수량 = $1,000 / $1.91 ≈ 523주
포지션 규모 = 523주 × $23.91 ≈ $12,504 (자본의 12.5%)
```
→ 사이징 자체는 리스크 관리 원칙에 부합하지만, 애초에 손익비가 기준 미달이라 이 사이징으로도 진입 매력이 낮음.

**진입가 재설계 대안**
- 손익비 2:1을 만족하려면 진입가를 낮춰야 함: 목표 $25.22 고정 시, 리스크를 $0.65 이하로 줄여야 함 → 진입가 약 $22.65(박스권 하단 근접) + 손절 $22.00 조합이 손익비 2:1에 가까움 ($1.31/$0.65×2 근사)
- 즉, 현재가가 아니라 **박스권 하단($22.5~$23.0) 재테스트 시 진입**이 손익비상 합리적.

## 리스크
- 52주 고점 근접권 진입 시 되돌림 리스크(추격매수 함정)
- 배당성향 GAAP EPS 기준 187% — AFFO 기준으로는 정상(약 69~70%)이나 GAAP 순이익 자체는 여전히 얇음
- 애널리스트 커버리지 3명뿐 — 컨센서스 신뢰도 낮음
- EPS 서프라이즈 폭 4분기 연속 축소 — 모멘텀 둔화 신호일 수 있음
- 유동비율 데이터 소스 간 상충(0.21x vs 1.976x) — 재무 안전성 판단 불확실성 존재
- 부채/자본 1.27x로 매년 상승 추세 — ATM 지분 발행으로 조달 중, 희석 리스크

## 최종 의견
손절 지점도 정해지고, 추세도 확인되고, 펀더멘털도 뒷받침됩니다. 하지만 지금 이 가격에서 사면 손익비가 1:1도 안 됩니다. 제가 인터뷰한 위저드들은 좋은 종목이라고 아무 가격에서나 사지 않았습니다. 오닐이라면 "베이스를 다시 만들 때까지 기다려라"라고 했을 것이고, 데니스라면 "손익비가 안 나오는 거래는 애초에 거래가 아니다"라고 했을 겁니다. 지금은 관망하고, 박스권 하단($22.5 부근) 재테스트나 신규 브레이크아웃(신고가 $25.22 상향 돌파 후 눌림목)을 기다리는 것이 규율입니다. 기다리는 것도 유효한 포지션입니다.
