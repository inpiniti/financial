# WPM 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 5/10

## 한 줄 결론
펀더멘털은 마커스가 좋아할 만큼 강하지만, 추세는 데니스와 오닐이 싫어할 만큼 꺾여 있다 — 지금은 "기다리는 것도 포지션"인 국면이다. 다만 명확한 트리거가 잡히면 손익비 좋은 진입이 가능한 종목이다.

## 내 철학에서 본 이 기업
저는 인터뷰한 16명의 위저드 중 누구를 데려와도 이 종목에 지금 당장 들어가라고는 하지 않을 겁니다. 이유는 회사가 나쁘다는 게 아니라 — 타이밍이 안 맞다는 겁니다. 오닐이라면 이렇게 말했을 겁니다. "이익은 완벽한데, 차트가 아직 준비가 안 됐다." 마커스라면 "펀더멘털과 기술적 신호가 같은 방향을 가리킬 때만 큰 확신을 갖는다"고 했을 텐데, 지금 WPM은 정확히 그 반대 상황입니다 — 실적은 4분기 연속 서프라이즈, 애널리스트는 전원 매수 등급, 그런데 주가는 2월 고점 $163.65에서 -34.5% 빠졌습니다. 이 괴리 자체가 "시장이 뭔가 알고 있거나, 혹은 밸류에이션 되돌림이 진행 중"이라는 신호입니다. 어느 쪽인지는 차트가 말해줄 겁니다. 지금은 그 답을 기다릴 시점입니다.

### 2) 시나리오별 트레이딩 플랜

**시나리오 A — 추세 반전 확인 후 돌파 매수 (선호 시나리오)**

- **진입가**: $128.00 (박스권 상단 $126~132 돌파 확인 시, 종가 기준)
- **손절가**: $118.00 (돌파 실패·되돌림 시 직전 저항 하회, 진입가 대비 -7.8%)
- **1차 목표가**: $163.00 (2월 고점 재접근, 진입가 대비 +27.3%)
- **2차 목표가(추세 지속 시)**: $171.00 (애널리스트 평균 목표가, 진입가 대비 +33.6%)
- **손익비 계산**: 리스크 = $128 - $118 = $10 / 보상(1차) = $163 - $128 = $35 → **손익비 3.5 : 1**
- **손익비 계산(2차 목표)**: 보상 = $171 - $128 = $43 → **손익비 4.3 : 1**

**시나리오 B — 현재가 근처 저가 매수 (공격적, 비선호)**

- **진입가**: $108.53 (현재가)
- **손절가**: $87.00 (52주 저점 $87.96 소폭 하회, 진입가 대비 -19.8%)
- **목표가**: $132.00 (직전 박스권 상단 회복, 진입가 대비 +21.6%)
- **손익비 계산**: 리스크 = $21.53 / 보상 = $23.47 → **손익비 1.09 : 1**
- 평가: 손절 폭이 -19.8%로 과도하게 넓고 손익비도 1:1 수준에 불과 — 데니스·오닐 원칙(손절은 좁게, 보상은 크게) 위반. **채택하지 않음.**

**시나리오 C — 되돌림 매수 (박스권 하단 지지 확인 시)**

- **진입가**: $108.53 (현재가 부근, 지지 확인 조건부)
- **손절가**: $103.00 (진입가 대비 -5.1%, 최근 저점 이탈)
- **목표가**: $126.00 (박스권 하단 재테스트 후 상단 접근, 진입가 대비 +16.1%)
- **손익비 계산**: 리스크 = $5.53 / 보상 = $17.47 → **손익비 3.16 : 1**
- 평가: 손익비는 양호하나 "지지 확인" 자체가 데이터 팩만으로는 검증 불가(거래량·일봉 패턴 데이터 없음) → 조건부 관망 유지.

### 3) 포지션 사이징 계산 (하이트 규칙: 단일 거래 손실 1~2%)

가정: 계좌 규모 $100,000, 단일 거래 최대 손실 한도 1.5% = $1,500

**시나리오 A 기준 (진입 $128, 손절 $118, 주당 리스크 $10)**
- 최대 매수 수량 = $1,500 / $10 = **150주**
- 포지션 규모 = 150주 × $128 = $19,200 (계좌의 19.2%)
- 실제 최대 손실액 = 150주 × $10 = $1,500 (계좌의 1.5%, 한도 준수)

**시나리오 C 기준 (진입 $108.53, 손절 $103, 주당 리스크 $5.53)**
- 최대 매수 수량 = $1,500 / $5.53 = **271주**
- 포지션 규모 = 271주 × $108.53 = $29,412 (계좌의 29.4%)
- 실제 최대 손실액 = 271주 × $5.53 = $1,499 (계좌의 1.5%, 한도 준수)

→ 손절 폭이 좁을수록(시나리오 C) 같은 리스크 예산으로 더 큰 포지션을 잡을 수 있다는 것이 하이트 원칙의 핵심입니다. 다만 진입 신뢰도는 A가 높습니다(추세 확인).

## 리스크

- **가장 큰 리스크는 "싸다"는 이유로 하락 추세 중에 매수하는 것**입니다. 마커스의 첫 교훈: 포지션이 불리하게 움직이면 빠르게 인정하고 빠져나와야 한다. 현재 WPM은 5개월째 고점 대비 하락 중이며, "이미 34% 빠졌으니 바닥"이라는 판단은 희망이지 전략이 아닙니다.
- **은 가격 연동 레버리지**: 데이터 팩에 따르면 2026년 초 주가 급등은 은 가격 급등(+161% YoY) 레버리지 효과였습니다. 이는 상승 시 유리하지만 하락 시에도 동일하게 작동합니다 — 최근 조정도 귀금속가 조정과 연동될 가능성이 높으나 데이터 팩에 확정 근거는 없습니다(확인 불가).
- **밸류에이션 부담**: PER(trailing) 27.4x, PBR 5.3x, EV/EBITDA 20.7x — 무차입 고마진 기업 프리미엄을 감안해도 낮지 않은 수준. 펀더멘털 우호적 신호(PEG 0.43)와 상충되는 대목이라 밸류에이션 하나만으로 판단하지 않습니다(저는 밸류에이션 신봉자가 아닙니다).
- **거래량·일봉 패턴 데이터 부재**: 돌파/이탈 확인에 필요한 거래량 데이터, 이동평균선 데이터가 데이터 팩에 없어 시나리오 A·C의 "확인" 조건을 지금 이 자리에서 검증할 수 없습니다. 이는 이 보고서의 구조적 한계입니다.
- **공매도 비율 확인 불가**: 수급 심리 판단에 필요한 정보 결측.

## 최종 의견

제가 인터뷰한 위저드들에게 지금 이 차트를 보여주면 다들 같은 질문을 던질 겁니다. "추세가 어디 있습니까?" 실적은 완벽합니다 — 4분기 연속 서프라이즈, 순이익 YoY +129%, 무차입 경영, 애널리스트 16명 전원 매수. 하지만 주가는 2월 고점에서 34% 넘게 빠졌고, 아직 반등 추세를 확인할 신호가 없습니다. 오닐이라면 "펀더멘털이 아무리 좋아도 시장이 확인해주기 전엔 사지 않는다"고 했을 것이고, 세이코타라면 "포지션이 없다는 것도 하나의 포지션"이라고 했을 겁니다.

저라면 지금 당장 사지 않습니다. 대신 $126~132 박스권 상단 돌파를 기다리겠습니다. 돌파가 확인되면 진입가 $128, 손절 $118(리스크 7.8%), 1차 목표 $163(손익비 3.5:1)로 명확한 계획 하에 들어갑니다. 만약 지금 당장 포지션을 잡고 싶은 충동이 든다면, 그것은 분석이 아니라 "떨어지는 걸 보고만 있기 싫은" 감정입니다. 희망은 전략이 아닙니다.
