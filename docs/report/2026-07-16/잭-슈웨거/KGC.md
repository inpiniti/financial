# KGC 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-16 | 투자의견: 관망 (조건부 소액 진입) | 확신도: 5/10

## 한 줄 결론
추세가 꺾인 지 얼마 안 된 하락 국면에서 손절선과 손익비를 먼저 정해놓고, 지지가 확인되기 전까지는 소액·타이트한 손절로만 접근한다 — 지금 전액을 태우는 건 "돌아오겠지"라는 희망일 뿐이다.

## 내 철학에서 본 이 기업
저는 이 기업이 금을 얼마나 잘 캐는지, PER이 몇 배인지는 부차적으로 봅니다. 제가 인터뷰한 마켓 위저드들이라면 딱 두 가지부터 물었을 겁니다. "지금 추세는 어느 방향인가?" 그리고 "틀렸을 때 어디서 빠져나올 것인가?"

KGC는 최근 30일 -18.4%, 최근 1분기 -26.2% 하락 중입니다. 52주 고점 $39.11 대비 -39.5%. 이건 명백한 하락 추세이고, 원인도 뚜렷합니다 — 매파적 Fed 신호로 달러 강세, 미-이란 양해각서로 안전자산 수요 감소, 금값 조정에 금광주 전반 차익실현. 펀더멘털(1Q2026 조정 EPS $0.71, FCF 4분기 연속 사상최대)은 여전히 훌륭하지만, 마이클 마커스라면 "펀더멘털과 기술적 신호가 반대 방향을 가리킬 때는 확신을 낮춰야 한다"고 했을 겁니다. 지금이 정확히 그런 상황입니다 — 실적은 사상 최고, 주가는 4개월째 급락.

## 밸류에이션 판단 (계산 과정 명시)

저는 PER이나 DCF로 "적정가"를 산출하는 사람이 아닙니다. 제가 산출하는 건 진입가·손절가·목표가·손익비·포지션 크기입니다.

**1. 진입가**
- 현재가 $23.66을 기준가로 사용. 단, 하락 추세가 아직 확인된 반등 신호 없이 진행 중이므로 전량 진입이 아니라 "1차 소액 진입 + 지지 확인 후 추가" 방식 권고.
- 1차 진입가: **$23.66** (현재가, 시험적 소액)

**2. 손절가 (오닐의 7~8% 룰 적용)**
```
손절가 = 진입가 × (1 - 8%)
       = 23.66 × 0.92
       = 21.77 → $21.70 (직전 저점권 살짝 하회, 라운드)
손절폭 = 23.66 - 21.70 = $1.96/주 (-8.3%)
```

**3. 목표가 (최근 가격 구조 기반, 컨센서스 그대로 쓰지 않음)**
- 애널리스트 평균 목표가 $36.51(+54%)은 참고만 함 — 추세가 확인되지 않은 상태에서 컨센서스를 목표로 삼는 건 "희망을 전략으로 쓰는" 행위.
- 보수적 1차 목표: 직전 하락 시작 전 지지 레벨 **$28.30** (25-12 종가권)
- 공격적 2차 목표(추세 재개 확인 시 트레일링): **$32.80** (26-04 되돌림 고점)

**4. 손익비**
```
보수적: (28.30 - 23.66) / (23.66 - 21.70) = 4.64 / 1.96 = 2.37 : 1
공격적: (32.80 - 23.66) / 1.96            = 9.14 / 1.96 = 4.66 : 1
```
데니스의 원칙 — "틀렸을 때 빨리 손절, 맞았을 때 최대한 보유" — 을 따르면 2차 목표까지 트레일링 스탑으로 홀딩할 가치는 있음. 다만 1차 진입 시점의 손익비 2.37:1은 최저 기준(통상 2:1 이상)을 겨우 통과하는 수준.

**5. 포지션 사이징 (단일거래 리스크 1% 룰, 계좌 $100,000 가정)**
```
허용 리스크 금액 = 계좌 × 1% = $1,000
주당 리스크        = 진입가 - 손절가 = $1.96
매수 가능 주식수   = 1,000 / 1.96 ≈ 510주
투입 자본          = 510 × 23.66 ≈ $12,067 (계좌의 약 12.1%)
```
1주 리스크가 진입가 대비 8.3%로 상대적으로 넓기 때문에, 자본 배분 대비 리스크 비중은 낮게 눌러야 함. 확신도가 낮은 지금은 이 사이징의 절반(약 250~260주, 계좌의 약 6%)으로 시작하는 것을 권고.

## 리스크
- **매크로 변수 예측 불가**: 금값은 Fed 정책·지정학 리스크에 연동. 매파적 FOMC 지속이나 지정학 긴장 추가 완화 시 금값·금광주 동반 추가 조정 가능 — 이건 어떤 분석으로도 예측할 수 없는 영역, 손절로만 대응 가능.
- **연승 후 자만 금지**: KGC는 2023~2025년 실적이 매년 큰 폭 개선(순이익 -605M→2,390M)됐고 이게 오히려 "이 정도면 무조건 반등한다"는 과신을 유발하기 쉬움. 실적이 아무리 좋아도 추세가 하락이면 시장이 옳다.
- **손절 미이행 리스크**: 애널리스트 목표가($36.51~54)에 대한 기대가 손절 규칙 무시를 유혹할 수 있음 — 이것이 마켓 위저드들의 최악의 손실이 발생한 전형적 패턴.
- **유동성/공매도**: 공매도 비율 1.71%, short ratio 1.9일로 유동성 리스크는 낮음.

## 최종 의견
제가 인터뷰한 트레이더들에게 이 차트를 보여줬다면 하나같이 이렇게 물었을 겁니다. "실적이 사상 최고인데 주가가 4개월째 빠지고 있다면, 지금 당신이 보는 건 기회인가 함정인가?" 답은 아직 모릅니다 — 추세가 아직 확인되지 않았으니까요. 펀더멘털은 훌륭합니다. FCF 사상최대, D/E 0.08, ROE 27.9%. 하지만 마이클 마커스의 조건 — 펀더멘털과 기술적 신호가 같은 방향 — 이 충족되지 않은 지금은 전액 베팅할 자리가 아닙니다.

그래서 저는 "관망"이되, 완전한 관망이 아니라 손절선과 손익비를 먼저 정해놓고 소액으로 시험 진입하는 조건부 접근을 제시합니다. 손절폭 8.3%, 손익비 최소 2.37:1, 계좌 리스크 1% 이내 — 이 세 가지가 갖춰지지 않으면 진입 자체를 하지 마십시오. 희망은 전략이 아닙니다.
