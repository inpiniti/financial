### 손익비 계산

```
주당 손실폭(Stop)   = $15.20 - $13.90 = $1.30 (-8.6%)
주당 이익폭(T1)      = $20.46 - $15.20 = $5.26 (+34.6%)
주당 이익폭(T2)      = $23.63 - $15.20 = $8.43 (+55.5%)

손익비(T1) = 5.26 / 1.30 = 4.05 : 1
손익비(T2) = 8.43 / 1.30 = 6.48 : 1
```

리처드 데니스의 원칙대로 손익비가 4~6배면 승률 30%만 넘어도 기대값이 플러스입니다. 이 구조 자체는 매력적입니다.
