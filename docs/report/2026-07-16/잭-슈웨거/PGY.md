# PGY 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-16 | 투자의견: 관망 | 확신도: 5/10

## 한 줄 결론
펀더멘털은 흑자전환·가이던스 상향으로 우호적이지만, 기술적으로는 52주 저가권 박스권($15.6~19)에 머물며 2026-02-09 하루 -24% 급락 전례가 있는 Beta 5.34짜리 종목이다. 추세와 펀더멘털이 아직 같은 방향을 가리키지 않으므로, 박스권 상단 돌파를 확인할 때까지 기다리는 것도 유효한 포지션이다.

## 내 철학에서 본 이 기업
제가 인터뷰한 마이클 마커스는 펀더멘털과 기술적 분석이 같은 방향을 가리킬 때 가장 큰 확신을 가졌다고 했습니다. PGY는 지금 그 조건을 충족하지 못합니다. FY2025 순이익이 -$401M 적자에서 $81.4M 흑자로 전환했고, 1Q2026 EPS도 컨센서스를 크게 상회했으며, 2026년 순이익 가이던스도 $100~150M로 상향됐습니다 — 펀더멘털은 분명 개선되고 있습니다. 그러나 주가는 52주 고가 $44.99에서 저가 $10.40까지 급락한 뒤 현재 $17.96, 레인지 하위 약 27% 지점에 머물러 있습니다. 이 종목이 2026-02-09 단 하루 만에 -24% 급락한 전례(네트워크 볼륨 성장 미달, 매출 컨센서스 하회)를 가지고 있다는 사실은, 실적 발표일마다 갭 리스크가 현실화될 수 있다는 것을 증명합니다. 폴 튜더 존스라면 "얼마를 벌 수 있는가"보다 "얼마를 잃을 수 있는가"를 먼저 물었을 겁니다.

## 밸류에이션 판단 (계산 과정)

트레이더 관점에서는 내재가치보다 진입/손절/목표/손익비/사이징이 핵심이다.

### 가격 구조 확인
- 최근 1개월 박스권: 상단 저항 $18.25~19 / 하단 지지 $15.6~16
- 52주 레인지: $10.40 ~ $44.99
- 2025-07 전고점: 약 $31
- 애널리스트 목표주가: 평균 $26.50 / 중앙값 $25.50

### 시나리오 A — 박스권 상단 돌파 매매 (선호)

| 항목 | 값 | 근거 |
|---|---|---|
| 진입가 | $19.00 | 박스권 상단 돌파 확인 시 |
| 손절가 | $15.50 | 박스권 하단 이탈 지점 |
| 1차 목표가 | $25.50 | 애널리스트 목표주가 중앙값 |
| 2차 목표가 | $31.00 | 2025-07 전고점 |
| 손절폭 | -18.4% (진입가 대비) | |
| 손익비(1차) | (25.5-19)/(19-15.5) = 6.5/3.5 = **1.86:1** | |
| 손익비(2차) | (31-19)/(19-15.5) = 12/3.5 = **3.43:1** | |

### 시나리오 B — 현재가 즉시 진입

|---|---|---|

### 포지션 사이징 (계좌 $100,000 가정, 단일거래 리스크 한도 1.5%)

- 최대 손실 허용액 = $100,000 × 1.5% = $1,500
- 시나리오 A: 리스크/주 = $19.00-$15.50 = $3.50 → 포지션 = $1,500/$3.50 ≈ **428주** (매입금액 약 $8,132, 계좌의 약 8.1%)
- 시나리오 B(타이트 손절): 리스크/주 = $17.96-$16.50 = $1.46 → 포지션 = $1,500/$1.46 ≈ **1,027주** (매입금액 약 $18,447, 계좌의 약 18.4%) — Beta 5.34 종목 대비 과도, 단일거래 매입금액 상한을 계좌의 10~12%로 별도 제한 권고
