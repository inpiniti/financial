# CTRE 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-16 | 투자의견: 관망(눌림목 대기 후 소규모 진입) | 확신도: 4/10

## 한 줄 결론
펀더멘털은 우호적(FFO 가이던스 상회 추세)이지만 기술적으로는 $41~43 저항대를 아직 뚫지 못한 박스권 종목이다 — 지금 쫓아가지 말고, 저항 돌파 확인 또는 지지선 재테스트 중 하나를 기다려라. 손절 폭을 좁게 설계할 수 있는 자리가 아니면 진입하지 않는다.

## 내 철학에서 본 이 기업

FFO는 가이던스를 상회하고(YoY +14%), 애널리스트 13명 중 12명이 매수 이상 등급을 주고 있죠. 하지만 GAAP EPS는 4분기 연속 컨센서스를 하회했고(최근 -1.3%로 폭은 줄었지만), 주가는 5월 유상증자 이후 $37~41 사이를 두 달째 오가고 있습니다. 오닐이라면 "신고가 부근에서 거래량을 동반한 돌파가 나오기 전까지는 매수하지 않는다"고 했을 겁니다. 지금은 그 돌파가 확인되지 않은 상태입니다.

## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

저는 내재가치를 산출하지 않습니다. 대신 진입가·손절가·목표가·손익비·포지션 사이징을 계산합니다.

### 시나리오 A — 저항 돌파 추격 진입 (선호하지 않음, 참고용)
- 진입가: $41.50 (52주 고가 $43.08 부근 돌파 확인 후)
- 손절가: $39.00 (돌파 실패 시 되돌림 한계선, 진입가 대비 -6.0%)
- 목표가: $45.31 (애널리스트 평균 목표가) → 진입가 대비 +9.2%
- 손익비 = (45.31 - 41.50) / (41.50 - 39.00) = 3.81 / 2.50 = **1.52 : 1**
- 평가: 손익비 1.5배 수준으로 마켓 위저드 기준(통상 2~3:1 이상 선호)에 못 미침. 오닐 원칙(매수가 대비 7~8% 하락 시 손절)과도 손절 폭이 애매하게 걸침 → 채택 보류.

### 시나리오 B — 지지선 재테스트 진입 (선호)
- 진입가: $37.50 (최근 6월 저점 $37.1 부근 지지 확인 시, 되돌림 매수)
- 손절가: $35.50 (지지 붕괴 확인선, 진입가 대비 -5.3%)
- 목표가 1차: $41.50 (직전 저항, +10.7%)
- 목표가 2차: $45.31 (애널리스트 평균, +20.8%)
- 손익비(1차) = (41.50-37.50)/(37.50-35.50) = 4.00/2.00 = **2.0 : 1**
- 손익비(2차) = (45.31-37.50)/(37.50-35.50) = 7.81/2.00 = **3.9 : 1**
- 평가: 손절 폭이 좁고 손익비가 2:1~3.9:1로 충분히 매력적. 데니스의 원칙("틀렸을 때 빨리, 맞았을 때 오래")에 부합하는 자리.

### 포지션 사이징 (시나리오 B 기준, 래리 하이트 방식: 단일 거래 손실 한도 1% 가정)
- 가정 계좌 규모: $100,000 (예시 — 실제 계좌 규모에 비례 적용)
- 허용 손실액 = 계좌의 1% = $1,000
- 주당 손실 = 진입가 $37.50 - 손절가 $35.50 = $2.00
- 포지션 규모(주식 수) = $1,000 / $2.00 = **500주**
- 포지션 금액 = 500주 × $37.50 = $18,750 (계좌의 18.75%)
- 리스크 상한 2% 적용 시: 허용 손실 $2,000 → 1,000주, 포지션 금액 $37,500(계좌의 37.5%) — 단일 종목 집중도가 높아지므로 1% 룰 권장.

## 최종 의견
