## 한 줄 결론
"얼마를 벌 수 있는가보다 얼마를 잃을 수 있는가"부터 물어야 하는 종목이다 — 손익비 자체는 나쁘지 않지만, 배당 정책 급변이라는 펀더멘털 훼손 직후라 추세 확인이 안 됐다. 지금은 "기다리는 것도 포지션"이다.

## 내 철학에서 본 이 기업
나는 회사를 평가하지 않는다. 이 거래가 어떤 조건에서 이기고 어떤 조건에서 지는지, 그리고 졌을 때 얼마를 잃는지를 평가한다. PFLT는 2026-05-07 배당 정책 전면 개편(월 $0.1025 고정 → 기본 $0.08 + 변동 보충배당)이라는 명백한 악재로 52주 고점 $10.88에서 현재 $7.21까지 -33.7% 급락했다. 이건 마커스가 말하는 "펀더멘털과 기술적 신호가 같은 방향"의 반대 케이스다 — 펀더멘털은 나빠졌고(배당 삭감), 기술적으로는 하락 추세가 막 저점권 횡보로 전환되는 애매한 구간이다. 이런 상황에서 "NAV 대비 31% 할인이니 싸다"는 논리로 들어가는 건 내가 인터뷰한 트레이더들이 가장 경계한 "언젠가 돌아올 것"이라는 희망 매매다.

## 밸류에이션 판단 — 이건 밸류에이션이 아니라 리스크 설계다

나는 적정가를 계산하지 않는다. 대신 "이 거래에 들어간다면 어디서, 왜, 얼마를 걸고, 언제 틀렸다고 인정할 것인가"를 계산한다.

### 1) 진입 조건
- 현재가 $7.21, 52주 저점 $7.02~7.18 부근에서 급락 후 횡보 중 — 아직 상승 추세 미확인.
- 진입은 지금 당장이 아니라 **저점권 지지 재확인 + 거래량 동반 반등** 조건부. 조건 충족 시 진입가 **$7.20 부근**으로 가정해 아래 계산.

### 2) 손절가
- 52주 저점 $7.02 하향 이탈 확정(종가 기준) 시 즉시 손절: **$6.85**
- 진입가 대비 손실폭: (7.20 - 6.85) / 7.20 = **-4.86%**

### 3) 목표가 (시나리오)
| 시나리오 | 목표가 | 근거 |
|---|---|---|
| 1차 (할인율 정상화) | $8.30 | NAV $10.47 대비 할인율 -31% → -20%로 축소되는 수준. 급락 전 지지선 근접 |
| 2차 (신뢰 회복) | $9.20 | 배당 개편 발표 직전 갭 구간, 애널리스트 최저 목표가($9.00)에 근접 |
| 참고: 애널리스트 평균 목표가 | $10.08 | 배당컷 이전 미조정 가능성 높아 신뢰도 낮음 — 목표에서 제외 |

### 4) 손익비 계산
```
1차 목표: 보상 = 8.30 - 7.20 = 1.10 / 리스크 = 7.20 - 6.85 = 0.35
→ 손익비 = 1.10 / 0.35 = 3.14 : 1

2차 목표: 보상 = 9.20 - 7.20 = 2.00 / 리스크 = 0.35
→ 손익비 = 2.00 / 0.35 = 5.71 : 1
```
데니스의 원칙대로 "틀렸을 때 빨리, 맞았을 때 오래"가 성립하는 숫자다. 손익비 자체는 매력적이다.

### 5) 포지션 사이징 (계좌 $100,000 가정, 단일 거래 리스크 1% 룰)
```
허용 손실액 = $100,000 × 1% = $1,000
주당 리스크 = $7.20 - $6.85 = $0.35
이론적 최대 수량 = $1,000 / $0.35 ≈ 2,857주
이론적 매수금액 = 2,857주 × $7.20 ≈ $20,571 (계좌의 20.6%)
```
→ 단일 종목 20.6% 배분은 하이트의 분산 원칙 위반. 종목당 비중 상한(10%)을 별도로 캡.
```
실제 권고 배분 = $10,000 → 1,388주
실제 리스크액 = 1,388주 × $0.35 = $486 (계좌의 0.49%)
```
**진입가 $7.20 / 손절가 $6.85 / 1차 목표 $8.30 / 2차 목표 $9.20 / 손익비 3.1~5.7:1 / 권고 배분 계좌의 10%(리스크 0.5%) / 조건 미충족 시 미진입.**

## 리스크
- 배당 정책 개편(2026-05-07)이 시장에 완전히 소화됐는지 미확인 — 추가 하락 여력 존재.
- FY2026 Q3 실적 발표(2026-08-10)가 다음 트리거. 손절가 $6.85 도달 전에 실적이 먼저 나올 수 있어 이벤트 리스크 큼.
- 비발생대출 0.5%는 낮지만 저금리 국면 진입 시 변동금리 대출 포트폴리오(99%)의 NII가 구조적으로 축소될 수 있음 — 배당 추가 조정 리스크.
- 애널리스트 목표가($10.08)가 미조정 상태일 가능성 — 이를 근거로 한 낙관은 신뢰하지 않는다.
- 52주 저가 정확값 자체가 소스 간 $7.02 vs $7.18로 불일치 — 손절 트리거 판단에 노이즈.

## 최종 의견
제가 인터뷰한 트레이더들에게 이 차트를 보여줬다면 십중팔구 이렇게 물었을 겁니다. "배당을 깎았는데 왜 지금 사려고 하는가?" NAV 대비 31% 할인이라는 숫자는 유혹적이지만, 마커스라면 "펀더멘털과 기술적 신호가 같은 방향을 가리키지 않으면 확신을 갖지 말라"고 했을 겁니다. 지금 이 종목은 펀더멘털(배당컷, NII 압박)은 아래를 가리키고, 가격은 저점권에서 숨을 고르고 있을 뿐입니다. 손익비 3:1 이상이라는 계산 자체는 매력적이지만, 그건 진입 조건이 충족됐을 때의 이야기입니다. 조건이 안 맞는데 "싸 보여서" 들어가는 건 데니스가 말한 규율이 아니라 세이코타가 경계한 감정입니다. 저점권 지지가 거래량과 함께 확인되기 전까지는 관망하고, 확인되면 계좌의 10% 이내, 리스크 0.5% 이내로 소액 진입하십시오. 그리고 $6.85가 깨지면 이유를 묻지 말고 나오십시오. 희망은 전략이 아닙니다.
