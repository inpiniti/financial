# NOW 투자 보고서 — 잭 슈웨거 (마켓 위저드 — 주식 시장의 승자들)
> 날짜: 2026-07-16 | 투자의견: 관망 (실적 발표 후 재진입 검토) | 확신도: 5/10

## 한 줄 결론
추세도 시장 방향도 아직 확인되지 않은 상태에서 6거래일 앞으로 다가온 실적 발표(2026-07-22)를 앞두고 있다 — 이건 "기다림도 포지션이다"의 교과서적 상황이지, 지금 방아쇠를 당길 자리가 아니다.

## 내 철학에서 본 이 기업
저는 ServiceNow가 좋은 회사인지 나쁜 회사인지를 판단하는 사람이 아닙니다. 제가 인터뷰한 마이클 마커스, 리처드 데니스, 폴 튜더 존스 중 누구도 "이 회사가 훌륭한가"로 거래를 시작하지 않았습니다. 그들은 언제나 두 가지만 물었습니다. "이 거래가 틀렸을 때 어디서 빠져나올 것인가?"와 "지금 추세와 방향이 나에게 유리한가?" NOW의 현재 상태는 이 두 질문에 명확히 답하기 어려운 구간에 있습니다.

### 포지션 사이징 계산 (하이트의 1% 규칙 적용, 가정 계좌 $100,000)

시나리오 A(참고, 지금 진입 시 가정)로 계산 예시:

```
계좌 자본 = $100,000
단일 거래 최대 손실 허용액 = 계좌 자본 × 1% = $1,000
손절폭(주당) = 진입가 $104.73 - 손절가 $95.00 = $9.73
최대 매수 주식수 = 최대 손실 허용액 ÷ 손절폭
             = $1,000 ÷ $9.73
             ≈ 102주
포지션 규모 = 102주 × $104.73 ≈ $10,682 (계좌의 약 10.7%)
```

계좌의 1%(더 보수적으로는 2%까지)만 리스크에 노출하고, 나머지는 다음 기회를 위해 남겨둡니다. 실적 발표 전 변동성 확대 구간이므로 사이징을 표준보다 30~50% 축소하는 것을 권합니다(예: 리스크 0.5%, 약 51주).

## 리스크

- **실적 이벤트 리스크**: 2026-07-22 발표 전 진입은 손절가가 갭으로 뚫릴 위험이 있음 — 이것이 지금 관망을 권하는 가장 큰 이유
- **섹터 동조화 리스크**: IBM 프리어나운스가 개별 기업 문제가 아니라 "AI 인프라 투자가 SW 예산을 잠식한다"는 섹터 전체 내러티브를 건드림. NOW의 펀더멘털이 좋아도 섹터 리레이팅 국면에서는 저항하기 어려움(오닐의 M 원칙 — 종목은 시장/섹터를 이기지 못한다)
- **밸류에이션 부담**: trailing PER 62배, EV/EBITDA 36배는 여전히 고밸류에이션 구간 — 실적이 조금만 흔들려도 배수 조정 폭이 클 수 있음
- **소송·규제 리스크**: 증권사기 조사(진행 중, 확정 아님), API 보안 사고 — 재무적 임팩트는 현재 미미하나 헤드라인 리스크로 단기 변동성 유발 가능
- **공매도 6.03%, 숏레이쇼 2.13일**: 과도한 숏 스퀴즈 리스크는 낮으나, 실적 서프라이즈 시 숏 커버링에 의한 변동성 확대 가능성

## 최종 의견

제가 인터뷰한 사람 중 누구도 "실적 발표 6일 전, 섹터 전체가 동반 급락한 다음 날" 서둘러 들어가지 않았습니다. 마크 와인스타인이라면 이렇게 말했을 겁니다. "조건이 정렬되지 않았으면, 아무리 좋아 보여도 기다린다." NOW의 펀더멘털(매출 YoY +22%, 4분기 연속 서프라이즈)은 분명 매력적입니다. 하지만 지금은 기술적 신호와 펀더멘털 신호가 정반대 방향을 가리키고 있고, 손절 규칙이 실적 갭에 의해 무력화될 위험이 있는 자리입니다.

손익비 계산 자체는 3:1을 넘어 매력적이지만, 그 계산의 전제인 "손절가에서 정확히 빠져나올 수 있다"는 가정이 실적 발표 전에는 성립하지 않습니다. 희망은 전략이 아닙니다. 저는 여기서 기다립니다. 실적이 나오고 시장의 반응이 확인된 뒤, 추세가 재개되면 그때 위에서 계산한 사이징으로 들어갑니다. 실적이 실망스럽다면 저점 $81.24 재테스트까지 손대지 않습니다.
