## 밸류에이션 판단 (계산 과정을 표/식으로 명시)

저는 내재가치를 산출하지 않습니다. 대신 진입-손절-목표를 계산해 손익비로 거래 타당성을 검증합니다.

**기준 데이터**
- 현재가(진입 후보): $23.95
- 52주 고가: $24.43 / 52주 저가: $19.37
- 최근 박스권 하단(2026-02~03 조정 저점): 약 $21.00
- 애널리스트 목표가: 평균 $24.90 / 중앙값 $25.00 / 최고 $29.00 / 최저 $21.00

**진입가·손절가·목표가**

| 항목 | 값 | 근거 |
|---|---|---|
| 진입가 | $23.95 | 현재가, 박스권 상단 돌파 시도 구간 |
| 손절가 | $21.00 | 최근 박스권 하단 + 애널리스트 최저 목표가 일치 지점 |
| 1차 목표가 | $25.00 | 애널리스트 중앙값 |
| 확장 목표가 | $29.00 | 애널리스트 최고치 |

**손익비 계산**

- 리스크(주당 손실 폭) = $23.95 − $21.00 = **$2.95**
- 리워드1(1차 목표) = $25.00 − $23.95 = $1.05 → 손익비 = 1.05 / 2.95 ≈ **0.36 : 1** (불리)
- 리워드2(확장 목표) = $29.00 − $23.95 = $5.05 → 손익비 = 5.05 / 2.95 ≈ **1.71 : 1** (양호하나 실현 확률 낮음, 최고 목표가 1개 의견에 불과)

리처드 데니스의 원칙 — "틀렸을 때 빨리 손절, 맞았을 때 최대한 오래 보유" — 을 적용해도, 1차 목표(중앙값) 기준 손익비 0.36:1은 승률이 매우 높아야만 정당화됩니다. 컨센서스가 하향 조정 중인 지금 그 승률을 신뢰하기 어렵습니다.

**포지션 사이징 (단일 거래 리스크 1% 원칙, 래리 하이트 방식)**

- 가정 계좌 규모: $100,000
- 단일 거래 최대 허용 손실: $100,000 × 1% = $1,000
- 주당 리스크: $2.95
- 최대 매수 수량 = $1,000 / $2.95 ≈ **339주**
- 포지션 규모 = 339주 × $23.95 ≈ $8,119 (계좌의 약 8.1%)

이 사이징이라면 최악의 경우(손절가 도달) 손실은 정확히 계좌의 1%로 제한됩니다. 이것이 실행된다는 전제하에서만 진입을 검토할 수 있습니다.

## 리스크

- **손익비 열위**: 1차 목표 기준 손익비 0.36:1은 위저드들의 원칙(손익비 우위 확보)에 위배. 확장 목표를 쓰지 않는 한 이 거래는 수학적으로 매력적이지 않음.
- **펀더멘털 모멘텀 둔화**: 최근 30일 EPS 컨센서스 4건 하향, 7일 4건 하향 — 추세가 꺾일 조짐일 수 있음.
- **고배당 유혹**: 10.71%라는 배당수익률은 "이자수익을 노리고 손절 없이 버티고 싶은" 심리적 함정을 만들 수 있음. 이는 세이코타가 경고한 "감정적 결정"의 전형.
- **52주 고점 근접**: 돌파 실패 시 되돌림 리스크. 돌파 확인 전 진입은 추격매수에 가까움.
- **BDC 특성상 시장 전체(금리·신용 사이클) 방향 미확인**: 데이터팩에 섹터 지표 없음, 오닐의 "M(시장 방향)" 체크 불가.
