# YMM 투자 보고서 — 애스워스 다모다란 (주식 가치평가 완벽 정리)
> 날짜: 2026-07-16 | 투자의견: 매수 | 확신도: 6/10

## 한 줄 결론
스토리(중국 독점적 화물매칭 플랫폼의 성숙기 전환)와 숫자(FCF 수익률 약 7.9%, 순현금, 낮은 할인율)가 상당히 일치하고, 시장이 요구하는 성장률이 현재 규모의 사업에서 충분히 달성 가능한 수준까지 낮아져 있어 — 안전마진은 있으나 VIE/상장폐지 리스크로 확신도는 제한된다.

## 내 철학에서 본 이 기업
YMM은 이미 고성장기를 상당 부분 통과한 기업입니다. 2022→2025년 영업이익률이 -2.4%에서 33.2%로 급등했고, 이는 지속 가능한 마진 구조가 아니라 이제 막 정상화된 것으로 봐야 합니다. 순현금(총현금 18,464백만 CNY가 시총을 초과)에 무차입에 가까운 재무구조(D/E 0.09%)는 위험을 낮추는 요인이지만, 베타 0.305는 지나치게 낮아 그대로 신뢰하기 어렵습니다 — 중국 ADR이라는 국가·규제 위험이 저베타에 온전히 반영되지 않았을 가능성이 큽니다. 이것이 제가 할인율에 컨트리 리스크 프리미엄을 별도로 얹는 이유입니다.

## 밸류에이션 판단 (계산 과정)

### 0. 환율 프록시 산출
데이터 팩에 USD/CNY 환율이 명시되지 않아, trailing PER와 CNY EPS로 역산.

```
암묵환율 = (Trailing PER × 희석EPS 2025) / 현재가
        = (14.60 × 4.20 CNY) / $9.20
        = 61.32 / 9.20
        = 6.665 CNY/USD  (실제 시장 환율 수준과 근접 → 프록시로 채택)
```

### 1. 기초 자기자본 FCF (2025년 실적 기준)
```
FCF(CNY) = 4,497백만
FCF(USD) = 4,497 / 6.665 = $674.7M
발행주식수 = 932.51M
주당 FCF(FCFPS) = $674.7M / 932.51M = $0.7236
FCF 수익률(현재가 대비) = 0.7236 / 9.20 = 7.87%
```

### 2. 할인율(자기자본비용, Ke) 산정
```
Ke = Rf + β×ERP + 컨트리리스크 프리미엄
   = 4.3% + 0.305×4.5% + 3.0%
   = 4.3% + 1.37% + 3.0%
   = 8.67% → 보수적으로 9.0% 채택 (VIE/상장폐지 리스크 반영)
```

### 3. 2단계 DCF — 시나리오별 산정
고성장기 5년, 이후 영구성장률(g∞) 2.5%로 수렴. 마진은 2025년 수준(순이익률 35.3%대는 다소 이례적 고점으로 보아 정상화 마진 28~30%대로 조정) 가정.

**공통 가정**: 기초 FCFPS $0.7236, 할인율 9.0%, 영구성장률 2.5%

| 시나리오 | 고성장기(1~5yr) FCF 성장률 | 논리 |
|---|---|---|
| 비관 | 0% (정체, Q2 가이던스 매출 감소 반영) | 화물매칭 시장 성숙, 규제 리스크로 마진 압박 |
| 기본 | 8% | 완료주문 +14.3%, MAU +12.7% 등 물량 성장은 지속되나 매출·마진 증가율은 둔화 반영 |
| 낙관 | 15% | Giga.AI·부가서비스(신용·보험·ETC) 확장이 마진 개선으로 이어지는 경우 |

**계산 (기본 시나리오, g=8%, 5년 후 g∞=2.5%로 수렴)**

| 합계(1~5yr PV) | | | **3.5196** |

```
5년차 FCFPS = 1.0632
터미널가치(5년말) = FCFPS5 × (1+g∞) / (Ke - g∞)
              = 1.0632 × 1.025 / (0.09-0.025)
              = 1.0898 / 0.065
              = 16.766
터미널가치 현재가치 = 16.766 × 0.6499 = 10.897

기본 시나리오 주당가치 = 3.520 + 10.897 = $14.42
```

**비관 시나리오 (g=0%, 5년간 FCFPS 고정 $0.7236)**

### 4. 역산 DCF (Implied Growth Rate)
현재가 $9.20을 정당화하는 5년 성장률 g를 구합니다 (Ke=9%, g∞=2.5%, 기초 FCFPS=0.7236 고정).

```
목표: PV(1~5yr FCF) + PV(터미널가치) = $9.20

시행착오(수치 탐색):
  g = -3% 대입 시:
    FCFPS(1~5): 0.7019, 0.6809, 0.6605, 0.6407, 0.6215
    할인계수 적용 PV 합 = 0.6438+0.5730+0.5099+0.4538+0.4038 = 2.584
    5년차 FCFPS=0.6215, 터미널가치=0.6215×1.025/0.065=9.802
    터미널 PV = 9.802×0.6499 = 6.370
    합계 = 2.584 + 6.370 = $8.95  (근접, 목표 $9.20보다 살짝 낮음)

  g = -2% 대입 시:
    FCFPS(1~5): 0.7091, 0.6949, 0.6810, 0.6674, 0.6540
    PV 합 ≈ 0.6505+0.5849+0.5259+0.4728+0.4250 = 2.659
    5년차 FCFPS=0.6540, 터미널가치=0.6540×1.025/0.065=10.313
    터미널 PV = 10.313×0.6499 = 6.702
    합계 = 2.659 + 6.702 = $9.36  (목표 $9.20 살짝 초과)

  보간(-3%~-2% 사이): g ≈ -2.3%
```

**결론: 현재가 $9.20은 향후 5년간 FCF가 연평균 약 -2%~-2.5% "감소"한다는 가정 하에서만 정당화됩니다.** 그러나 데이터상 완료주문 +14.3%, 화주 MAU +12.7% 등 물량 지표는 여전히 성장 중이고, Q2 가이던스의 매출 감소는 일시적(마진 조정·회계상 요인) 요인으로 보이므로, 시장이 가격에 반영한 성장률 가정은 지나치게 비관적이라고 판단합니다.

## 리스크
- **VIE/중국 ADR 상장폐지 리스크**: JPMorgan은 강제 상장폐지 시 약 $277M 규모 패시브 자금 유출을 지적. 이는 DCF에 담기지 않는 꼬리위험(tail risk)으로, 할인율의 컨트리리스크 프리미엄(3.0%p)으로 부분 반영했으나 완전한 헤지는 아님.
- **저베타(0.305)의 신뢰성**: 실제 위험 대비 과소평가된 베타일 가능성 — 시장이 인식하는 위험(중국 정책·상장폐지)을 통계적 베타가 충분히 담지 못함.
- **마진 정상화의 지속가능성**: 33%대 영업이익률이 이례적 고점일 수 있음. 이미 2025년 기준으로도 주식보상비용·Giga.AI R&D 확대가 순이익을 압박 중(1Q2026 순이익 YoY 감소).
- **부실채권비율(NPL) 상승**: 신용중개 부문 NPL 2.9%→3.2%로 상승 — 부가서비스 확장의 자산건전성 리스크.
- **환율 프록시의 한계**: 데이터 팩에 명시적 환율이 없어 PER 역산으로 추정(6.665 CNY/USD)했으며, 실제 환율과 오차가 있을 수 있음. 이는 계산 전반의 오차 요인.

## 최종 의견
제 가치 추정은 거의 확실히 틀렸을 겁니다. 하지만 범위는 합리적입니다. 세 가지 시나리오 — 비관 $10.23, 기본 $14.42, 낙관 $19.18 — 모두 현재가 $9.20을 상회합니다. 특히 역산 DCF에서 시장은 향후 5년간 FCF가 매년 약 2~2.5% "감소"한다고 가정하고 있는데, 완료주문과 사용자 지표가 여전히 두 자릿수로 성장 중인 기업에 대해 이는 지나치게 비관적인 스토리입니다. 물론 VIE 구조와 상장폐지 리스크라는, 제 DCF 모형이 온전히 담아내지 못하는 꼬리위험이 이 할인의 상당 부분을 설명할 수 있습니다. 그것이 제가 확신도를 10이 아니라 6에 두는 이유입니다 — 숫자는 매수 신호를 보내지만, 스토리의 일부(규제 리스크)는 숫자로 완전히 번역되지 않습니다.
